# 3.4 伺服器配置

---

## ⚙️ 主要設定檔

| 檔案 | 用途 |
|------|------|
| `Channel.properties` | 頻道伺服器設定 |
| `World.properties` | 世界伺服器設定 |
| `ServerConstants.java` | 常數定義 |
| `_config.yaml` | HeavenMS 設定 |

---

## 🌐 IP 設定

### 客戶端 IP
客戶端的 IP 必須指向你的伺服器

### 伺服器 IP
確保以下 IP 正確：
- 外部 IP（供客戶端連接）
- 內部 IP（127.0.0.1 for localhost）

---

## 🔌 端口設定

| 服務 | 預設端口 |
|------|----------|
| Login | 7575 |
| World | 7576 |
| Channel | 7577+ |
| HTTP | 80 |

---

## 🌏 WVS 設定

### WVS 是什麼？
Web View Server，處理遊戲內網頁

### 架設步驟

1. 安裝 WampServer
2. 放入 WVS 檔案到 `www` 目錄
3. 設定 Apache 端口
4. 客戶端連接到正確的 WVS URL
