# 2.3 STRedit 字串修改

---

## 🎯 STRedit 是什麼？

STRedit 是一個專門用於修改 MapleStory 客戶端字串的工具，支援：
- 修改長度不同的字串
- 編輯 UI 文字
- 修改版本顯示

---

## 📥 下載與安裝

1. 從 RaGEZONE 論壇下載 STRedit
2. 下載 `jieziban.rar`（韓文輸入工具）
3. 解壓縮到同資料夾

---

## 📝 使用步驟

### 修改字串

```
1. 執行 STRedit.exe
2. File → Open → 選擇脫殼後的 MapleStory.exe
3. 搜尋要修改的文字（如 "beginner"）
4. 選中後直接輸入新文字（如 "新手"）
5. 存檔
```

### 修改版本號

1. 搜尋 `ver. %`
2. 修改為想要的版本顯示
3. 存檔

---

## ⚠️ 重要提醒

- **必須先脫殼**才能使用 STRedit！
- 修改前**務必備份**！
- 部分版本需要不同的處理方式
