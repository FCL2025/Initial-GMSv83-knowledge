# 2.2 Hex 編輯器修改

---

## 🛠️ 必備工具

| 工具 | 用途 |
|------|------|
| **HxD** | 免費 hex 編輯器 |
| **Hex Workshop** | 付費 hex 編輯器，功能更強 |

---

## 📝 修改步驟

### Step 1：開啟檔案

1. 下載並開啟 HxD 或 Hex Workshop
2. 拖曳 `MapleStory.exe` 到編輯器
3. 或使用 `File → Open`

### Step 2：搜尋 IP

1. 按下 `Ctrl + F` 開啟搜尋
2. 選擇 **Text** (文字搜尋)
3. 輸入 `127.0.0.1`（或要修改的 IP）
4. 點擊 **OK**

### Step 3：修改 IP

找到後，直接在文字上輸入新 IP：

```
原始：127.0.0.1 (13個字元)
新：  192.168.1.100 (15個字元)
```

### ⚠️ 注意事項

- **長度相同**是關鍵！
- 如果新 IP 比原 IP 長，需要用其他方法
- 建議使用 **STRedit** 處理長度不同的情況

---

## 💡 範例

### 範例：將 localhost 改為實際 IP

```
1. HxD 開啟 MapleStory.exe
2. 搜尋 "127.0.0.1"
3. 找到後，直接輸入你的 IP
   例如：192.168.1.100
4. 存檔 (Ctrl + S)
5. 完成！
```

---

## 🔍 進階技巧

### 如何找到正確的 IP？

有時候 IP 不是以 `127.0.0.1` 形式存在，可能是：
- 數字形式：`2130706433`（十六進位 `7F000001`）
- 需要用 **十六進位** 模式搜尋

### 十六進位計算

```
127.0.0.1 = 7F 00 00 01 (hex)
192.168.1.1 = C0 A8 00 01 (hex)
```
