# 4.1 Wireshark 抓包教學

---

## 🎯 目的

分析 MapleStory 網路封包，理解遊戲通訊協定的運作方式

---

## 🛠️ Wireshark 基本操作

### 安裝
1. 下載：https://www.wireshark.org/
2. 安裝時選擇 WinPcap

### 開始捕獲

1. 選擇網路介面卡
2. 點擊 **Start**
3. 執行遊戲動作
4. 點擊 **Stop**

---

## 🔍 過濾條件

### 基本過濾

| 過濾條件 | 說明 |
|----------|------|
| `tcp` | 只看 TCP 封包 |
| `tcp.port == 7575` |只看 port 7575 |
| `ip.addr == 192.168.1.1` |只看特定 IP |

### 組合過濾

```
tcp.port == 7575 && ip.addr == 你要監控的IP
```

---

## 📊 分析封包

### 查看內容

1. 點擊一個封包
2. 查看 **Packet Details**
3. 查看 **Packet Bytes**

### 找出Opcode

1. 找到 `COutPacket` 或 `CInPacket`
2. 分析封包結構
3. 記錄 Opcode 和資料格式

---

## 💡 範例：找出登入封包

```
1. 過濾：tcp.port == 7575
2. 在客戶端點擊登入
3. 找到發送的封包
4. 分析結構
```
