## 🚀 快速開始

1. **克隆此倉庫**
```bash
git clone https://github.com/FCL2025/Initial-GMSv83-knowledge.git
```

2. **用 Obsidian 開啟**
   - 下載並安裝 [Obsidian](https://obsidian.md/)
   - 開啟 `MapleStory-v83` 資料夾作為 Vault

3. **安裝建議主題與插件**
   - Theme: Minimal by Kepano
   - Fonts: iA Writer Quattro, Geist Mono
   - 7 大核心插件：Calendar, Hider, Kanban, Minimal Theme Settings, Outliner, Style Settings, Terminal
