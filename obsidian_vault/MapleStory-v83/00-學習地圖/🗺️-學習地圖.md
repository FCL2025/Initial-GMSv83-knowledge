## 🆕 2026-03-25 最新更新重點

### Cosmic v83 - 最推薦的开源模拟器
- **GitHub**: https://github.com/P0nk/Cosmic
- **Cosmic-client**: https://github.com/P0nk/Cosmic-client
- Java 21 + Docker 支持
- 提供完整架設文件和預設帳號（admin/admin）
- 長期目標：擺脫 custom WZ，使用 vanilla WZ

### 客戶端修改重點更新
- **Cosmic-client 安裝**：移除 HShield + MapleStory.exe + Patcher.exe + ASPLnchr.exe，複製 custom WZ + localhost exe
- **HxD IP 修改**：搜尋 `127.0.0.1`，三個連續出現，全部替換
- **Windows Defender**：修改後的客戶端會被標記為病毒，需加入排除清單
- **LAN 連線**：`127.0.0.1` → 伺服器 LAN IP，防火牆開啟 7575/8484

### BlackCipher NGS Bypass (2025-11)
- NGClient.dll side-loading 模擬技術
- 不需記憶體修補即可繞過 anticheat
- 來源：https://guidedhacking.com/threads/nexon-game-security-bypass-blackcipher-anticheat-emulation.21131/

### WZ 修改最新方法
- **lvmax 移除**（任務等級限制）：HaRepacker → Quest.wz → Classic XML → Regex `lvmax="\d+"\s*` → 留空
- **WZ 編輯黃金法則**：始終從 client WZ 匯出到 server XML，不要反向操作
- **GMS (old) 加密**：v83 WZ 編輯時必須選擇 `GMS (old)` 格式

### Maple_Pshark 核心功能 (2026)
- Hook 加密函數，直接截獲明文（繞過 AES/Shanda）
- localhost 直接使用，無需 proxy
- 封包修改 + 注入 + 規則過濾
- GitHub: https://github.com/obstriker/Maple_Pshark

### NPC腳本源間相容性 (2026-03)
- **Cosmic**：Nashorn (JDK 8+)，移除了 `importPackage`
- **最佳實踐**：使用 `Packages.xxx` 完整路徑，避免 `importPackage`
- NPCConversationManager.java: https://github.com/ronancpl/HeavenMS/blob/master/src/scripting/npc/NPCConversationManager.java

### MapleRef Wiki Send/Recv Opcode 分離原則
- S→C (Send)：伺服器發送到客戶端
- C→S (Recv)：客戶端發送到伺服器
- 同一個數字可能代表不同含義，視方向而定！
- https://mapleref.fandom.com/wiki/Operation_Code
