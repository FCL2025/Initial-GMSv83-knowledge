# 4.5 MapleProxy — 動態封包編輯器

---

## 概述

MapleProxy 是一款專為 MapleStory v83 設計的動態封包編輯工具，允許即時修改來往於客戶端和伺服器之間的封包。

**來源**: https://forum.ragezone.com/threads/mapleproxy-v83.1111000/

**發布時間**: 2016年（RaGEZONE），2026年仍然實用

---

## 核心功能

| 功能 | 說明 |
|------|------|
| **動態封包修改** | 不同於靜態注入器，MapleProxy 可在傳輸過程中修改封包 |
| **封包轉發** | 可將流量轉發到其他伺服器（類似 BungeeCord for Minecraft） |
| **多人伺服器支援** | 連接多個伺服器實現負載均衡 |

---

## 與其他工具的比較

| 工具 | MapleProxy | Maple_Pshark | 靜態封包注入器 |
|------|-----------|-------------|--------------|
| 封包修改時機 | **傳輸中** | 傳輸後 Hook | 預先注入 |
| 封包轉發 | ✅ | ❌ | ❌ |
| 多人伺服器 | ✅ | ❌ | ❌ |
| 學習難度 | 中等 | 簡單 | 簡單 |

## 使用方式

### 基本語法
```
localhost.exe [目標IP] [目標端口]
```

### 範例
```
localhost.exe 127.0.0.1 8480
```

**參數說明**：
- `localhost.exe`：MapleProxy 的執行檔名稱
- `127.0.0.1`：目標伺服器 IP
- `8480`：目標伺服器端口

### 完整流程

```
1. 啟動 MapleStory 私人伺服器（監聽 127.0.0.1:8484）
2. 啟動 MapleProxy（指定轉發到 127.0.0.1:8484）
3. 啟動 MapleStory 客戶端（連線到 localhost:8480）
4. MapleProxy 攔截並可修改所有來往封包
```

## 動態封包修改流程

### 攔截封包
```
1. 在 MapleProxy 中執行特定遊戲操作
2. 觀察封包列表
3. 找到感興趣的封包（按 opcode 或方向篩選）
```

### 修改封包
```
1. 右鍵點擊封包
2. 選擇 "Edit Packet"
3. 修改 hex 值或解析後的內容
4. 確認發送
```

### 重新發送
```
1. 選擇已攔截的封包
2. 點擊 "Resend"
3. 封包會被發送到目標（客戶端或伺服器）
```

## 應用場景

### 1. 測試伺服器功能
- 修改封包內容測試邊界條件
- 繞過簡單的客戶端驗證
- 模擬異常封包

### 2. 開發新功能
- 快速驗證封包結構是否正確
- 測試客戶端對自訂封包的回應
- 調試伺服器和客戶端的溝通問題

### 3. 學習封包結構
- 即時觀察封包內容
- 配合 MapleRef Wiki 理解 opcode 含義
- 追蹤特定操作引發的封包序列

### 4. 多人伺服器架設
```
┌─────────────┐
│   客戶端    │
└──────┬──────┘
       │ :8480
       ↓
┌─────────────┐
│ MapleProxy  │ ← 可修改封包
└──────┬──────┘
       │ :8484
       ↓
┌─────────────┐     ┌─────────────┐
│ Server 1    │     │ Server 2    │
│ (頻道1)     │ ←→  │ (頻道2)     │
└─────────────┘     └─────────────┘
```

## 局限性

| 局限性 | 說明 |
|--------|------|
| 僅支援 v83 | 不相容其他版本 |
| 需要注入/Loader | 需要正確的啟動方式 |
| 不支援加密移除 | 無法解密加密封包 |

---

## 與 Maple_Pshark 的選擇

| 場景 | 推薦工具 |
|------|---------|
| 僅需觀察封包 | Maple_Pshark |
| 需要修改並重新發送 | MapleProxy |
| 需要封包轉發/負載均衡 | MapleProxy |
| 快速 Hook 加密函數 | Maple_Pshark |
| IDA + 離線分析 | MaplePacketPuller |

---

## 注意事項

- MapleProxy 僅供學習和研究用途
- 使用封包修改可能違反遊戲服務條款
- 請勿用於作弊或破壞遊戲平衡

### Mob.wz 怪物修改（原文重點）

#### GodMode (Miss) - 怪物命中率設 0
```
工具：deMSwZ
步驟：
1. 開啟 mob.wz
2. 取消勾選所有項目（包含 view only）
3. 只勾選 acc
4. 將 acc 值設為 0
5. Process All → Save
```

#### Monster Speed 126
```
工具：HaRepacker
步驟：
1. 開啟 mob.wz
2. 找到怪物 ID
3. 點擊 [+] info
4. 選擇 speed，將值改為 126
5. Save
```

#### No Jump（跳躍怪物失效）
```
工具：HaRepacker
步驟：
1. 開啟跳躍怪物 ID
2. 移除 "jump"
3. Save
```
