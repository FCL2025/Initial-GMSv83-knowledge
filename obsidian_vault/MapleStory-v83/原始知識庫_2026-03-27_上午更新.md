## 🆕 MapleClientEditTemplate — 通用 MapleStory 客戶端編輯框架 (GitHub 2025-2026)

**GitHub**: https://github.com/MinimumDelta/MapleClientEditTemplate

**作者**: Erik A (Minimum Delta)

### 核心定位
> "An increasingly generic and comprehensive MapleStory client editing framework. Intended to abstract away some of the MapleStory client editing learning curve."

這是一個**通用客戶端編輯框架**，旨在簡化 MapleStory 客戶端修改的學習曲線。

### 技術架構

| 層次 | 說明 |
|------|------|
| **GenericLauncher.exe** | 啟動 MapleStory 並注入 DLL |
| **LEN.dll** | 核心修改 DLL，包含所有記憶體編輯和 Hook |
| **Common/** | Windows API Hooks（非版本特定） |
| **Maple Templates** | MapleStory 函數 Hook 模板 |

### Windows Hooks 功能（Common/）

這些 Hook 不需要等待客戶端脫殼，注入後立即可用：

| Hook | 功能 |
|------|------|
| `CreateMutexA` | 客戶端啟動觸發點 |
| `WinSock (WS2_32)` | 網路攔截（IP 修改用） |
| `Kernel32.dll` | 記憶體讀寫 |

### MapleStory 函數 Hook（版本特定）

這些 Hook 需要等待客戶端脫殼後才能使用：

| Hook | 功能 |
|------|------|
| `SendPacket` | 發送封包攔截 |
| `RecvPacket` | 接收封包攔截 |
| `CClientSocket` | 客戶端網路類 |

### 與舊版 localhost enabler 的差異

| 功能 | MapleClientEditTemplate | 舊版 localhost enabler |
|------|------------------------|----------------------|
| IP 修改 | ✅ WinSock Hook | ✅ 直接修改記憶體 |
| 封包攔截 | ✅ | ❌ |
| 反作弊 | 可配置開關 | 固定 |
| 學習曲線 | 較低（有模板） | 較高 |
| 客製化程度 | 高（模板系統） | 低 |

### 使用流程（v83 為例）

```
Step 1: 編譯專案
  → 需安裝 detours.lib 到 Debug folder
  → 設定 VS 專案屬性：$(SolutionDir)Common;%(AdditionalIncludeDirectories)
  → 編譯為 x86（x64 不相容）

Step 2: 配置 Common/Common.h
  → 開關 WinSock Hook（v83 可關閉）
  → 開關 instaject（立即注入 vs 等 CreateMutexA）
  → 設定目標 IP 和 Port

Step 3: 使用 GenericLauncher 啟動
  → 自動啟動 MapleStory 並注入 LEN.dll

Step 4: Debug 輸出
  → 使用 DebugView 查看輸出日誌
  → 網址：https://docs.microsoft.com/en-us/sysinternals/downloads/debugview
```

### MapleClientEditTemplate 的設計優點

1. **抽象化 Hook 系統**：封裝 Windows API Hook，降低學習門檻
2. **模板系統**：提供 MapleStory 函數 Hook 模板，可直接使用或修改
3. **開關可控**：每個功能都有開關，可針對不同版本配置
4. **調試友好**：使用 DebugView 直接輸出日誌

### 教學影片
**YouTube**: https://www.youtube.com/watch?v=Yr5IOhc5sPk

### 與 MapleEzorsia-v2 的比較

| 特性 | MapleClientEditTemplate | MapleEzorsia-v2 |
|------|------------------------|------------------|
| 用途 | 通用編輯框架 | localhost enabler + DLL |
| 客製化 | 高（模板系統） | 中（配置為主） |
| 學習曲線 | 中等 | 較低 |
| 反作弊處理 | 手動配置 | 內建 |
| 多人模式支援 | 需額外配置 | 內建修復 |

### 常見問題

**Q: 編譯錯誤 (detours.lib not found)**
```
解決：下載 detours.lib 並放到 Debug folder
```

**Q: x64 編譯失敗**
```
解決：切換到 x86 (Debug x86)
```

**Q: 如何自訂 IP 修改？**
```
在 Common/Common.h 中設定：
MAPLE_REDIRECT_IP = "192.168.1.1"
MAPLE_REDIRECT_PORT = 8484
```

## 🆕 完整學習路徑建議（2026-03 更新）

### Level 1: 入門（1-2 週）
1. **架設私人伺服器**
   - 使用 MapleSolaxiaV2 或 HeavenMS
   - 完成基礎架設，理解客戶端-伺服器通訊
   - 練習 Hex 修改客戶端 IP

2. **WZ 編輯基礎**
   - HaRepacker 基本操作
   - 編輯 NPC 名稱、屬性
   - 編輯怪物屬性（HP、攻擊力）

### Level 2: 進階（2-4 週）
3. **封包分析**
   - Wireshark 抓包基礎
   - Maple_Pshark 使用
   - 理解 MapleCrypto 加密

4. **NPC 腳本編寫**
   - JavaScript 狀態機基礎
   - 編寫簡單商店 NPC
   - 編寫任務 NPC

### Level 3: 高級（1-2 個月）
5. **客戶端修改**
   - IDA 分析基礎
   - 使用 MapleClientEditTemplate
   - 自訂封包處理

6. **Opcode 分析**
   - IDA 靜態分析
   - 提取封包結構
   - 對照開源源碼
