### 2.4 localhost.exe IP 修改（Hex 編輯法）

```
目的：將 127.0.0.1 替換為自定義 IP（如 Hamachi IP）

工具：Neo Hex Editor

Step 1: 以十六進位模式搜尋 "127.0.0.1"（ASCII → Hex: 31 32 37 2E 30 2E 30 2E 31）
Step 2: 找到後在同位置寫入新 IP（如 25.34.56.78）
Step 3: 確認長度一致（IP 字串長度需相同，否則會錯位）
Step 4: 儲存並離開

⚠️ 注意：修改後的 IP 字串長度必須與原始 "127.0.0.1"（9字元）相同
```

## 三、Opcode 結構深度解析

### 3.1 v83 Opcode 基本原理

```
封包結構：
 [4 bytes Header][2 bytes Opcode][N bytes Data]

Header = MapleCrypto 混淆後的長度 + opcode
Data   = 加密的實際內容

第一個封包（連線初始）為明文（未加密）
```
