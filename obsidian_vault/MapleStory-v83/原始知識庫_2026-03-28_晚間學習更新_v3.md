## 3. Opcode 結構

### v83 Opcode 格式
- **Header**: 4 bytes (MapleCrypto)
- **Opcode**: 2 bytes (little endian)
- **Data**: 可變長度

### MapleCrypto 頭部混淆流程

**編碼（發送）：**
```
Step 1: verify = (opcode ^ iv[0]) + (iv[1] << 8)
Step 2: verify = (verify << 3) | (verify >> 13)  (rotate left 3)
Step 3: verify = verify & 0xFFFF
```

**解碼（接收）：**
```
Step 1: verify = (header >> 3) | ((header & 0x7) << 13)
Step 2: opcode = (verify - (iv[1] << 8)) ^ iv[0]
```

### AES-OFB 加密特點
- 兩個獨立 Cipher（客戶端→伺服器、伺服器→客戶端）
- 各有專屬 IV
- IV 每次發包後更新
- 使用 256-bit AES

### 重要 Opcode 速查表

| Opcode (hex) | 名稱 | 方向 | 說明 |
|--------------|------|------|------|
| 0x01 | LOGIN_PASSWORD | C→S | 登入請求 |
| 0x0D | HANDLE_PACKET | S→C | 伺服器回應 |
| 0x13 | CHANNEL_SELECT | C→S | 頻道選擇 |
| 0x14 | CHARACTER_SELECT | C→S | 角色選擇 |
| 0x2E | MOVE_PLAYER | C→S | 玩家移動 |
| 0x37 | FACE_EXPRESSION | C→S | 表情動作 |
| 0x4A | USE_ITEM | C→S | 使用物品 |
| 0x5A | NPC_TALK | C→S | NPC 對話 |
| 0x7B | DROP_ITEM | C→S | 丟棄物品 |
| 0x9C | WARP_TO_MAP | S→C | 地圖傳送 |
| 0xA8 | MAP_CRASH | S→C | 地圖崩潰 |

## 5. NPC 腳本編寫

### 腳本位置
```
source/scripts/npc/{npc_id}.js
source/scripts/npc/{npc_id}.java
```

### NPC ID 對照
- 9000000-9009999: 通用 NPC
- 9010000-9019999: 商店 NPC
- 9030000-9039999: 任務 NPC

### 動態商店腳本系統

RaGEZONE 會員開發的 SQL-free 方案：
- 不需重啟伺服器即可更新商店
- 使用 JSON/腳本定義商品
- 可動態讀取玩家資料

### 基本 NPC 腳本範例
```javascript
var status = 0;

function start() {
    status = 0;
    cm.sendSimple("歡迎來到#b自由市場#k！\r\n#b#L0#購買道具#l\r\n#L1#出售道具#l");
}

function action(mode, type, selection) {
    if (mode == -1 || (mode == 0 && status == 1)) {
        cm.dispose();
        return;
    }
    
    if (mode == 1) status++;
    else status--;
    
    if (status == 0) {
        // 初始對話
    } else if (status == 1) {
        if (selection == 0) {
            cm.sendSimple("請選擇：\r\n#L0#火inct蓓#l\r\n#L1#水瓶#l");
        } else {
            cm.dispose();
        }
    }
}
```

### 常用 API
| 函數 | 說明 |
|------|------|
| cm.sendNext(text) | 下一頁對話 |
| cm.sendYesNo(text) | 是/否選項 |
| cm.sendSimple(text) | 簡單選項選單 |
| cm.gainItem(id, qty) | 給予物品 |
| cm.gainMeso(amt) | 給予金幣 |
| cm.getPlayer() | 取得玩家物件 |
| cm.startQuest(id) | 開始任務 |
| cm.completeQuest(id) | 完成任務 |

### NPC 除錯技巧
1. `@npc {id}` - GM 命令直接召喚 NPC
2. 查看伺服器日誌
3. 確認 NPC ID 與檔名完全匹配
4. `@reloadnpc` - 重新載入腳本
