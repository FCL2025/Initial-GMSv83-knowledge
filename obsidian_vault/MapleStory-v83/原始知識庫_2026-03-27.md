## 一、MapleSolaxiaV2 完整架設指南（新發現）

**來源**: https://github.com/skhzhang/MapleSolaxiaV2

### 1.1 環境需求總表
| 軟體 | 版本 | 用途 |
|------|------|------|
| JDK/JRE | 7 (Java 7) | 伺服器運行 |
| NetBeans | 8.0.2 | 開發IDE |
| WampServer | 2.0i | MySQL資料庫 |
| MySQL Workbench | 5.2.39 | 資料庫管理 |
| Hamachi | - | 網路隧道（可選）|
| JCE Unlimited Policy | Unlimited Strength | AES-256 加密支援 |

**JCE Policy 替換位置**:
```

C:\Program Files\Java\jdk1.7.0_01\jre\lib\ext
C:\Program Files\Java\jdk1.7.0_01\jre\lib\security
```

### 1.2 SQL 安裝流程
```
1. File > Open Script > db_database.sql → 執行
2. File > Open Script > db_drops.sql → 執行（可選：新增NPC商店物品）
3. (可選) File > Open Script > db_shopupdate.sql → 執行
```

### 1.3 configuration.ini IP 配置
- 設定為 `localhost` 可只用本機
- 使用 Hamachi IP 可讓其他玩家連線
- 或使用 port-forwarding（非 Hamachi 方法）

### 1.4 客戶端完整架設步驟（新發現）
```
1. 使用 ManagerMsv83.exe 安裝 MapleStory 到 C:\Nexon\MapleStory
2. 刪除: HShield資料夾、ASPLauncher.exe、MapleStory.exe、patcher.exe
3. 解壓 localhost.exe 到客戶端資料夾
4. 用修改過的 WZ 檔案覆蓋（client_wz 資料夾）
5. 如需自訂IP：
   - 用 Neo Hex Editor 開啟 localhost.exe
   - 搜尋 "127.0.0.1" 
   - 替換為目標IP（三個位置都要改）
```

### 4.2 WZ 加密機制（mapleref.fandom.com）
```
ASCII: char ^ (0xAA + offset) ^ key[offset]
Unicode: char ^ (0xAAAA + offset) ^ ((key[offset * 2 + 1]) << 8) + key[offset * 2]
```

## 五、Opcode 結構分析

### 5.1 v83 Opcode 特點
- Opcode 為**第一個 2 bytes**（早期版本為 1 byte）
- 最大可用 Opcode: 65,536 (2^16)
- 兩組不同Opcode：**Send (C→S)** 和 **Recv (S→C)**

### 5.2 IDA 分析技巧
所有客戶端Recv封包都經過:
```
CClientSocket::ProcessPacket → CField::OnPacket / CLogin::OnPacket
```
因此找 packet handler 很容易。

### 5.3 MapleCrypto 頭部混淆演算法
```python
# 加密端：對 opcode 進行混淆
def maple_encrypt_header(opcode, iv):
    verify = (opcode ^ iv[0]) + (iv[1] << 8)
    verify = (verify << 3) | (verify >> 5)  # 位旋轉 3 位
    return verify & 0xFFFF

# 解密端：反向操作
def maple_decrypt_header(encrypted, iv):
    verify = encrypted
    verify = (verify >> 3) | ((verify << 5) & 0xFFFF)  # 反向旋轉
    return (verify - (iv[1] << 8)) ^ iv[0]
```
