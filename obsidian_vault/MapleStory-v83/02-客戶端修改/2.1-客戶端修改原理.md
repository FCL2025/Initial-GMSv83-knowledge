# 2.1 客戶端修改原理

---

## 🎯 為什麼要修改客戶端？

MapleStory 客戶端出廠時設定為連接到**官方伺服器**：

```
正常流程：
客戶端 ──→ 官方伺服器 (login.maplestory.nexon.com:7575)
                    ↑
              這是我們要改掉的！
```

為了讓客戶端連接到**我們的私人伺服器**，必須修改客戶端內的 IP 位址。

---

## 🔍 修改的基本原理

### IP 位址在哪裡？

客戶端中儲存了官方伺服器的 IP 位址，通常以 **字串** 形式存在：

```
原始內容：[127.0.0.1] 或 [203.78.xx.xx]
         ↓
修改後：[你的伺服器 IP 或網域]
```

### 修改位置

| 位置 | 說明 |
|------|------|
| `MapleStory.exe` | 主執行檔，包含連線 IP |
| `login.cfg` | 某些版本有獨立的設定檔 |

---

## 🌐 連線流程解析

```
1. 啟動客戶端
2. 客戶端讀取設定的 IP (如 127.0.0.1)
3. 嘗試連接到該 IP 的 port 7575
4. 連線成功 → 進入登入畫面
5. 發送帳密到伺服器認證
```

---

## ⚠️ 常見問題

### Q: 修改後無法連接？
- 檢查 IP 是否正確
- 確認伺服器已啟動
- 檢查防火牆是否阻擋

### Q: 多個 IP 要修改？
- 通常只需修改**第一個**出現的 IP
- 其餘 IP 可能是備用或內網 IP

## 🆕 Cosmic-Client 完整客戶端設定指南 (官方, 2025-2026)

**來源**：https://github.com/P0nk/Cosmic-client

Cosmic 是目前最活躍的 v83 模擬器，其客戶端設定方式已形成標準流程：

### 事前準備
- MapleGlobal-v83-setup.exe（原廠安裝程式，2008年代）
- HeavenMS-localhost-WINDOW.exe（即修改過的客戶端）
- cosmic-wz（自訂 WZ 檔案）

### 設定步驟（官方流程）
```
1. 執行 MapleGlobal-v83-setup.exe，選擇安裝目錄（預設 C:\Nexon\MapleStory）
2. 移除 HShield 目錄（整個刪除）
3. 刪除以下檔案：
   - ASPLauncher.exe
   - MapleStory.exe
   - Patcher.exe
4. 複製 cosmic-wz 中的所有 WZ 檔案到安裝目錄並覆蓋
5. 複製 localhost.exe 到安裝目錄
6. 雙擊 localhost.exe 啟動客戶端
```

### ⚠️ 病毒誤判問題
```
Windows Security 會誤判修改過的客戶端為病毒
解決方案：在「病毒與威脅防護設定」中將下載和安裝目錄加入排除清單
```

### HxD IP 修改步驟（官方說明）
```
1. 下載並安裝 HxD：https://mh-nexus.de/en/hxd/
2. 用 HxD 開啟客戶端 exe（建議先複製一份備份）
3. 按 Ctrl+F 搜尋文字字串 "127.0.0.1"
4. 會找到三個相連的 IP 位址
5. 分別覆蓋為目標 IP 位址
6. 儲存
```

### 客戶端疑難排解
| 問題 | 解決方案 |
|------|---------|
| 客戶端無法啟動 | 狂按雙擊客戶端，有機會成功開啟 |
| 看到「無法連線到登入伺服器」 | 確認伺服器已啟動 |
| Windows 刪除客戶端exe | 將目錄加入 Windows Security 排除清單 |
| 顯示舊版 patcher 畫面 | 需使用與音源匹配的 localhost 客戶端版本（RaGEZONE 2026-01） |

## 🆕 DLL 注入式客戶端修改 - MapleEzorsia (2023-2026 新方案)

**來源**：https://github.com/izarooni/MapleEzorsia

傳統的客戶端修改需要 Hex 編輯或脫殼，但現在有一種更優雅的方式：**DLL 記憶體注入**。

### 核心技術：Detours API 攔截

MapleEzorsia 使用 Microsoft 的 **Detours** 庫來攔截 Windows API 和遊戲函數呼叫，無需修改原始 .exe 檔案。

**優點**：
- 不需要脫殼
- 不修改原始二進位檔
- 可卸載/更新修改，不破壞原客戶端
- 可配置（config.ini）

**安裝方式**：
```
v83 MapleStory 目錄：
1. 將 nmconew.dll → 改名為 nmconew2.dll
2. 將 MapleEzorsia.dll 放入目錄
3. 可選：建立 config.ini 配置檔
4. 執行 MapleStory.exe
```

**config.ini 功能表**：
```ini
[general]
width=1280       ; 遊戲解析度
height=720
discord=false    ; 是否啟用 Discord RPC
debug=false       ; 是否顯示除錯主控台
```

**提供的功能**：

| 功能 | 說明 |
|------|------|
| 自訂解析度 | 修改遊戲視窗和畫布解析度 |
| 視窗置中 | 啟動時自動置中 |
| 聊天紀錄擴展 | 64 行 → 127 行 |
| 去除登入書本邊框 | 高解析度下自動隱藏 |
| Fear 效果修復 | LimitedView 在任何解析度都正常 |
| 啟用最小化按鈕 | 原本被隱藏的功能 |
| 跳過開機動畫 | 節省啟動時間 |
| Discord RPC | 遊戲狀態顯示在 Discord |

### 與傳統 Hex 修改的比較

| 方法 | 優點 | 缺點 |
|------|------|------|
| Hex 編輯 | 簡單直接 | 需脫殼，破壞原檔 |
| Themida 脫殼 | 可完整修改 | 繁瑣，容易出錯 |
| **DLL 注入 (Ezorsia)** | **不破壞原檔，可配置** | 需 x86 編譯環境 |
