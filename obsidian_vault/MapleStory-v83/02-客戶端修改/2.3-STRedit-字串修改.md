# 2.3 STRedit 字串修改

---

## 🎯 STRedit 是什麼？

STRedit 是一個專門用於修改 MapleStory 客戶端字串的工具，支援：
- 修改長度不同的字串
- 編輯 UI 文字
- 修改版本顯示

---

## 📥 下載與安裝

1. 從 RaGEZONE 論壇下載 STRedit
2. 下載 `jieziban.rar`（韓文輸入工具）
3. 解壓縮到同資料夾

---

## 📝 使用步驟

### 修改字串

```
1. 執行 STRedit.exe
2. File → Open → 選擇脫殼後的 MapleStory.exe
3. 搜尋要修改的文字（如 "beginner"）
4. 選中後直接輸入新文字（如 "新手"）
5. 存檔
```

### 修改版本號

1. 搜尋 `ver. %`
2. 修改為想要的版本顯示
3. 存檔

---

## ⚠️ STREDIT v118+ 提示處理

使用 STREDIT 開啟客戶端時，軟體會詢問是否為 v118+ 版本：

### v83 客戶端
- **按「否 (No)」**
- 然後選擇「IP 搜尋 (IP Search)」
- 點擊「強制修改 IP (Force to a IP)」按鈕
- 輸入目標 IP 地址

### 操作流程
```
1. 執行 STRedit.exe
2. File → Open → 選擇 localhost.exe
3. 當詢問「Is this v118+?」時 → 選擇「否 (No)」
4. 當詢問「IP Search?」時 → 選擇「是 (Yes)」
5. 點擊「Force to a IP」按鈕
6. 輸入你的伺服器 IP
7. 存檔
```

### STREDIT 功能速查
| 功能 | 說明 |
|---|---|
| Force to a IP | 強制修改為指定 IP |
| AD URL | 修改廣告 URL |
| Forgot ID/Password URLs | 修改密碼找回 URL |
| Version String | 修改版本顯示字串 |

## 🔧 IP 批量修改方法

如果有多個 IP 字串需要修改（如 127.0.0.1 出現多次）：

### HxD Hex 編輯法
1. 用 HxD 開啟 `localhost.exe`
2. `Ctrl+F` 搜尋 `127.0.0.1`
3. 確認長度一致後直接替換（保持位元組長度相同）
4. 存檔

### ⚠️ 注意事項
- **長度必須相同**：127.0.0.1 是 9 位元組，替換 IP 也必須是 9 位元組
- 不足時可用 `0x00` 填充，但建議使用長度相同的 IP
- 例如：`192.168.1.1` (13 bytes) 不能直接替換 `127.0.0.1` (9 bytes)
