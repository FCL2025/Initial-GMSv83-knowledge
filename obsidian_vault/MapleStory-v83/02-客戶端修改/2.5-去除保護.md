## 🆕 BlackCipher / NGS Anticheat Bypass (2025-11 新方法)

**來源**: https://guidedhacking.com/threads/nexon-game-security-bypass-blackcipher-anticheat-emulation.21131/

### 繞過原理
MapleStory 使用的 Nexon Game Security (NGS, 原名 BlackCipher) 透過心跳（heartbeat）機制驗證客戶端完整性。

**繞過方法**:
1. 使用假的 `NGClient.dll` 進行 **side-loading**
2. 假的 DLL 攔截伺服器的 heartbeat 請求
3. 模擬合法的安全客戶端回應
4. 結果：完全繞過 anticheat，不再擔心客戶端偵測

### 技術特點
| 項目 | 說明 |
|------|------|
| 不需要記憶體修補 | 安全合法 |
| 不需要移除完整性檢查 | 模擬而非破壞 |
| 模擬安全客戶端行為 | 可用於封包分析、記憶體修改等研究 |

### 與早期 HS Bypass 的區別
| 方案 | 原理 | 適用版本 |
|------|------|----------|
| HS Bypass (早期) | DLL 注入，繞過 HackShield | 舊版 MapleStory |
| NGS Bypass (新) | NGClient.dll side-loading，模擬 heartbeat | v83+ (含 BlackCipher) |

## 🆕 fonts folder 問題修復 (RaGEZONE, 2026-03)

**問題徵兆**:
- 客戶端啟動後文字無法顯示
- 只有背景圖片和框架
- UI 文字全部變成方塊或空白

**根本原因**:
- 缺少 fonts folder
- 字體檔案未正確複製
- 某些改過的客戶端移除了 fonts

**解決方案**:
```
Step 1: 從正常客戶端複製 fonts folder
Step 2: 粘貼到目標客戶端目錄（MapleStory/fonts/）
Step 3: 確認 fonts folder 有讀取權限
Step 4: 若問題持續，從 MapleSolaxiaV2 資源包取得完整 fonts
```

**MapleSolaxiaV2 完整資源包**:
```
Google Drive: https://drive.google.com/drive/folders/0BzDsHSr-0V4MYVJ0TWIxd05hYUk
包含 v83 客戶端完整檔案 + HaRepacker + SQL 腳本 + fonts
```
