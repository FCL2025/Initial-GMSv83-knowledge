## 一、登入器/客戶端修改 (Launcher/Client Editing)

### 最新資源：AugurMS 案例研究
- **AugurMS** 是一個現代化的 v83 私人伺服器，使用 Electron 構建登入器
- GitHub: https://github.com/themrzmaster/augurms
- 特點：
  - 使用 Claude AI 作為 Game Master
  - 自定義 WZ 文件和客戶端補丁
  - 自動更新機制 (SHA256校驗)

### 客戶端修改流程 (v83)
1. **取得客戶端**：使用 v83 MapleStory 客戶端
2. **移除保護**：刪除 `HShield/`、`ASPLauncher.exe`、`MapleStory.exe`、`patcher.exe`
3. **替換為本地登入器**：`localhost.exe` 或自定義 `.exe`
4. **編輯WZ文件**：使用 HaRepacker-resurrected，加密選擇 **"GMS (old)"**
5. **HEX編輯IP**：如果伺服器IP不是 `127.0.0.1`，需要HEX編輯 `localhost.exe`

### 關鍵修改點
```
原版需要修改：
- MAPLE_VERSION (客戶端版本校驗)
- MAPLE_TYPE (伺服器類型)
- AES user key 和 IV transformation key
```

### 客戶端加密類型
- **v83 使用 AES加密** (SnowMS/MapleStory比較新版本)
- 早期版本使用 Shanda 自定義加密 (靜態，不依賴外部資訊)
- v118+ GMS 使用動態 AES key (每版本變化)

## 二、抓包分析 (Packet Capture Analysis)

### 推薦工具

#### 1. MaplePacketPuller (Python)
- GitHub: https://github.com/Bratah123/MaplePacketPuller
- 功能：
  - 分析 IDA 生成的偽代碼
  - 格式化為伺服器端的 encode/decode 格式
  - 顯示封包結構
  - 搜索 InHeader opcodes
- 環境需求：Python 3.6+、IDA Pro

#### 2. Maple_Pshark
- GitHub: https://github.com/obstriker/Maple_Pshark
- v83 專用封包攔截器
- 功能：
  - Hook 加密函數
  - 攔截 Recv/Send 訊息
  - 修改封包內容
  - 注入封包
  - 基於規則的封包過濾

#### 3. Wireshark
- 通用網路抓包工具
- 設置 TCP 追蹤和十六進制顯示
- 過濾：`tcp.port == 8484` (登入) 或 `tcp.port == 7575` (遊戲)

### 封包基礎知識
- **第一個發送的封包總是明文**（不加密）
- MapleStory 封包走 TCP socket
- 封包格式：**Little Endian**
- Opcode 是辨識碼，決定封包類型

## 三、Opcode 結構 (Opcode Structure)

### Opcode 基本結構
- **Opcode 位置**：封包的前2個字節（short/INT16）
- **Opcode 作用**：辨識封包類型（移動、攻擊、對話等）
- **版本差異**：
  - 早期版本：Opcode 是 1 byte
  - v83：Opcode 是 2 bytes (short)
  - 最大值：65,536 (2^16)

### 封包加密 (v83)
- **AES 加密**：使用 MapleAESOFB 類
- **自定義加密**：MapleCustomEncryption
- **密鑰配置**：
  ```java
  MapleAESOFB.java 包含：
  - AES user key
  - IV transformation key
  - 版本校驗 (MAPLE_VERSION)
  ```

### 封包格式
```
[4 bytes: Length][2 bytes: Opcode][Data...]
或
[4 bytes: Length][Opcode][Data...] (帶加密)
```

## 四、WZ 修改 (WZ Editing)

### 主要WZ文件
| WZ 文件 | 用途 |
|---------|------|
| `Item.wz` | 道具、裝備屬性 |
| `Mob.wz` | 怪物屬性（HP、MP、速度、技能、經驗值等） |
| `Skill.wz` | 技能效果、傷害、行數 |
| `Map.wz` | 地圖物件、登入畫面 |
| `Sound.wz` | 音效、BGM |
| `Character.wz` | 角色外觀 |

### WZ 編輯器
- **HaRepacker-resurrected** (推薦)
  - 支援 GMS old 加密
  - 開源專案
- **K rt (韓國開發者工具)**
- **GMS/WZEdit**

### Mob.wz 編輯重點
- `speed`: 移動速度
- `HP`/`MP`: 生命值/魔法值
- `EXP`: 經驗值
- `bodyAttack`: 碰撞傷害 (編輯後可無敵但攻擊會掉線)
- `jump`: 跳躍動畫

### Skill.wz 編輯
- `damage`: 技能傷害
- `hit`: 攻擊行數
- 需要匯編知識才能深度修改

### 錯誤處理
- **錯誤碼 -2147287038**: 通常是 `login.img` 格式問題
- **匯入新地圖**: 需要同時匯入 mobs、tiles、background 等

## 五、NPC 腳本編寫 (NPC Scripting)

### 腳本範例

#### 基本對話腳本
```javascript
function start() {
    cm.sendOk("歡迎來到我的商店！我有各種道具和裝備。");
    cm.dispose();
}
```

#### 選單腳本
```javascript
function start() {
    cm.sendSimple("歡迎來到我的商店！\r\n#L0#購買道具#l\r\n#L1#購買裝備#l");
}

function action(mode, type, selection) {
    if (mode == 1) {
        if (selection == 0) {
            cm.openShop(100);
        } else if (selection == 1) {
            cm.openShop(200);
        }
    }
    cm.dispose();
}
```

### 常用 NPC 腳本函數

| 函數 | 功能 |
|------|------|
| `cm.getPlayer()` | 取得當前玩家資訊 |
| `cm.getMap()` | 取得當前地圖資訊 |
| `cm.getParty()` | 取得隊伍資訊 |
| `cm.getMeso()` | 取得金錢數量 |
| `cm.gainMeso(x)` | 給予玩家金錢 |
| `cm.gainItem(itemId, quantity)` | 給予道具 |
| `cm.openShop(shopId)` | 開啟商店 |
| `cm.warp(mapId, portal)` | 傳送玩家 |
| `cm.dispose()` | 結束腳本 |

### 範例：隨機猜數遊戲
```javascript
var guessNumber = Math.floor(Math.random() * 100);
var playerGuess;

function start() {
    cm.sendNext("猜猜我想的是哪個數字? (1-100)");
    playerGuess = -1;
}

function action(mode, type, selection) {
    if (mode == 1) {
        playerGuess = selection;
        if (playerGuess == guessNumber) {
            cm.sendOk("恭喜你! 你猜對了! 數字是 " + guessNumber);
            cm.dispose();
        } else if (playerGuess < guessNumber) {
            cm.sendNext("你猜的數字太小了! 再試一次吧!");
        } else {
            cm.sendNext("你猜的數字太大了! 再試一次吧!");
        }
    }
}
```

3. 複製 client/cosmic-wz/ 中的 .wz 文件
4. 複製 AugurMS.exe
5. 啟動 AugurMS.exe

### Hex 編輯快速參考
- 127.0.0.1 的 Hex 表示：31 32 37 2E 30 2E 30 2E 31
- 端口 8484 的 Hex（小端）：A4 21
- 端口 7575 的 Hex（小端）：7C 1D
