# 5.7 最新開源工具與案例分析 (2026-03-29 更新)

## 開源專案推薦

### 1. AugurMS - AI 驅動的現代 v83 伺服器
- **GitHub**: https://github.com/themrzmaster/augurms
- **特點**：
  - 使用 Claude AI 作為 Game Master
  - Electron 登入器 + 自動更新
  - 自定義 WZ 和客戶端
  - 完整的管理面板

#### 架構
```
augurms/
├── server/          # Java (Cosmic v83) + Docker
├── dashboard/      # Next.js 15 + API
├── launcher/       # Electron (Windows)
├── client/         # 補丁客戶端 + WZ
└── mcp-server/     # MCP 工具伺服器
```

#### 部署方式
```bash
# 遊戲伺服器 (需要 Docker)
cd server && docker compose up

# 管理面板
cd dashboard && npm install && npm run dev

# 登入器
cd launcher && npm install && npm start -- --dev
```

### 2. MaplePacketPuller - IDA 封包結構分析
- **GitHub**: https://github.com/Bratah123/MaplePacketPuller
- **用途**：從 IDA Pro pseudocode 分析封包結構
- **環境需求**：
  - Python 3.6+
  - IDA Pro 7.0+

#### 使用流程
1. IDA Pro 反編譯客戶端
2. 複製函數 pseudocode 到 .txt
3. 放到 `Functions/` 目錄
4. 運行 `main.py`
5. 查看 `FuncOutput/` 輸出
