## 🆕 Opcode結構

### Opcode 核心概念
> "Actions that are defined by an identifier called an Operation Code (opcode for short), which is the first 2 bytes of the packet (a short)"

### Send vs Receive Opcode 區分
| 方向 | 說明 | Opcode 集合 |
|------|------|------------|
| **S→C (Send)** | 伺服器發送，客戶端接收 | Server Opcode Set |
| **C→S (Recv)** | 客戶端發送，伺服器接收 | Client Opcode Set |

**重要**：同一個 Opcode 數字可能代表不同含義，視方向而定！

### v83 Opcode 現況 (RaGEZONE, 2024-12)
> "v83 的 Opcode 幾乎全部被映射，不需要額外追蹤封包，唯一需要的反向工程是修改客戶端時"

### IDA 分析Opcode — 進階技巧
**來源**: 
- https://forum.ragezone.com/threads/getting-packet-structures-opcodes-using-ida.792436/
- https://forum.ragezone.com/threads/how-does-one-find-a-structure-of-a-packet-via-ida.1162654/

**分析關鍵**：
1. 找 `CClientSocket::ProcessPacket` — 客戶端接收封包的總入口
2. 從這裡根據 header 分發到 `CField::OnPacket` 或 `CLogin::OnPacket`
3. 這就是為什麼有 opcode 清單就能輕鬆找到客戶端接收封包的處理函數

### MapleCrypto Header 加密/解密公式 (v83 核心)
```java
// 加密端（伺服器/客戶端發送）
public byte[] getPacketHeader(int opcode, int length, int iv) {
    int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
    verify = (verify << 3) | (verify >>> 5);
    verify &= 0xFFFF;
    return new byte[] {
        (byte) (verify & 0xFF),
        (byte) ((verify >>> 8) & 0xFF),
        (byte) (length & 0xFF),
        (byte) ((length >>> 8) & 0xFF)
    };
}

// 解密端
public int retrieveOpcode(byte[] header, int iv) {
    int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
    verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
    return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
}
```

### CClientSocket 核心函數 IDA 位址表 (v83)
| 函數 | 位址 | 說明 |
|------|------|------|
| `CClientSocket::Close` | `0x00440B2A` | 關閉連接 |
| `CClientSocket::OnConnect` | `0x0043F9CD` | 連接建立 |
| `CClientSocket::OnRead` | `0x00440932` | 讀取網路資料 |
| `CClientSocket::ProcessPacket` | `0x00441181` | 封包分發處理 |
| `CClientSocket::SendPacket` | `0x00440B47` | 發送封包 |
| `CClientSocket::ManipulatePacket` | `0x0044104D` | 封包加密/解密處理 |

### NPC 腳本結構
```javascript
// 範例 NPC 腳本結構
var status = 0;

function start() {
    cm.sendNext("Hello, welcome to my shop!");
}

function action(mode, type, selection) {
    if (mode == 1) {
        status++;
    } else {
        cm.dispose();
        return;
    }
    
    if (status == 1) {
        cm.sendNextPrev("What would you like to buy?");
    } else if (status == 2) {
        cm.openShop(123); // 商店 ID
        cm.dispose();
    }
}
```
