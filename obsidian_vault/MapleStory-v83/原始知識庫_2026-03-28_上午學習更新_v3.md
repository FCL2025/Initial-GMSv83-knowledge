**Opcode 原文定義**：

## 3. Opcode 結構 — v83 封包通信底層

### 3.1 MapleCrypto 加密機制深入

**頭部格式 (4 bytes)**：
```
[2 bytes: 封包長度][2 bytes: 混淆後的 opcode/verify值]
```

**Opcode 混淆公式**：
```python
# 版本號處理
version_outgoing = -(version + 1)  # 發送到客戶端
version_incoming = version         # 發送到伺服器

# verify 值計算
verify = (opcode ^ iv[0]) + (iv[1] << 8)
verify = (verify << 3) | (verify >> 13)  # rotate left 3 bits
```

**Little Endian 格式注意**：
> "when that value is put in Little Endian format it becomes 30 11 00 00."

### 3.2 CInPacket 讀取函數 (IDA)

| IDA 函數 | MaplePacketWriter 方法 | 讀取大小 |
|---------|----------------------|---------|
| `CInPacket::Decode1` | `write` | 1 byte |
| `CInPacket::Decode2` | `writeShort` | 2 bytes |
| `CInPacket::Decode4` | `writeInt` | 4 bytes |
| `CInPacket::DecodeBuffer` | `writeLong` | 8 bytes |
| `CInPacket::DecodeStr` | `writeMapleAsciiString` | 可變 |

### 3.3 CClientSocket::ProcessPacket 分發流程

```
CClientSocket::ProcessPacket
    ↓
根據 opcode header 分發到:
    - CLogin::OnPacket (登入 opcode)
    - CWvsContext::OnPacket (warp2map 之前的 opcode)
    - CField::OnPacket (遊戲中所有其餘 opcode)
```

### 3.4 MapleLib — Go 語言 MapleStory 工具庫

**GitHub**: https://github.com/Francesco149/maplelib

**封包加密範例（Go）**：
```go
// 初始化 MapleCrypto (v62~v83)
initVector := [4]byte{}
rand.Read(initVector[:])
crypt := maplelib.NewCrypt(initVector, 62)  // version=62

// 建構封包
p := maplelib.NewPacket()
p.Encode4(0x00000000)  // 加密頭部預留
p.Encode2(255)
p.EncodeString("Hello world!")
p.Encode4s(-5000)

// 加密
crypt.Encrypt([]byte(p))

// 解密（跳過前4位元組頭部）
p = p[4:]
crypt.Decrypt([]byte(p))
```
