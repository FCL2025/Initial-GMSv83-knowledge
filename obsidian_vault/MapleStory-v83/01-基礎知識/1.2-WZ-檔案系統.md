# 1.2 WZ 檔案系統

---

## 📦 WZ 是什麼？

WZ 是 MapleStory 使用的**遊戲資源檔案格式**，類似於 XML 但經過加密和壓縮。

```
MapleStory.exe ←→ WZ 檔案 ←→ 遊戲資源（地圖、怪物、道具、腳本等）
```

---

## 📁 常見 WZ 檔案

| 檔案 | 內容 |
|------|------|
| `Map.wz` | 地圖資料 |
| `Mob.wz` | 怪物資料 |
| `Npc.wz` | NPC 資料 |
| `Item.wz` | 道具資料 |
| `Skill.wz` | 技能資料 |
| `String.wz` | 字串顯示 |
| `UI.wz` | 介面資源 |
| `Sound.wz` | 音效 |

---

## 🔧 WZ 編輯工具

### 主要工具：HaRepacker
- 用途：打包/解包 WZ 檔案
- 支援版本修改
- 官方下載：RaGEZONE 論壇

### 使用範例：版本修改

```
1. 用 HaRepacker 打開 Map.wz
2. 修改版本號（如 98 → 83）
3. 存檔
4. 完成！
```

### ⚠️ 注意事項

- **Skill.wz** 在 v94+ 有結構變化，可能導致斷線
- **UI.wz** 只需修改部分模組（如 UIWindow.img）
- 修改前**務必備份**原始檔案

---

## 📝 實作：修改 WZ 版本

### 情境：想把 v98 的 WZ 用在 v83 伺服器

1. **開啟 HaRepacker**
2. **File → Open → 選擇 Map.wz (v98)**
3. **修改版本號**（通常是檔案內的版本數值）
4. **另存新檔**為 Map.wz
5. **對其他 WZ 檔案重複同樣步驟**

### 可轉換的檔案
- ✅ Map.wz
- ✅ Mob.wz
- ✅ Item.wz
- ✅ String.wz
- ⚠️ UI.wz（部分）
- ❌ Skill.wz（不建議）
