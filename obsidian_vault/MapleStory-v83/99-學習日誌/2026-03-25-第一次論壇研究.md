## 📌 登入器/客戶端修改

### MapleSolaxiaV2 完整架設流程 (RaGEZONE, Jonnylin13)

**GitHub**: https://github.com/jonnylin13/Maple83

**完整架設步驟**（精簡版）：

```
1. 安裝 JDK 7 (jdk-7u79-windows-x64.exe)
2. 安裝 WampServer + MySQL Query Browser
3. 安裝 Hamachi（可選，用於對外連線）
4. 複製伺服器源到 C:\Nexon\MapleSolaxia
5. 執行 SQL 腳本：db_database.sql + db_drops.sql
6. 手動在 accounts 表建立第一個帳戶
7. 修改 configuration.ini 中的 IP
8. NetBeans 開啟專案，Clean and Build
9. 執行 launch.bat 啟動伺服器
```

**客戶端設定**：
```
1. 安裝 MapleGlobal-v83-setup.exe
2. 刪除 HShield 目錄、ASPLauncher.exe、MapleStory.exe、Patcher.exe
3. 覆蓋 WZ 檔案（client_wz）
4. 使用 localhost.exe 啟動客戶端
5. 若非本機，需用 HxD 修改 localhost.exe 中的 IP
```

**jce_policy 無限強度加密原則**：
```
覆蓋以下位置的 jars：
- C:\Program Files\Java\jre7\lib\security
- C:\Program Files\Java\jdk1.7.0_01\jre\lib\security
- 兩者的 lib/ext 目錄
```

**重要警告**：
> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate some kind of bugs afterwards.

### STREDIT vs HxD 客戶端 IP 修改

| 工具 | 適用情境 | 優點 |
|------|---------|------|
| **STREDIT** | 修改 localhost.exe | 自動處理 IP 長度，無需手動計算位元組 |
| **HxD** | 修改任何 exe | 可視化強，但需手動對齊位址 |

**STREDIT 操作流程**：
```
1. 開啟 STREDIT v2.9.2.0
2. 載入 localhost.exe
3. 詢問是否為 v118+ → 選「No」
4. 詢問 IP 搜尋 → 選「Yes」
5. 點擊「Force to a IP」，輸入目標 IP
6. 儲存並關閉
```

**HxD 操作流程**：
```
1. 用 HxD 開啟 localhost.exe（建議先複製備份）
2. 按 Ctrl+F 搜尋 "127.0.0.1"
3. 會找到三個相連的 IP 位址
4. 分別覆蓋為目標 IP 位址（注意位址對齊）
5. 儲存
```

## 📌 Opcode 結構

### MapleCrypto Header 加密/解密公式

**加密端（發送）**:
```java
public byte[] getPacketHeader(int opcode, int length, int iv) {
    int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
    verify = (verify << 3) | (verify >>> 5);
    verify &= 0xFFFF;
    return new byte[] {
        (byte) (verify & 0xFF),
        (byte) ((verify >>> 8) & 0xFF),
        (byte) (length & 0xFF),
        (byte) ((length >>> 8) & 0xFF)
    };
}
```

**解密端（接收）**:
```java
public int retrieveOpcode(byte[] header, int iv) {
    int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
    verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
    return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
}
```

### IDA Decode 函數對照表

| IDA 函數 | MaplePacketWriter 方法 | 讀取大小 |
|---------|----------------------|---------|
| `Decode1` | `mplew.write` | 1 byte |
| `Decode2` | `mplew.writeShort` | 2 bytes |
| `Decode4` | `mplew.writeInt` | 4 bytes |
| `DecodeStr` | `mplew.writeMapleAsciiString` | 可變 |

### CClientSocket::ProcessPacket 封包分發流程

```
CClientSocket::ProcessPacket()
    ↓
根據 MapleCrypto header 解密
    ↓
提取 opcode (2 bytes, little endian)
    ↓
switch(opcode)
    ↓
CLogin::OnPacket()    // 登入畫面
    或
CField::OnPacket()    // 遊戲中
```

### HaRepacker 導出 WZ 為 XML (RaGEZONE, 2023)

**來源**: https://forum.ragezone.com/threads/community-wz-img-repack-for-v83.1221835/

**步驟**:
```
1. 用 HaRepacker 打開 WZ 檔案
2. 選擇要導出的節點
3. 點擊 Export → 選擇 XML (Private Server) 格式
4. 導出後可使用文字編輯器修改
5. 重新導入 WZ
```

**注意**:
- v83 WZ 加密格式需選擇 `GMS (old)`
- 不要直接刪除/重新導入節點，可能造成數據丟失
- 建議使用 HaRepacker UI 直接編輯數值

### NX 格式 - WZ 的高效替代方案 (RaGEZONE, 2026-02)

**來源**：https://forum.ragezone.com/threads/wz-and-nx-file-questions.1260605/

**背景**：
- NX 格式是 WZ 的二進位替代方案
- 比 WZ/xml 性能更好（載入速度更快、檔案更小）
- 可從 WZ 轉換為 NX
- v83 主要仍使用 WZ 格式，NX 為較新方案

## 📌 NPC 腳本編寫

### 狀態機核心概念

```javascript
status = -1 → 初始調用 start()
    ↓
status = 0 → 第一次 sendNext/sendSimple
    ↓
status = 1 → 玩家回應後
    ↓
... 依此類推
```

### Mode 參數含義

| Mode 值 | 意義 |
|---------|------|
| 1 | 點擊"下一頁"或選項 |
| 0 | 點擊"返回" |
| -1 | 關閉對話框 (ESC/X) |

### NPC 對話框類型

| Type 值 | 對話框類型 | 使用方法 |
|---------|-----------|---------|
| 0 | 標準對話 | `sendNext`, `sendOk`, `sendPrev` |
| 1 | Yes/No | `sendYesNo` |
| 2 | 選項選單 | `sendSimple` |
| 3 | 數字輸入 | `sendGetNumber` |
| 12 | 接受/拒絕 | `sendAcceptDecline` |

### NPC 對話格式標籤

| 標籤 | 功能 |
|------|------|
| `#b` | 藍色文字 |
| `#r` | 紅色文字 |
| `#g` | 綠色文字 |
| `#l` | 選項標記 |
| `#i[itemid]` | 物品圖片 |
| `#h#` | 玩家名稱 |
| `#c[itemid]#` | 持有物品數量 |
| `#t[itemid]` | 物品名稱 |

### 腳本源相容性問題

| 問題 | 解決方案 |
|------|---------|
| `importPackage is not defined` | JDK 7+ 廢棄，移除該行 |
| `SavedLocationType is not defined` | 使用相容格式 |
| NPC 腳本不相容 | 使用普遍 `cm.` 函數 |

### 快速傳送 NPC 腳本源 (RaGEZONE, 2020)

```javascript
var status = 0;

function start() {
    cm.sendYesNo("嘿，你想傳送到 Ludibrium 嗎？");
}

function action(mode, type, selection) {
    status++;
    if (mode != 1) {
        if(mode == 0 && type != 1)
            status -= 2;
        else if(type == 1 || (mode == -1 && type != 1)) {
            if(mode == 0) {
                cm.sendOk("嗯... 看來你還有事要在這裡處理？");
                cm.dispose();
                return;
            }
        }
    }

    if (status == 1) {
        if (cm.haveItem(4310000))
            cm.sendNext("#b檢測到快速傳送票#k。\r\n看來你有這張票！");
        else
            cm.sendNext("當然可以！我帶你去 Ludibrium。");
    } else if (status == 2) {
        cm.warp(220000000);
        cm.dispose();
    }
}
```
