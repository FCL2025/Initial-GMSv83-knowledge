## 🆕 最新論壇資源挖掘 (2026-03-28)

### MapleEzorsia V2 最新動態 (GitHub)

**來源**: https://github.com/444Ro666/MapleEzorsia-v2

**Ezorsia V2 核心特點總結**:
- **第一個公開 v83 獨立 DLL 客戶端** (2023-11-16)
- 無需脫殼，適用於任何乾淨安裝的 v83 客戶端
- 無 WZ 衝突，生成獨立的 EzorsiaV2_UI.wz
- HD 解析度支援 (800x600 → 自定義)
- 可加載 3 個第三方 DLL
- 大量原廠 Bug 修復 (Cash Shop、Mulung Dojo UI 等)

**重要發現 - Admin 提權移除**:
- 問題：編輯 Admin elevation 後，客戶端仍要求管理員權限
- 根本原因：當執行檔命名為 `MapleStory.exe` 時，Windows UAC 會繞過自訂的 `asInvoker` 設定
- 解決方案：
  1. 使用 Hendi's Hex Editor 工具包編輯 `asInvoker` 部分
  2. 將執行檔改名為 `MapleStory2.exe` 或其他名稱
  3. Windows 就會正確應用 `asInvoker` 設定

**編譯平台**：
- 僅支援 x86 (32-bit)
- 不支援 x64

**Magpie 推薦搭配**:
- 將 1280x720 的遊戲畫面無損投射到全螢幕
- 避免全螢幕拉伸，保持 16:9 比例

### HeavenMS 專案深入分析 (2026-03-28)

**來源**: https://github.com/ronancpl/HeavenMS

**專案特點**:
- 基於 MapleSolaxiaV2 的改進版
- Java 8 建置
- 大量伺服器核心修改
- 完整 NPC 腳本源碼集合

**重要說明**:
> "In this project, many gameplay-wise issues generated from either the original WZ files and the server sources have been partially or completely solved. Considering the use of the provided edited WZ's and server-side wz.xml files should be of the greatest importance"

**架設流程摘要**:
1. 安裝 WampServer2.0i.exe (MySQL server)
2. 安裝 Java 8 SDK & NetBeans bundle
3. 建立資料庫：執行 db_database.sql, db_drops.sql
4. 設定 configuration.ini 中的 IP
5. 使用 NetBeans 建立專案並設定 Libraries
6. 執行 launch.bat 啟動伺服器

**localhost 客戶端設定**:
1. 安裝 MapleStory 後刪除：HShield資料夾、ASPLnchr.exe、MapleStory.exe、Patcher.exe
2. 放入修改後的 localhost.exe
3. 使用提供的 custom WZ 檔案替換原廠 WZ

**WZ 編輯重要提醒**:
- 使用 HaRepacker 4.2.4，encryption "GMS (old)"
- 客戶端 WZ 編輯後 → 必須用 "Private Server..." 匯出到伺服器的 wz.xml
- 保持 server-client 同步至關重要

### IntransigentMS 腳本專案 (ECMAScript 6)

**來源**: https://github.com/NoetherEmmy/intransigentms-scripts

**特點**:
- 使用 Nashorn 引擎執行 JavaScript
- 遵循 ECMAScript 6 標準
- 完整開源的 NPC 腳本源碼集合

**現代寫法範例**:
```javascript
// 使用 const 宣告
const MapleCharacter = Java.type("net.sf.odinms.client.MapleCharacter");

// 使用箭頭函數
arr.map(x => x * 2);

// 模板字串
`你的職業是 ${jobName}`;

// 除錯輸出
print("Debug message");
```

**與傳統寫法差異**:
| 特性 | 傳統寫法 | ECMAScript 6 寫法 |
|------|---------|------------------|
| 類型導入 | `importPackage(Packages.xxx)` | `const Xxx = Java.type("xxx")` |
| 除錯輸出 | `cm.dispose()` + `cm.sendNext()` | `print("debug")` |
| Lambda | 不支援 | `arr.map(x => x * 2)` |

## 重點學習筆記

### 登入器/客戶端修改要點

1. **Ezorsia V2 + Hendi's Client 組合** (2026-03 最佳方案)
   - dinput8.dll → HD 解析度
   - Hendi's MapleStory.exe → Admin 移除 + 4G Flag
   - 改名避免 UAC 問題

2. **STREDIT** 用於 localhost IP 修改
   - v83 選 "No" (非 v118+)
   - 選 "Yes" IP 搜尋模式

3. **4G Flag 修改**
   - PE Header 偏移 0x0000013E
   - 0x0F → 0x2F

### Opcode 結構學習

- 封包格式：`[Header(4 bytes)][Opcode(2 bytes)][Data]`
- 三層加密：AES-OFB + MapleCrypto + Shanda
- 第一個封包是 Hello Packet (明文)

### WZ 修改流程

1. HaRepacker 開啟客戶端 WZ
2. 編輯內容
3. "Private Server..." 匯出到伺服器 XML
4. 保持 server-client 同步

### NPC 腳本相容性

- 使用普遍支援的 `cm.` 函數
- 避免 `importPackage` (JDK 7+ 已廢棄)
- 不同音源 (Cosmic/Kinoko/HeavenMS) 可能需要調整
