## 三、Opcode結構

### 3.1 MapleCrypto 頭部混淆公式

**加密端**：
```java
public byte[] getPacketHeader(int opcode, int length, int iv) {
    int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
    verify = (verify << 3) | (verify >>> 5);
    verify &= 0xFFFF;
    return new byte[] {
        (byte) (verify & 0xFF),
        (byte) ((verify >>> 8) & 0xFF),
        (byte) (length & 0xFF),
        (byte) ((length >>> 8) & 0xFF)
    };
}
```

**解密端**：
```java
public int retrieveOpcode(byte[] header, int iv) {
    int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
    verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
    return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
}
```

### 3.2 Send vs Recv Opcode 區分

**URL**: https://mapleref.fandom.com/wiki/Operation_Code

> "Transmitting from the server and receiving on the client has one set of agreed upon Opcodes, where transmitting from the client and receiving on the server has yet another."

| 方向 | 說明 | Opcode 集合 |
|------|------|------------|
| **S→C (Send)** | 伺服器發送到客戶端 | Server Opcode Set |
| **C→S (Recv)** | 客戶端發送到伺服器 | Client Opcode Set |

### 3.3 v83 IDA 分析Opcode 進階技巧 (RaGEZONE)

**URL**: https://forum.ragezone.com/threads/getting-packet-structures-opcodes-using-ida.792436/

**分析關鍵**：
1. 找 `CClientSocket::ProcessPacket` - 客戶端接收封包的總入口
2. 從這裡根據 header 分發到 `CField::OnPacket` 或 `CLogin::OnPacket`
3. 這就是為什麼有 opcode 清單就能輕鬆找到客戶端接收封包的處理函數

### 3.4 IDA Decode 函數對照表

| IDA 函數 | MaplePacketWriter 方法 | 讀取大小 |
|---------|----------------------|---------|
| `Decode1` | `mplew.write` | 1 byte |
| `Decode2` | `mplew.writeShort` | 2 bytes |
| `Decode4` | `mplew.writeInt` | 4 bytes |
| `DecodeStr` | `mplew.writeMapleAsciiString` | 可變 |
| `DecodeBuffer(x, 8)` | `mplew.writeLong` | 8 bytes |

### 5.4 動態 NPC 商店腳本源 (RaGEZONE, 2016)

**URL**: https://forum.ragezone.com/threads/dynamic-npc-shop-script-v83.1089979/

```javascript
var status = 0;
var items = Array(
    Array(1302000, 10000, 1),
    Array(1302001, 20000, 1)
);

function action(mode, type, selection) {
    if (mode == -1) { cm.dispose(); return; }
    if (mode == 0 && type == 1) { cm.dispose(); return; }
    if (mode == 1) status++; else status--;
    
    if (status == 0) {
        var msg = "歡迎光臨！\r\n請選擇商品：\r\n";
        for (var i = 0; i < items.length; i++) {
            msg += "#L" + i + "##i" + items[i][0] + "# - " + items[i][1] + " 金幣#l\r\n";
        }
        cm.sendSimple(msg);
    } else if (status == 1) {
        var item = items[selection];
        if (cm.getMeso() >= item[1] && cm.canHold(item[0])) {
            cm.gainMeso(-item[1]);
            cm.gainItem(item[0], item[2]);
            cm.sendOk("購買成功！");
        } else {
            cm.sendOk("金幣不足或背包已滿！");
        }
        cm.dispose();
    }
}
```
