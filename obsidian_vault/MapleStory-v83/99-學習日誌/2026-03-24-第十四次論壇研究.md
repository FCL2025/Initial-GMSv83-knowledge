## 🆕 Cosmic 伺服器最新架設指南 (2026)

**來源**：https://github.com/P0nk/Cosmic

Cosmic 是目前最活躍的 v83 模擬器，基於 HeavenMS，於 2021 年 3 月啟動。

### 環境需求
- **Java 21**（推薦 Amazon Corretto）
- **IntelliJ IDEA**（推薦 IDE）
- **MySQL Community Server 8+**
- **HeidiSQL**（資料庫客戶端）
- **客戶端**：https://github.com/P0nk/Cosmic-client

### 架設步驟摘要
1. 安裝 MySQL 8+，設定 root 密碼
2. 用 HeidiSQL 建立名為 `cosmic` 的資料庫 schema
3. Clone Cosmic repo，設定 `config.yaml` 中的 `DB_PASS`
4. 執行 `net.server.Server` 啟動伺服器
5. 出現「Cosmic is now online」即成功

### Docker 支援
```bash
docker compose up
```

### GM 帳號預設
- Username: `admin`
- Password: `admin`
- Pin: `0000`
- Pic: `000000`

## 🆕 v83 WZ 編輯實戰技巧 (maplesecrets)

**來源**：http://maplesecrets.blogspot.com/2014/05/tutorial-wz-editing-for-maplestory-d.html

### Mob.WZ 編輯重點
| 欄位 | 說明 |
|------|------|
| `info.speed` | 怪物移動速度 (範圍 0-126) |
| `info.jump` | 怪物跳躍高度 |
| `info.bdyAttack` | 身體碰撞攻擊 (設為 0 = 無敵) |
| `info.acc` | 怪物命中率 (設為 0 = 攻擊必miss) |
| `info.MADamage` | 魔法攻擊力 |
| `info.PADamage` | 物理攻擊力 |
| `info.HP` | 怪物血量 |
| `info.MP` | 怪物魔法值 |
| `info.exp` | 擊殺經驗值 |
| `info.level` | 怪物等級 |

### 實用 Mob 編輯
#### 怪物極速 (Speed 126)
```
HaRepacker → Mob.wz → [怪物ID] → info → speed → 126 → Save
```

#### 無敵模式 (GodMode)
```
1. Open deMSwZ
2. Open mob.wz
3. Deselect everything except bodyAttack
4. Set bodyAttack = 0
5. Process All → Save
```

#### 怪物不失蹤 (Pacifist GodMode)
```
1. HaRepacker → Mob.wz → [怪物ID] → info
2. Add property: name="notAttack", value=1, type=wzCompressedIntProperty
```

#### 移除怪物跳躍 (No Jump)
```
HaRepacker → Mob.wz → [怪物ID] → 刪除 "jump" 屬性 → Save
```

## 🆕 NPC 腳本源相容性重點 (RaGEZONE, 2026-03)

**來源**：https://forum.ragezone.com/threads/kinoko-npc-scripts.1261538/

### 音源版本差異
- Cosmic、HeavenMS、Kinoko 的 NPC 腳本解析器有差異
- 較舊腳本可能在新解析器不相容
- 不同版本的 `cm.` 函數實作可能略有不同

### 解決方案
1. 確認目標音源版本
2. 使用普遍支援的 `cm.` 函數
3. 避免使用特定版本專有的 Package 路徑
4. 先在本地測試再部署
