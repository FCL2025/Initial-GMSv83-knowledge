## 本次新增知識

### 1. 登入器/客戶端修改
- **MapleEzorsia V2 README 完整翻譯**：設計哲學、技術實現、編譯設置
- **config.ini 注釋問題**：避免在 `;` 後面添加注釋
- **EzorsiaV2_UI.wz**：建議將所有 WZ 編輯放在此檔案中避免衝突

### 2. 抓包分析
- **MaplePE 完整技術架構**：
  - MaplePE.exe = UDP 伺服器接收封包資訊
  - Packet.dll = 注入目標進程，Hook 封包讀寫函數
  - 封包發送前提：伺服器必須先回應過至少一個封包
- **BB+ 版本崩潰防護**：Hook MakeBufferList 而非 SendPacket

### 3. Opcode結構
- **SpiritIDAPlugin 環境支援**：IDA 7.0-7.5 全支援，Python 3.6-3.8 支援
- **while() 迴圈限制**：建議設 GET_ALL_DECODES=true 提高準確度

### 4. WZ修改
- **Gold Standard 持續更新**：GMS (old) 加密 + UI 直接修改 + Private Server XML 導出
- **EzorsiaV2_UI.wz**：第三方修改的最佳存放位置

### 5. NPC腳本編寫
- **IntransigentMS ES6 規範**：
  - 使用 `Java.type()` 而非 `importPackage`
  - 使用 `print()` 除錯輸出
  - 支援模板字串和箭頭函數
- **Cosmic 專案定位**：純伺服器專案，Vanilla gameplay 原則
