## 🆕 Cosmic v83 模擬器最新動態 (2026-03)

### Cosmic 模擬器環境需求更新

**重要更新**：Cosmic 的環境需求與舊版 HeavenMS/MapleSolaxiaV2 有所不同：

| 軟體 | 舊版需求 (HeavenMS) | Cosmic 需求 | 變化 |
|------|-------------------|-------------|------|
| Java | JDK 8 (7u79) | **Java 21** | ⚠️ 重大更新 |
| MySQL | 5.5+ | **MySQL 8+** | ⚠️ 升級 |
| 設定檔 | configuration.ini | **config.yaml** | YAML 格式 |
| JCE | 需要 | 不再需要 | ✅ 簡化 |
| IDE | NetBeans | IntelliJ IDEA | 推薦 |

### Cosmic 架設流程（2026 最新）

**前置作業**：
1. 下載並安裝 [MySQL Community Server 8+](https://dev.mysql.com/downloads/mysql/)
2. 設定 root 密碼（請牢記！）
3. 下載並安裝 [HeidiSQL](https://www.heidisql.com/download.php)
4. 建立新 Session，連線到資料庫
5. 建立名為 `cosmic` 的資料庫

**伺服器架設**：
```bash
# 1. 環境需求
- Java 21 (推薦 Amazon Corretto)
- IDE (推薦 IntelliJ IDEA)

# 2. Clone 專案
git clone https://github.com/P0nk/Cosmic.git

# 3. 設定資料庫密碼
# 開啟 config.yaml，找到 "DB_PASS"，設定為你的 MySQL root 密碼

# 4. 啟動伺服器
# main 方法位於 net.server.Server
# 若看到 "Cosmic is now online" 表示啟動成功
```

**Docker 支援**：
```bash
# 若有 Docker Desktop，可使用 docker compose 啟動
docker compose up
```

### Cosmic vs 舊版模擬器比較

| 特性 | Cosmic (2021-2026) | HeavenMS/MapleSolaxia |
|------|-------------------|----------------------|
| Java 版本 | 21 | 8 |

## 🆕 Cosmic-client 客戶端設定 (2026-03 更新)

**GitHub**: https://github.com/P0nk/Cosmic-client

### 重要安全提醒
```
⚠️ 客戶端會被 Windows Security 標記為病毒並自動刪除！
這是因為客戶端已被修改用於本地遊戲和自訂功能。
解決方法：將下載目錄和安裝目錄加入 Virus & threat protection 排除清單。
```

### 客戶端架設步驟
```
Step 1: 執行 MapleGlobal-v83-setup.exe，選擇安裝目錄
        （預設 C:\Nexon\MapleStory）

Step 2: 移除以下檔案（與私服連線無關且會干擾）：
        - HShield/ (整個資料夾)
        - ASPLnchr.exe
        - MapleStory.exe
        - Patcher.exe

Step 3: 複製 cosmic-wz 中的所有 WZ 檔案到安裝目錄並覆蓋

Step 4: 複製 client (HeavenMS-localhost-WINDOW.exe) 到安裝目錄

Step 5: 雙擊 client 啟動（伺服器需先行啟動）
```

### HxD IP 修改步驟
```
1. 下載並安裝 HxD：https://mh-nexus.de/en/hxd/
2. 用 HxD 開啟 client exe（建議先複製一份備份）
3. 按 Ctrl+F，選擇 "Text-string"，搜尋 "127.0.0.1"
4. 會找到三個連續出現的 IP 位址
5. 在第一個 "127" 前面開始輸入目標 IP（覆蓋模式）
6. 三個 IP 都完成替換後儲存
```

### 疑難排解
| 問題 | 解決方案 |
|------|---------|
| 客戶端無法啟動 | 狂按雙擊 client，有機會成功開啟 |
| "Unable to connect to login server" | 確認伺服器已啟動 |
| Windows 刪除客戶端 | 將目錄加入 Windows Security 排除清單 |
| 客戶端 exe 被隔離 | 從隔離區恢復並加入排除 |

## 🆕 MapleSolaxiaV2 / HeavenMS 架設對照 (傳統方法)

**GitHub**: https://github.com/skhzhang/MapleSolaxiaV2

### 環境需求（傳統）
| 軟體 | 版本 | 必要性 |
|------|------|--------|
| JDK | 7u79 | 必要（必須 JDK 7） |
| JCE | 7 Unlimited | 必要 |
| MySQL | 5.5+ | 必要 |
| WampServer | 2.0i | MySQL 管理 |
| NetBeans | 8.0.2 | 開發用 |

### 架設步驟摘要
```
Step 1: 安裝 JDK 7 + JCE Unlimited Policy
Step 2: 安裝 WampServer + MySQL
Step 3: 執行 SQL 初始化（db_database.sql + db_drops.sql）
Step 4: 手動在 accounts 資料表建立第一個帳號
Step 5: 設定 configuration.ini 中的 IP
Step 6: NetBeans Clean and Build + launch.bat
```

### WZ 編輯黃金法則
```
⚠️ DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

正確流程：
1. HaRepacker 4.2.4 開啟 GMS (old) 加密的客戶端 WZ
2. 使用 UI 直接修改數值（禁止刪除/重新導入節點！）
3. 儲存 WZ
4. 在 HaRepacker 中選擇 "Private Server..." 導出
5. 將導出的 XML 覆蓋到伺服器的 WZ.XML 目錄
```
