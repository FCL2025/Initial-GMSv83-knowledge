### 2. 完整 v113 私服架設教學（maplepstutorial.com）

**架設流程**：
```
1. 安裝 Java JDK/JRE（v8u172）
2. 安裝 XAMPP（資料庫）
3. 安裝 Navicat（資料庫管理）
4. 安裝 NetBeans（Java IDE）
5. 匯入 SQL 檔案（testms.sql）
6. 啟動伺服器（server.swing.WvsCenter）
7. 進入遊戲
```

**GM 權限設定**：
- 資料庫 `characters` 表中 `gm` 欄位設為 `100`（最高 GM）
- 角色需下線後修改才生效

**常見問題**：
- 碰怪就死：清除 `csitem` 資料表
- 無法撿道具：使用 `!禁止` 指令

### 3. v83 LAN 多人連線錯誤 -2147467259 排查（RaGEZONE, 2週前）

**問題**：PC1（主機）正常，PC2 連線時錯誤碼 -2147467259

**原因分析**：
| 可能原因 | 說明 |
|---------|------|
| 防火牆阻擋 | 7575/8484 端口被阻擋 |
| IP 修改不完整 | 客戶端未正確指向主機 IP |
| 端口未轉發 | 路由器未設定端口轉發 |
| IV 不同步 | AES 加密 IV 問題 |

**解決步驟**：
```
1. 確認兩台 PC 在同一網段（能互相 ping）
2. 用 HxD 修改 PC2 客戶端 IP 為主機 LAN IP（不要用 127.0.0.1）
3. 開放 Windows 防火牆 7575 和 8484 端口
4. 確認伺服器監聽 IP 為 0.0.0.0（非 127.0.0.1）
```

**來源**：https://forum.ragezone.com/threads/need-help-v83-lan-server-works-on-pc1-but-pc2-gets-error-2147467259.1262187/

### 8. v83 商場 Dupe 封包分析（PDFCOFFEE）

**來源**：https://pdfcoffee.com/v83-dupe-pdf-free.html

**Opcode 0x6F (SHOP_BUY) 結構**：
```
6F 00 20 00 FF FF FF 80 00
```
- `6F 00`：Opcode（little endian）
- `20 00`：商店 ID
- `FF FF FF 80`：物品索引 + 最大 Mesos 標誌
- `00`：結束標記

> ⚠️ 僅供學習研究用途，請勿用於作弊
