## 📌 1) 登入器/客戶端修改

### STREDIT v2.9.2.0 - 官方 localhost 編輯器

**來源**：https://forum.ragezone.com/threads/release-stredit-localhost-editor.810762/

STREDIT 是一款專門用於編輯 MapleStory localhost.exe 的工具，支援修改：
- IP 位址
- 結束廣告 URL
- 忘記帳號/密碼 URL
- 及其他加密字串

**STREDIT 操作步驟**：
```
1. 下載 STREDIT
2. 啟動 STREDIT
3. 在 STREDIT 中開啟 localhost.exe
4. 當 STREDIT 詢問是否為 v118+ → 選「No」（v83 適用）
5. 當 STREDIT 詢問 IP 搜尋 → 選「Yes」
6. 點擊「Force to a IP」按鈕
7. 輸入你的目標 IP
8. 儲存
```

### HxD Hex 編輯器 - 手動 IP 修改

**來源**：https://forum.ragezone.com/threads/video-guide-how-to-make-a-maplestory-v83-private-server-–-full-setup-tutorial.1243566/

當 STREDIT 無法使用時，可使用 HxD 手動修改：

**HxD 步驟**：
```
1. 下載並安裝 HxD：https://mh-nexus.de/en/hxd/
2. 用 HxD 開啟 localhost.exe（先複製一份備份）
3. 按 Ctrl+F 搜尋「127.0.0.1」
4. 會找到三個相連的 IP 位址
5. 分別覆蓋為目標 IP 位址
6. 儲存
```

## 📌 2) 抓包分析

### 完整 v83 封包流向圖

**來源**：https://forum.ragezone.com/threads/how-does-one-find-a-structure-of-a-packet-via-ida.1162654/

**CClientSocket::ProcessPacket 完整流向**：
```
TCP 接收資料
    ↓
CClientSocket::ProcessPacket() [入口函數]
    ↓  解析 MapleCrypto header (4 bytes)
    ↓  解密並提取 opcode (2 bytes, little endian)
    ↓  剩餘為 payload
CInPacket::Decode() 封包解碼
    ↓
switch(opcode) 分發
    ↓
CLogin::OnPacket()    // 登入狀態
    或
CField::OnPacket()    // 遊戲中狀態
    ↓
具體的封包處理函數
```

### MapleCrypto Header 核心公式

**Header 加密（發送端）**：
```java
int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
verify = (verify << 3) | (verify >>> 5);
verify &= 0xFFFF;
return new byte[] { (byte)(verify & 0xFF), (byte)((verify >>> 8) & 0xFF),
                     (byte)(length & 0xFF), (byte)((length >>> 8) & 0xFF) };
```

**Header 解密（接收端）**：
```java
int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
```

### 封包長度識別技巧

| 封包類型 | 長度特徵 |
|---------|---------|
| 登入請求 (0x01) | 約 50-100 bytes（帳密加密後） |
| 角色列表 (0x03) | 可變（依角色數量） |
| 移動 (0x14) | 約 40-60 bytes |
| NPC 對話 (0x28) | 約 20-30 bytes |

| `DecodeBuffer(x, 13)` | `mplew.writeAsciiString(str, 13)` | 固定13字節字串 |

### IDA 找 Opcode 的方法

**方法 1：AOB 搜尋**
- 在 IDA Hex View 搜尋已知 opcode 的 byte pattern
- 例如搜尋 `01 00`（LOGIN_PASSWORD 的 little endian）

**方法 2：StringPool 搜尋**
- IDA 中搜尋 `push opcode` 或 `68`（push instruction）
- 找到 StringPool 引用的位置

**方法 3：函數名搜尋**
- 搜尋 `SendPacket`、`OnPacket`、`CClientSocket`
- 追蹤封包處理邏輯

| Type 值 | 方法 | 說明 |

| 0 | `sendNext`, `sendPrev` | 標準對話 |

| 3 | `sendGetNumber` | 數字輸入 |
| 4 | `sendGetText` | 文字輸入 |
| 12 | `sendAcceptDecline` | 接受/拒絕 |

### Mode 參數含義

| Mode 值 | 意義 |
|---------|------|
| 1 | 玩家點擊了「下一步」或選項 |
| 0 | 玩家點擊了「返回」(back button) |
| -1 | 玩家關閉了對話框 (ESC 或 X) |

### 狀態機最佳實踐

```javascript
var status = 0;

function action(mode, type, selection) {
    if (mode == -1) {
        // 關閉對話 → 釋放資源
        cm.dispose();
    } else {
        if (mode == 0 && type == 1) {
            // 強制關閉（如在 sendNext 中按「返回」）
            cm.dispose();
            return;
        }
        
        if (mode == 1)
            status++;
        else
            status--;
        
        // 處理各狀態
        switch (status) {
            case 0: ...
            case 1: ...
        }
    }
}
```

### 職業二轉 NPC 腳本源

```javascript
var status = 0;
var jobId = 0;

function start() {
    cm.sendNext("恭喜 #b10級#k！請選擇二轉職業：\r\n#b#L0#火與毒法師#l\r\n#L1#冰與雷法師#l\r\n#L2#僧侶#l");
}

function action(mode, type, selection) {
    if (mode != 1) {
        if (mode == 0) cm.sendOk("想好再來找我。");
        cm.dispose();
        return;
    }
    status++;
    if (status == 1) {
        if (selection == 0) jobId = 210;
        else if (selection == 1) jobId = 220;
        else jobId = 230;
        cm.sendYesNo("確定要選擇 #b" + getJobName(jobId) + "#k 嗎？");
    } else if (status == 2) {
        cm.gainJob(jobId);
        cm.sendOk("恭喜！你現在是 #b" + getJobName(jobId) + "#k 了！");
        cm.dispose();
    }
}

function getJobName(id) {
    if (id == 210) return "火與毒法師";
    if (id == 220) return "冰與雷法師";
    return "僧侶";
}
```
