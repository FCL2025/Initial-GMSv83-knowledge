## 2) 抓包分析

### Maple_Pshark（GitHub: obstriker/Maple_Pshark）
- 專為 v83 MapleStory 私人伺服器設計的封包截獲器
- 完全相容 v83 封包格式
- 核心功能：Hook 加密函數，直接截獲明文
- 支援封包修改和注入
- 規則化封包操作和過濾

### MapleShark（Diamondo25/MapleShark）
- 基於 SharpPcap 的 MapleStory sniffer
- 支援 ScriptDotNet (S#) 腳本引擎自訂封包解析
- 可批次命名封包結構

### Wireshark 抓包過濾表達式（v83 專用）
```
tcp.port == 7575 || tcp.port == 8484
```
- 7575：登入端口
- 8484：遊戲端口

## 3) Opcode 結構

### 核心發現：v83 Opcode 現況（RaGEZONE, 2024-12）
- ✅ 幾乎全部被映射，不需要額外追蹤封包
- ✅ 大多數 opcode 表已在開源專案（Cosmic、HeavenMS）中完整提供
- ⚠️ 唯一需要的反向工程是**修改客戶端時**

### MapleRef Wiki（https://mapleref.fandom.com/wiki/Packets）
- Opcode 是封包開頭用於識別類型的唯一識別符
- **Send (S→C)** 和 **Recv (C→S)** 使用不同的 Opcode 集合
- 同一個數字在不同方向代表完全不同的含義

### IDA 分析 Opcode 步驟（RaGEZONE）
1. 找 `CClientSocket::ProcessPacket` — 客戶端接收封包的總入口
2. 從這裡根據 header 分發到 `CField::OnPacket` 或 `CLogin::OnPacket`
3. 這就是為什麼有 opcode 清單就能輕鬆找到客戶端接收封包的處理函數

### CInPacket Decode 函數對照表
| IDA 函數 | 伺服器端寫入方法 | 讀取大小 |
|---------|----------------|---------|
| Decode1 | mplew.write(1) | 1 byte |
| Decode2 | mplew.writeShort(1) | 2 bytes |
| Decode4 | mplew.writeInt(1) | 4 bytes |
| Decode8 | mplew.writeLong(1) | 8 bytes |
| DecodeStr | mplew.writeMapleAsciiString() | 可變 |
| DecodeBuffer(x, n) | mplew.write(n bytes) | n bytes |

## 4) WZ 修改

### WzImg-MCP-Server（AI 輔助 WZ 分析，2026-03-05）
- GitHub: https://github.com/lastbattle/WzImg-MCP-Server
- 74 tools across 10 categories
- AI 助手可直接讀取 IMG 數據、檢查屬性、搜尋資源、導出圖片/音頻
- 使用 Model Context Protocol (MCP)

### WZ vs NX 格式
- NX 效能更佳，檔案更小
- 主流伺服器使用 WZ，官方伺服器偏好 NX
- HaRepacker 可直接轉換

### v83 WZ 編輯重點
- 使用 HaRepacker 編輯 .wz 檔案
- 修改怪物屬性（Attack, Defense, HP, Drop）
- NPC.wz 可用於自訂 NPC 外觀
- Skill.wz 需要 assembly 修改（難度較高）

## 5) NPC 腳本編寫

### NPC TYPE 值（RaGEZONE, 2026-03）
| Type 值 | 名稱 | 客戶端行為 |
|---------|------|-----------|
| 0 | 普通 NPC | 點擊後發送 NPC_TALK 封包 |
| 1 | 商店 NPC | 點擊後直接開啟商店介面 |
| 2~6 | 特殊 NPC | 拍賣、保險箱等 |

### 腳本源相容性問題（RaGEZONE, 2026-03）
- 不同音源（Cosmic、Kinoko、HeavenMS）的 NPC 腳本解析器版本不同
- 較舊腳本在新解析器可能不相容
- 避免使用 `importPackage`（JDK 7+ 已廢棄）
- 建議使用普遍支援的 `cm.` 函數

### NPC 腳本範例（動態商店）
```javascript
var status = 0;

function start() {
    cm.sendSimple("歡迎！請選擇服務:\n #b#L0#購買藥水#l\n #L1#購買彈藥#l\n #L2#離開#l");
}

function action(mode, type, selection) {
    if (mode == 1) status++;
    else status--;

    if (status == 0) {
        if (selection == 0) {
            if (cm.getMesos() >= 1000) {
                cm.gainMesos(-1000);
                cm.gainItem(2000002, 100, true);
                cm.sendNext("交易完成！");
            } else {
                cm.sendNext("楓幣不足！");
            }
            cm.dispose();
        } else if (selection == 1) {
            cm.openShop(9010000); // 開啟商店
            cm.dispose();
        } else {
            cm.dispose();
        }
    }
}
```
