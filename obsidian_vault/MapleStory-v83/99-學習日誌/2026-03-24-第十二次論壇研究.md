## 📌 1. 登入器/客戶端修改

### STREDIT v118+ 提示處理
- v83 客戶端使用 STREDIT 時，回答「否 (No)」
- 然後選擇「IP Search」，點擊「Force to a IP」按鈕
- 輸入目標 IP 完成修改

### localhost.exe 修改方式
- 使用 HxD 或 STREDIT 修改 IP 位址
- 必須保持位元組長度一致（127.0.0.1 是 9 bytes）
- 替換 IP 位址時需注意長度相同

### 常用工具
- **STREDIT** (RaGEZONE): 字串/IP 修改
- **HxD**: Hex 編輯
- **Cheat Engine**: 記憶體修改

- 加密流程：Shanda + AES

## 📌 3. Opcode 結構

### 雙層加密架構
```
客戶端發送：原始封包 → Shanda 加密 → AES 加密 → 傳送
伺服器接收：AES 解密 → Shanda 解密 → 處理
```

### 登入封包範例
- 加密後：`64 A4 9F 07 FD 4D...`
- AES 解密後：`71 1A 5B 6A 7E B9...`
- Shanda 解密後：`E2 FB 40 7E 67 08...`
- 原始 opcode：`E2 FB`（應為 `01 00` 即 LOGIN_PASSWORD）

### v83 Opcode 格式
- Opcode 為 2 bytes (little endian)
- MapleCrypto 頭部混淆機制使用 IV + 位旋轉

### CRC32 GameGuard
- 客戶端有舊版 GameGuard CRC32 校驗
- 附加 opcode 0x13/19
- 私服環境無實際作用，可忽略

## 📌 4. WZ 修改

### ⚠️ v83 特殊要求
- **不能使用原始 WZ 檔案**（來自 Nexon/Wizet）
- 必須使用**自定義修改過的 WZ**
- 原始 WZ 可能導致遊戲內錯誤

### v83 專用 WZ 資源
- **MapleSolaxiaV2**: Google Drive (包含 client_wz)
- **Cosmic Client**: github.com/MagMC/MagMCMapleClient

### WZ 加密格式
- 使用 **GMS (old)** 加密格式
- HaRepacker-resurrected 支援

### 常見 WZ 用途
| WZ 檔案 | 用途 |
|---|---|
| Character.wz | 角色造型、裝備外觀 |
| Item.wz | 物品資料、裝備屬性 |
| Mob.wz | 怪物資料、外觀 |
| Map.wz | 地圖圖塊、物件 |
| Skill.wz | 技能效果、數值 |
| Quest.wz | 任務資料 |
| Npc.wz | NPC 造型 |
| String.wz | 文字（道具名稱、描述等） |
| Sound.wz | 音效 |
| UI.wz | 介面資源 |

## 📌 5. NPC 腳本編寫

### NPCConversationManager 常用方法

#### 對話發送
```javascript
cm.sendNext("文字");           // 下一步按鈕
cm.sendNextPrev("文字");        // 上一步 + 下一步
cm.sendPrev("文字");            // 返回按鈕
cm.sendOk("文字");              // 確定按鈕
cm.sendYesNo("文字");           // 是/否按鈕
cm.sendSimple("文字\r\n#b#L0#選項0#l\r\n#L1#選項1#l");
cm.sendStyle("文字", styleId);  // 髮型/染色選擇
cm.sendGetNumber("文字", defaultNum, min, max);
cm.sendGetText("文字");
cm.dispose();                   // 結束腳本
```

#### 玩家狀態查詢
```javascript
cm.getLevel()                   // 等級
cm.getJob()                     // 職業
cm.getMesos()                   // 楓幣
cm.haveItem(itemId, count)     // 是否持有物品
```

#### 玩家操作
```javascript
cm.gainItem(itemId, count);    // 給予物品
cm.gainMesos(amount);           // 給予/收取楓幣
cm.changeJob(jobId);            // 轉職
cm.openShop(shopId);            // 開啟商店
cm.warp(mapId);                 // 傳送地圖
```

### 動態商店腳本
- 可用 NPC 腳本實現動態商店（不依賴 SQL）
- 每當 NPC 對話時讀取庫存資料
- 免除修改資料庫後重啟伺服器的麻煩
