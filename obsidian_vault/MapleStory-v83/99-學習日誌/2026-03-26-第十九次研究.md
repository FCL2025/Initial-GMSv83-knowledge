## 二、本次最新發現

### 2.1 NPC TYPE 值含義解析（RaGEZONE, 2026-03）

**來源**: https://forum.ragezone.com/threads/npc-type.1262188/

| Type 值 | 名稱 | 客戶端行為 | 腳本需求 |
|---------|------|-----------|---------|
| `0` | 普通 NPC | 點擊後發送 NPC_TALK 封包 | 需要 `.js` 腳本 |
| `1` | 商店 NPC | 點擊後直接開啟商店介面 | 需要/不需要 `.js` 腳本 |
| `2~6` | 拍賣 NPC | 點擊後開啟拍賣功能 | 不需要腳本 |

**重要發現**：Type 值決定客戶端如何處理 NPC 互動，而不是腳本本身的類型！

### 2.2 NPC.tags 翻譯問題（RaGEZONE, 2026-02）

**來源**: https://forum.ragezone.com/threads/help-on-editing-npc-tags.1261250/

**問題**：Custom V18 Florist Files 中的 NPC tags 無法找到翻譯來源

**解決方案**：
1. 在 `client/client/` 資料夾中搜尋相關文字

3. `NPC.wz` 中的 `tags` 欄位可能來自 `String.wz` 而非直接定義

### 2.6 localhost.exe 運作原理深入解析

**發現**：`localhost.exe` 並非簡單 IP 替換工具，而是：

```
┌─────────────────────────────────────────────┐
│  localhost.exe                             │
│  1. 作為 proxy 攔截遊戲流量                 │
│  2. 解析版本號和加密參數                    │
│  3. 將流量轉發到 localhost 的私服           │
│  4. 處理版本驗證                           │
└─────────────────────────────────────────────┘
```

**無法正常工作原因**：
| 問題 | 原因 | 解決方案 |
|------|------|---------|
| 連線後馬上斷線 | 版本號不匹配 | 確認客戶端版本=83 |
| 完全無法連線 | localhost.exe 沒有在運行 | 先啟動 localhost.exe |
| 提示 HackShield 錯誤 | 客戶端保護未移除 | 刪除 HShield 資料夾 |
