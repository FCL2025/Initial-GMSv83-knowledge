### botnored v83 類別架構

**來源**: https://botnored.weebly.com/blog/maplestory-classes-v83

**核心原則**：
> 第一個發送/接收的封包永遠是明文（未加密）

**客戶端類別對照**：

| 類別 | 功能 |
|------|------|
| `CWvsContext` | 世界狀態（角色資料、伺服器列表） |
| `CClientSocket` | 網路連接（Send/Recv） |
| `CField` | 地圖/遊戲場景處理 |
| `CLogin` | 登入畫面處理 |

**封包處理流程**：
```
CClientSocket::ProcessPacket()
    ↓
根據 Header 分發
    ↓
CField::OnPacket() 或 CLogin::OnPacket()
    ↓
處理具體 Opcode
```

## 🆕 3. Opcode 結構

### Send/Recv Opcode 基本概念

**來源**: https://mapleref.fandom.com/wiki/Operation_Code

| 方向 | 說明 | Opcode 集合 |
|------|------|------------|
| **S→C (Send)** | 伺服器 → 客戶端 | Server Opcode Set |
| **C→S (Recv)** | 客戶端 → 伺服器 | Client Opcode Set |

**重要原則**：
> 同一個 Opcode 數字在不同方向可能代表不同含義！

### v83 Opcode 現況

**來源**: https://forum.ragezone.com/threads/packet-opcode-analysis.1238027/

**結論**：
- ✅ v83 Opcode 已完全映射，無需反向工程追蹤
- ✅ 開源專案（Cosmic、HeavenMS、MapleSolaxiaV2）已完整提供
- ⚠️ 唯一需要反向工程的情境：**客戶端修改**

### CInPacket 解碼函數對照

| C++ (IDA) | Java (伺服器) | 大小 |
|-----------|--------------|------|
| `Decode1` | `mplew.write(1)` | 1 byte |
| `Decode2` | `mplew.writeShort()` | 2 bytes |
| `Decode4` | `mplew.writeInt()` | 4 bytes |
| `Decode8` | `mplew.writeLong()` | 8 bytes |
| `DecodeBuffer(x, n)` | `mplew.write(n bytes)` | n bytes |
| `DecodeStr` | `mplew.writeMapleAsciiString()` | 可變 |
| `DecodePos` | 特殊：座標編碼 | 特殊 |

| status | 意義 |
|--------|------|

### 動態 NPC 商店腳本

```javascript
var status = 0;
var items = Array(
    Array(1302000, 10000, 1),
    Array(1302001, 20000, 1),
    Array(1302002, 30000, 1)
);

function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && type == 1) {
            cm.dispose();
            return;
        }
        if (mode == 1) status++;
        else status--;

        if (status == 0) {
            var msg = "歡迎光臨！\r\n請選擇商品：\r\n";
            for (var i = 0; i < items.length; i++) {
                msg += "#L" + i + "##i" + items[i][0] + "# - " + items[i][1] + " 金幣#l\r\n";
            }
            cm.sendSimple(msg);
        } else if (status == 1) {
            var item = items[selection];
            if (cm.getMeso() >= item[1]) {
                if (cm.canHold(item[0])) {
                    cm.gainMeso(-item[1]);
                    cm.gainItem(item[0], item[2]);
                    cm.sendOk("購買成功！");
                } else {
                    cm.sendOk("背包空間不足！");
                }
            } else {
                cm.sendOk("金幣不足！");
            }
            cm.dispose();
        }
    }
}
```
