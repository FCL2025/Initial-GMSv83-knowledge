### 1. 登入器/客戶端修改

#### Hex Edit 客戶端 IP 修改要點 (maplesecrets 2013, 2025 仍然適用)
**來源**: http://maplesecrets.blogspot.com/2013/04/how-to-hex-edit-clients-easily.html

**v83 Hex 編輯流程**：
1. 使用 Neo Hex Editor 或 HxD
2. 開啟 `localhost.exe`（修改過的客戶端入口）
3. 搜尋 `127.0.0.1` 字串
4. IP 位址中的 `.` 在 Hex 中表示為 `2e`
5. 若新 IP 比 127.0.0.1 更長，需用 `00` 填補剩餘空間
6. 儲存修改

**重要提醒**：
- 只修改第一個出現的 IP 即可
- 其餘 IP 可能是備用或內網 IP

### 2. WZ 編輯

#### HaRepacker-resurrected (2025 仍在更新)
**GitHub**: https://github.com/lastbattle/Harepacker-resurrected

v83 WZ 編輯工作流程：
```
1. HaRepacker 開啟 WZ 檔案
2. 選擇加密格式：GMS (old) for v83
3. 導航到目標節點
4. 直接修改數值（不要刪除/重新導入！）
5. 儲存 WZ
6. 導出為 XML（Private Server 格式）
7. 覆蓋到伺服器 WZ.XML 目錄
```

**重要原則**：
> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

#### WZ 怪物修改教學 (v83)
**來源**: http://maplesecrets.blogspot.com/2014/05/tutorial-wz-editing-for-maplestory-d.html

**怪物無敵/攻擊範圍修改**：
1. 打開 `mob.WZ`
2. 找到目標怪物 ID → attack1 → info → range
3. 修改 lt (left-top) 和 rb (right-bottom) 為 0,0
4. 可修改 attackAfter, effectAfter 為 65535
