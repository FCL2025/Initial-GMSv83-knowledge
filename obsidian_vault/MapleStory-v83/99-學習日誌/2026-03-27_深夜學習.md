## 1. 客戶端修改 (Client Modification)

### 1.1 Hex編輯方式
- 使用 **HxD** 或 **Neo Hex Editor** 進行二進制修改
- 主要修改目標：`localhost.exe` 中的IP位址
- MapleSolaxiaV2/HeavenMS 的 `localhost.exe` 需要Hex編輯以匹配伺服器IP

### 1.2 STREDIT.EXE
- 專用於修改客戶端IP的工具，比Hex編輯更直觀
- 可在 RageZone 論壇找到

### 1.3 客戶端來源
- v83 客戶端可通過Google搜索找到多個版本
- 建議使用 HeavenMS 提供的客戶端套件
- 客戶端工具下載：https://drive.google.com/drive/folders/0BzDsHSr-0V4MYVJ0TWIxd05hYUk

### 1.4 客戶端修改注意事項
- 某些防作弊系統(如HShield)需要移除
- 需刪除：`HShield`資料夾、`ASPLauncher.exe`、`MapleStory.exe`、`patcher.exe`
- 使用修改後的 `localhost.exe` 替換原版

## 2. 抓包分析 (Packet Analysis)

### 2.1 工具推薦
- **MaplePacketPuller** (Python): MapleStory 封包結構分析器
  - https://github.com/Bratah123/MaplePacketPuller
- **Maple_Pshark**: Mapleroyals 封包攔截器，專為v83設計
  - https://github.com/obstriker/Maple_Pshark
- **WPE Pro**: 經典封包編輯工具
- **Cheat Engine**: 可用於記憶體掃描和封包攔截

### 2.2 封包結構 (v83)
- **封包標頭(Packet Header)**: 4位元組
  - 包含：send IV、封包長度（不含標頭）、版本號
  - 發送到客戶端時版本為 -(version + 1)
- **Opcode操作碼**: 2位元組 (short)
  - 最大值：65,536 (2^16)
- **第一個發送/接收的封包永遠是明文（未加密）**

### 2.3 封包編輯器
- Kitterz Packet Editor (DLL): 可攔截和修改v83封包
- Blitz's Cheat Table for v83 (.CT): 包含多種作弊功能

## 3. Opcode結構

### 3.1 v83 Opcode 特點
- Opcode為2位元組(short)
- 幾乎所有v83的recv opcodes都已完整映射
- 可參考 MapleStory Reference Wiki (mapleref.fandom.com)

### 3.2 相關論壇資源
- **RaGEZONE - Packet Opcode Analysis**:
  https://forum.ragezone.com/threads/packet-opcode-analysis.1238027/
- 討論了如何追蹤和分析新版 MapleStory 的Opcode結構
- v83 版本相對穩定，Opcode研究價值高

### 3.3 版本比較
- 早期版本：Opcode為1位元組(byte)
- v83及之後：Opcode為2位元組(short)
- 這使得可用Opcode數量從255擴展到65,536

## 5. NPC腳本編寫

### 5.1 NPC腳本核心類別
- **NPCConversationManager.java** (HeavenMS)
  - https://github.com/ronancpl/HeavenMS/blob/master/src/scripting/npc/NPCConversationManager.java
- 腳本存放位置：`scripts/npc/` 目錄

### 5.2 常用NPC腳本函數
```javascript
// 對話相關
cm.sendNext("訊息內容");           // 發送下一段對話
cm.sendPrev("訊息內容");           // 發送上一段對話
cm.sendSimple("訊息內容\r\n#b#L0#選項1#l\r\n#L1#選項2#l");  // 簡單選擇對話
cm.sendYesNo("訊息內容");          // 是/否對話框

// 任務相關
cm.startQuest([questid]);          // 開始任務
cm.completeQuest([questid]);        // 完成任務
cm.forfeitQuest([questid]);        // 放棄任務
cm.getQuestStatus([questid]);       // 獲取任務狀態

// 玩家狀態
cm.getMeso();                       // 獲取金幣
cm.gainMeso(amount);               // 給予/扣除金幣
cm.getLevel();                     // 獲取等級
cm.getJob();                       // 獲取職業
```

### 5.3 腳本範例：職業選擇NPC
```javascript
var status = 0;

function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && type == 0) {
            cm.dispose();
            return;
        }
        if (mode == 1)
            status++;
        else
            status--;
            
        if (status == 0) {
            cm.sendSimple("選擇你的職業:\r\n#b#L0#火焰/毒系魔法師#l\r\n#L1#冰/雷系魔法師#l\r\n#L2#牧師#l\r\n#L3#選擇職業#l");
        } else if (status == 1) {
            if (selection == 0) {
                cm.sendNext("火焰/毒系魔法師...");
            } else if (selection == 1) {
                cm.sendNext("冰/雷系魔法師...");
            } else if (selection == 2) {
                cm.sendNext("牧師...");
            } else if (selection == 3) {
                // 職業進階邏輯
                cm.doJoBSelection();
            }
        }
    }
}
```
