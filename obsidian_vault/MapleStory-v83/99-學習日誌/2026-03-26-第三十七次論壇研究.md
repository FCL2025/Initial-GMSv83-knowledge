### 1. 登入器/客戶端修改

#### MapleEzorsia-v2 現況 (2023-11 首發，2026 仍為主流)
- **來源**: https://github.com/444Ro666/MapleEzorsia-v2
- **特點**: 第一個 standalone DLL client，無需 Hex 編輯
- **使用方式**: 只需將 `dinput8.dll` 放入遊戲目錄
- **config.ini 關鍵設定**:
  ```
  [Connection]
  serverIP=127.0.0.1
  serverPort=8484
  [Display]
  width=1280
  height=720
  ```

#### Hendi's 4G Client (2023-2024)
- **用途**: 啟用 4GB 記憶體支援（預設 32-bit 只有 2GB）
- **Hex 位置**: 0000013E，值 0f → 2f
- **目的**: 避免 HD 客戶端記憶體不足崩潰

#### Kimmy's v83 localhost Archive (RaGEZONE)
- **來源**: https://forum.ragezone.com/threads/maplestory-client-localhost-archive.1101897/

### 3. Opcode 結構

#### MapleRef Wiki - 權威 Opcode 來源
- **URL**: https://mapleref.fandom.com/wiki/Operation_Code
- **特點**: 
  - 完整的 Send (S→C) / Recv (C→S) Opcode 表
  - 封包結構說明
  - 各版本 Opcode 對照

#### v83 封包加密三層架構
```
Layer 1: MapleCrypto (頭部混淆) - 4 bytes header
Layer 2: Shanda Encryption (對稱加密) - v83 仍在使用
Layer 3: AES-OFB (主加密) - 256-bit key
```

#### MapleCrypto Header 加密公式
```java
// 加密端
int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
verify = (verify << 3) | (verify >>> 5);  // rotate left 3 bits
verify &= 0xFFFF;
```

### 4. WZ 修改

#### MapleSolaxiaV2 WZ 編輯黃金法則
> **DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.**

**正確流程**:
1. HaRepacker 4.2.4 開啟 GMS (old) 加密的客戶端 WZ
2. 使用 UI 直接修改數值（禁止刪除/重新導入節點！）
3. 儲存 WZ
4. 選擇 "Private Server..." 導出 XML
5. 覆蓋到伺服器的 wz.xml 目錄

### 5. NPC 腳本編寫

#### 音源相容性速查表

| 音源 | 解析器 | importPackage | ECMAScript 6 |
|------|--------|-------------|--------------|
| Cosmic | Nashorn | ❌ 已移除 | ✅ |
| IntransigentMS | Nashorn | ❌ | ✅ |
| Kinoko | 混合 | ⚠️ 警告 | ⚠️ |
| HeavenMS | JDK 6 時代 | ✅ 存在 | ❌ |

#### 相容寫法
```javascript
// ✅ 所有音源通用
var MapleCharacter = Packages.client.MapleCharacter;
cm.getPlayer();
cm.gainItem(itemId, count);

// ✅ Nashorn ES6
const MapleCharacter = Java.type("net.sf.odinms.client.MapleCharacter");

// ❌ 不相容 (JDK 7+ 已廢棄)
importPackage(Packages.client);
```
