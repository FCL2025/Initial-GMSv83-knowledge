## 📌 HeavenMS 最新 localhost.exe 修改 (RaGEZONE, May 2024)

**GitHub**: https://github.com/ronancpl/HeavenMS

### Hex Editor 修改 IP 步驟（完整流程）

> 若非 `localhost` 作為目標 IP，需用 Hex Editor 修改 localhost.exe 中的 IP

**步驟**：
1. 用 Neo Hex Editor（或 HxD）開啟 `localhost.exe`
2. Ctrl+H 搜尋 `127.0.0.1`
3. 會找到 **三個** IP 位址位置（需全部修改）
4. 分別覆蓋為目標 IP（注意位址對齊）
5. 儲存

### Windows 8/10 兼容性問題處理

**症狀**：localhost.exe 無法啟動，但伺服器 log 有收到 ping

**解決方案**：
1. 解壓 `lolwut.exe`（從 `lolwut-v0.01.rar`）到客戶端資料夾
2. localhost.exe 屬性設定：
   - 相容模式：**Windows 7**
   - 色彩模式：取消降低色彩
   - 解析度：**640x480**
   - DPI：取消停用高 DPI
   - 執行：**系統管理員**
3. 使用 Fraysa's method 執行

### WZ 檔案置換注意事項

> ⚠️ **重要**：DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate some kind of bugs afterwards.

**正確流程**：
1. 用 HaRepacker 4.2.4 打開 WZ，encryption: `GMS (old)`
2. 直接用 UI 修改客戶端 WZ
3. 儲存客戶端 WZ
4. **最後**才 Export 為 XML 給伺服器端

## 📌 MapleRef Wiki - Opcode 結構重點摘要

**URL**: https://mapleref.fandom.com/wiki/Operation_Code

### v83 Opcode 核心特性
- **Opcode 為 2 bytes** (short)，早期版本為 1 byte
- 最大 Opcode 數：65,536（2^16）
- **客戶端接收 (Recv)** 和 **客戶端發送 (Send)** 有不同的 Opcode 表
- v83 大部分 Opcode 已被完整映射

### Opcode 與封包關係
```
每個客戶端操作對應一個 Opcode
Opcode 是封包的第一個字段（2 bytes）

客戶端發送 (C→S):
  Opcode (2 bytes) + 資料...

伺服器回應 (S→C):
  Opcode (2 bytes) + 資料...
```

- 完整封包結構說明

## 📌 NPC 腳本源間相容性最新討論 (RaGEZONE, 2026-03)

**URL**: https://forum.ragezone.com/threads/kinoko-npc-scripts.1261538/

### 問題背景
- 不同音源（Cosmic、Kinoko、HeavenMS）的 NPC 腳本解析器版本不同
- 較舊的腳本在新解析器可能不相容
- 主要差異：`importPackage`、`SavedLocationType` 處理

### 最佳相容性實踐

```javascript
// ✅ 最佳實踐：避免特定音源專有函數
var status = 0;

function start() {
    cm.sendNext("你好！");
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
        return;
    }
    
    if (mode == 0 && type != 1) {
        cm.dispose();
        return;
    }
    
    if (mode == 1)
        status++;
    else
        status--;
        
    // 使用普遍的 cm. 函數
    if (status == 1) {
        cm.sendSimple("選擇：\r\n#L0#選項1#l\r\n#L1#選項2#l");
    } else if (status == 2) {
        if (selection == 0) {
            cm.warp(100000000);
        } else {
            cm.dispose();
        }
    }
}
```

### 腳本相容性矩陣

| 函數/寫法 | Cosmic | Kinoko | HeavenMS |
|-----------|--------|--------|---------|
| `cm.sendNext()` | ✅ | ✅ | ✅ |
| `cm.dispose()` | ✅ | ✅ | ✅ |
| `cm.warp(mapId)` | ✅ | ✅ | ✅ |
| `importPackage()` | ❌ | ❌ | ❌ |
| `SavedLocationType` | ⚠️ | ⚠️ | ⚠️ |
