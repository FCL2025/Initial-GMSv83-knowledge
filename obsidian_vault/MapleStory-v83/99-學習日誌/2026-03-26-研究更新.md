### Hex 編輯 localhost.exe 核心原則 (maplesecrets)

**來源**: http://maplesecrets.blogspot.com/2013/04/how-to-hex-edit-clients-easily.html

**三層修改方法**：
1. **Cheat Engine + OllyDbg**：動態修改記憶體位址
2. **Hex 編輯器 (HxD)**：直接修改 .exe 中的 IP 位址
3. **OllyDbg 複製到執行檔**：將記憶體修改寫回磁片

**核心要點**：
- v83 localhost.exe 中的 IP 以 **明文** 形式儲存（可直接 Ctrl+F 搜尋 "127.0.0.1"）
- 需修改 **三處** IP 位址
- IP 長度必須一致，否則需補 \x00 (NULL)

### botnored - v83 封包類別架構 (botnored.weebly.com)

**來源**: https://botnored.weebly.com/blog/maplestory-classes-v83

**重要原則**：
> **第一個發送/接收的封包永遠是純文字（未加密）**

**客戶端主要類別**：
| 類別 | 功能 |
|------|------|
| `CWvsContext` | 世界狀態管理（角色資料、伺服器列表） |
| `CClientSocket` | 網路連接管理（Send/Recv） |
| `CField` | 當前地圖/遊戲場景處理 |
| `CLogin` | 登入畫面處理 |

**封包流向**：
```
CClientSocket::ProcessPacket()
    ↓
根據 Header 分發
    ↓
CField::OnPacket() 或 CLogin::OnPacket()
    ↓
處理具體 Opcode
```

### MapleRef Wiki - Send/Recv Opcode 核心概念

**來源**: https://mapleref.fandom.com/wiki/Operation_Code

> "Transmitting from the server and receiving on the client has one set of agreed upon Opcodes, where transmitting from the client and receiving on the server has yet another."

| 方向 | 說明 | Opcode 集合 |
|------|------|------------|
| **S→C (Send)** | 伺服器發送到客戶端 | Server Opcode Set |
| **C→S (Recv)** | 客戶端發送到伺服器 | Client Opcode Set |

**重要**：同一個 Opcode 數字可能代表不同含義，視方向而定！

### CClientSocket::ProcessPacket 封包分發流程

```
網路接收 (TCP)
    ↓
CClientSocket::ProcessPacket()
    ↓  MapleCrypto header 解密 → 提取 opcode (2 bytes)
    ↓
switch(opcode)
    ↓
CLogin::OnPacket()      // 登入畫面
    或
CField::OnPacket()      // 遊戲中
    ↓
呼叫具體的封包處理函數
```

## 🆕 4. WZ 修改

### Cosmic WZ 修改核心原則 (2026)

**來源**: https://github.com/P0nk/Cosmic

> **永遠從 client WZ 匯出到 server XML，不要反向操作！**

**正確流程**：
```
1. HaRepacker 開啟 client WZ（encryption: GMS (old)）
2. 使用 UI 直接修改數值（❗不要刪除/重新導入節點！）
3. 儲存 client WZ
4. HaRepacker → "Private Server..." 導出
5. 覆蓋伺服器的 WZ XML 目錄
```

**為什麼不能反向操作？**
- client WZ 包含 sprite 資源（圖片、音效等）
- server XML 只有邏輯數據（文字、數值）
- 從 server XML 匯入 client WZ 會造成 sprite 丢失

### Quest lvmax 批量移除 Regex (RaGEZONE, 2025-07)

**來源**: https://forum.ragezone.com/threads/maplestory-v83-method-to-disable-quest-level-restrictions.1250107/

**Notepad++ Regex**：
```
尋找：lvmax="\d+"\s*
取代：（留空）
```

### 創建自訂 NPC 完整流程

```
1. HaRepacker 導出 NPC.wz 為 XML
2. 在 XML 中新增自訂 NPC 的 frame 和 origin 設定
3. 複製 0001337.img 作為模板，修改尺寸
4. 將修改後的 img 放入 NPC.wz
5. 在伺服器端編寫對應的 .js 腳本
```

### 動態 NPC 商店腳本源 (RaGEZONE, 2016)

**來源**: https://forum.ragezone.com/threads/dynamic-npc-shop-script-v83.1089979/

**優點**：
- 不需修改 SQL 資料庫表格
- 不需重啟伺服器
- 動態調整商品和價格

```javascript
var status = 0;
var items = Array(
    Array(1302000, 10000, 1),
    Array(1302001, 20000, 1),
    Array(1302002, 30000, 1)
);

function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && type == 1) {
            cm.dispose();
            return;
        }
        if (mode == 1) status++;
        else status--;

        if (status == 0) {
            var msg = "歡迎光臨！\r\n請選擇商品：\r\n";
            for (var i = 0; i < items.length; i++) {
                msg += "#L" + i + "##i" + items[i][0] + "# - " + items[i][1] + " 金幣#l\r\n";
            }
            cm.sendSimple(msg);
        } else if (status == 1) {
            var item = items[selection];
            if (cm.getMeso() >= item[1]) {
                if (cm.canHold(item[0])) {
                    cm.gainMeso(-item[1]);
                    cm.gainItem(item[0], item[2]);
                    cm.sendOk("購買成功！");
                } else {
                    cm.sendOk("背包空間不足！");
                }
            } else {
                cm.sendOk("金幣不足！");
            }
            cm.dispose();
        }
    }
}
```

### 職業進階 NPC 腳本源

```javascript
var status = 0;
var jobName = "";
var jobId = 0;

function start() {
    cm.sendNext("恭喜你達到了 #b10級#k！現在可以進行 #b二轉#k 了。\r\n" +
        "#b#L0#火與毒法師#l\r\n#L1#冰與雷法師#l\r\n#L2#僧侶#l");
}

function action(mode, type, selection) {
    if (mode != 1) {
        if (mode == 0) cm.sendOk("好吧，等你想好了再來找我。");
        cm.dispose();
        return;
    }
    status++;
    if (status == 1) {
        if (selection == 0) { jobName = "火與毒法師"; jobId = 210; }
        else if (selection == 1) { jobName = "冰與雷法師"; jobId = 220; }
        else if (selection == 2) { jobName = "僧侶"; jobId = 230; }
        cm.sendYesNo("你確定要選擇 #b" + jobName + "#k 嗎？");
    } else if (status == 2) {
        cm.gainJob(jobId);
        cm.sendOk("恭喜！你現在是 #r" + jobName + "#k 了！");
        cm.dispose();
    }
}
```
