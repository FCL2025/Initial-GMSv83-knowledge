## 二、MapleSolaxiaV2 完整架設指南（新發現）

**來源**: https://github.com/skhzhang/MapleSolaxiaV2

### 2.1 環境需求
| 軟體 | 版本 | 用途 |
|------|------|------|
| JDK/JRE | 7 (Java 7) | 伺服器運行 |
| NetBeans | 8.0.2 | 開發IDE |
| WampServer | 2.0i | MySQL資料庫 |
| MySQL Workbench | 5.2.39 | 資料庫管理 |
| Hamachi | - | 網路隧道（可選）|

**重要**: JCE Policy 需替換為 Unlimited Strength 版本

### 2.2 SQL 安裝流程
```
1. File > Open Script > db_database.sql → 執行
2. File > Open Script > db_drops.sql → 執行
3. (可選) File > Open Script > db_shopupdate.sql → 執行
```

### 2.3 客戶端完整架設步驟（新發現）
```
1. 使用 ManagerMsv83.exe 安裝 MapleStory
2. 刪除: HShield資料夾、ASPLauncher.exe、MapleStory.exe、patcher.exe
3. 解壓 localhost.exe 到客戶端資料夾
4. 用修改過的 WZ 檔案覆蓋（client_wz 資料夾）
5. 如需自訂IP：用 Neo Hex Editor 開啟 localhost.exe
   - 搜尋 "127.0.0.1" → 替換為目標IP
```

### 4.2 MaplePacketPuller（Python）
**GitHub**: https://github.com/Bratah123/MaplePacketPuller

- Python 封包結構分析工具
- 自動解析封包 opcode

### 4.3 封包結構（v83）
```
封包頭部 (4 bytes):
  - MapleCrypto 混淆
  - 包含長度 + 混淆 opcode

Opcode (2 bytes, little endian):
  - 例如: 0x0001 → 01 00

第一個封包永遠明文（未加密）
```

## 五、Opcode 結構（mapleref.fandom.com）

### 5.1 v83 Opcode 特點
- Opcode 為**第一個 2 bytes**（早期版本為 1 byte）
- 最大可用 Opcode: 65,536 (2^16)
- 兩組不同Opcode：**Send (C→S)** 和 **Recv (S→C)**

### 5.2 封包頭部混淆機制
MapleStory 使用 MapleCrypto 頭部混淆：
- 4 bytes 頭部
- 包含長度資訊
- 包含混淆後的 opcode

### 5.3 IDA 分析技巧
所有客戶端Recv封包都經過:
```
CClientSocket::ProcessPacket → CField::OnPacket / CLogin::OnPacket
```
因此找 packet handler 很容易。

### 6.1 主要 WZ 檔案
| 檔案 | 內容 |
|------|------|
| Base.wz | 武器、技能、音效 |
| Character.wz | 角色、精靈穿戴 |
| Effect.wz | 傷害數字、墓碑、升級特效 |
| Etc.wz | 任務、BUFF、角色創建、技能資訊 |
| Item.wz | 所有物品圖片 |
| Map.wz | 所有地圖視覺 |
| Mob.wz | 所有怪物 |
| Npc.wz | NPC 資訊 |
| Quest.wz | 任務資料 |
| Sound.wz | 音樂音效 |
| String.wz | 文字字串 |
| Skill.wz | 技能 |

### 6.2 WZ 加密機制
```
ASCII: char ^ (0xAA + offset) ^ key[offset]
Unicode: char ^ (0xAAAA + offset) ^ ((key[offset * 2 + 1]) << 8) + key[offset * 2]
```
