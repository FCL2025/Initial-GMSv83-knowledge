### 3. MapleStory 封包加密三層架構

```
┌─────────────────────────────────────────────────────┐
│ Layer 1: MapleCrypto (頭部混淆)                     │
│   - 4 bytes header (混淆 opcode + length)           │
│   - 防止靜態分析 opcode                             │
│                                                        │
│ Layer 2: Shanda Encryption (對稱加密)               │
│   - IOBufferManipulator.Encrypt(pBuff)             │
│   - 在 AES 之前應用                                 │
│   - v83 仍在使用，v118+ 已移除                       │
│                                                        │
│ Layer 3: AES-OFB (主加密)                          │
│   - v83 主要加密方式                                │
│   - 256-bit key, 4-byte IV                          │
│   - 版本決定的靜態金鑰                              │
└─────────────────────────────────────────────────────┘
```

**v83 AES Key 參考 (CMS/Chinese MS v83)**：
```java
// AES UserKey (first column per row)
0x13, 0x08, 0x06, 0xB4, 0x1B, 0x0F, 0x33, 0x52

// IV Transformation Key: 0xC65053F2
// 客戶端版本: 0x23 (35 decimal)
```

### 4. MapleStory v83 Opcode 結構分析

#### CClientSocket::ProcessPacket 封包分發流程
```
網路接收 (TCP)
    ↓
CClientSocket::ProcessPacket()
    ↓  根據 MapleCrypto header 解密
    ↓  提取 opcode (2 bytes, little endian)
    ↓
switch(opcode)
    ↓
CLogin::OnPacket()      // 登入畫面狀態
    或
CField::OnPacket()      // 遊戲中狀態
```

#### v83 常用 Opcode 對照表

**登入相關 (C→S)**:
| Opcode (Hex) | 名稱 | 說明 |
|---|---|---|
| 01 00 | LOGIN_PASSWORD | 登入請求 |
| 02 00 | LOGIN_CHECK | 登入檢查 |
| 06 00 | SERVERLIST_REQUEST | 請求伺服器列表 |
| 07 00 | CHARLIST_REQUEST | 請求角色列表 |
| 08 00 | CHAR_SELECT | 選擇角色 |

**遊戲相關**:
| Opcode (Hex) | 名稱 | 說明 |
|---|---|---|
| 14 00 | MOVE_PLAYER | 移動 |
| 15 00 | CLOSE_RANGE_ATTACK | 近距離攻擊 |
| 28 00 | NPC_TALK | NPC 對話 |
| 6F 00 | SHOP_BUY | 商場購買 |

### 5. NPC 腳本編寫進階

#### 動態 NPC 商店腳本 (不需 SQL)
**來源**: https://forum.ragezone.com/threads/dynamic-npc-shop-script-v83.1089979/

**優點**：
- 即時生效
- 可版本控制
- 不需重啟伺服器
- 可動態定義物品、價格、數量

**基本結構**：
```javascript
var status = 0;
var items = Array(
    Array(4000000, 1000000, 1),  // itemId, price, quantity
    Array(4000001, 500000, 1),
    Array(4000002, 200000, 1)
);

function start() {
    cm.sendSimple("歡迎來到商店！\n#b" + 
        "#L0#購買物品#l\n" +
        "#L1#出售物品#l\n" +
        "#L2#離開#l");
}

function action(mode, type, selection) {
    if (mode == 1) {
        if (selection == 0) {
            // 顯示購買選項
            var selStr = "請選擇要購買的物品：\n";
            for (var i = 0; i < items.length; i++) {
                selStr += "#L" + i + "#" + items[i][0] + " - " + items[i][1] + " 楓幣#l\n";
            }
            cm.sendSimple(selStr);
        } else if (selection == 1) {
            cm.sendOk("出售功能待實現");
        } else if (selection == 2) {
            cm.dispose();
        }
    } else {
        cm.dispose();
    }
}
```
