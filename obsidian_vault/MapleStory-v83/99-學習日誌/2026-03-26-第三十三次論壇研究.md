## 1. 登入器/客戶端修改

### MapleEzorsia-v2 (GitHub 2026) - 免 Hex 編輯 HD 方案

**GitHub**: https://github.com/444Ro666/MapleEzorsia-v2

**突破性進展**：這是 2026 年最新方案，實現真正的**免 Hex 編輯**！

**原理**：
- 這個 DLL 修改一個乾淨的官方 v83 客戶端的記憶體位址
- 使遊戲能夠以 localhost 模式運行
- 同時將遊戲解析度改為 HD

**使用方法**：
```bash
1. 下載乾淨的官方 v83 MapleStory 客戶端
2. 安裝 Themida 或手動脫殼
3. 注入 MapleEzorsia-v2.dll
4. 啟動客戶端，連接到 127.0.0.1
```

**修改的位址類型**：
- 伺服器 IP/Port 導向 localhost
- 解析度修改（寬屏/HD）
- 去除版本檢查

**與傳統 Hex 修改的比較**：

| 方案 | 優點 | 缺點 |
|------|------|------|
| HxD Hex 修改 | 簡單直接 | 需手動找位址 |
| DLL 注入 (Ezorsia) | 免修改、可還原 | 需脫殼 |
| Themida + DLL | 安全、難分析 | 複雜度較高 |

### MapleStory v83 封包加密三層架構

```
┌─────────────────────────────────────────────────────┐
│ Layer 1: MapleCrypto (頭部混淆)                      │
│   - 4 bytes header (混淆 opcode + length)            │
│   - 防止靜態分析 opcode                             │
│                                                        │
│ Layer 2: Shanda Encryption (對稱加密)                 │
│   - Nexon 內部名稱：IOBufferManipulator             │
│   - 已於 v118+ 移除                                 │
│                                                        │
│ Layer 3: AES-OFB (主加密)                           │
│   - v83 主要加密方式                                │
│   - 256-bit key, 4-byte IV                          │
└─────────────────────────────────────────────────────┘
```

### MapleCrypto Header 加密/解密公式

**加密端（發送方）**:
```java
public byte[] getPacketHeader(int opcode, int length, int iv) {
    int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
    verify = (verify << 3) | (verify >>> 5);
    verify &= 0xFFFF;
    return new byte[] {
        (byte) (verify & 0xFF),
        (byte) ((verify >>> 8) & 0xFF),
        (byte) (length & 0xFF),
        (byte) ((length >>> 8) & 0xFF)
    };
}
```

**解密端（接收方）**:
```java
public int retrieveOpcode(byte[] header, int iv) {
    int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
    verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
    return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
}
```

### 動態 NPC 商店腳本（無需 SQL）

```javascript
/*
 * 動態 NPC 商店 - 不需要 SQL
 * NPC ID: 9010009
 */
var status = 0;
var items = Array(
    Array(1302000, 10000, 1),   // [物品ID, 價格, 庫存]
    Array(1302001, 20000, 1),
    Array(1302002, 30000, 1)
);

function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && type == 1) {
            cm.dispose();
            return;
        }
        if (mode == 1) status++;
        else status--;

        if (status == 0) {
            var msg = "歡迎光臨！\r\n請選擇商品：\r\n";
            for (var i = 0; i < items.length; i++) {
                msg += "#L" + i + "##i" + items[i][0] + "# - " + items[i][1] + " 金幣#l\r\n";
            }
            cm.sendSimple(msg);
        } else if (status == 1) {
            var item = items[selection];
            if (cm.getMeso() >= item[1]) {
                if (cm.canHold(item[0])) {
                    cm.gainMeso(-item[1]);
                    cm.gainItem(item[0], item[2]);
                    cm.sendOk("購買成功！");
                } else {
                    cm.sendOk("背包空間不足！");
                }
            } else {
                cm.sendOk("金幣不足！");
            }
            cm.dispose();
        }
    }
}
```
