## 1. 登入器 / 客戶端修改

### 🆕 MaplePatcher - 現代客戶端更新系統 (RaGEZONE)

**來源**：https://forum.ragezone.com/threads/maple-patcher.1170344/

MaplePatcher 是一個為 MapleStory 設計的現代化修補系統，約 90% 完成度：

| 功能 | 說明 |
|------|------|
| CRC32 完整性檢查 | 對比本地檔案與遠端主機的 CRC32，確保一致性 |
| 支援 .WZ 和解壓客戶端 | 可同時支援封裝和 Extract 兩種客戶端形態 |

| 自動下載 + 驗證 | 下載後自動校驗，失敗自動重試 |

**工作流程**：
```
玩家啟動 → MaplePatcher 檢查本地 CRC32
  → 對比遠端清單 → 下載差異更新 → 驗證 → 啟動遊戲
```

### 🆕 MapleShark ScriptDotNet (S#) 腳本引擎

GitHub: https://github.com/diamondo25/MapleShark

MapleShark 的腳本引擎使用 **ScriptDotNet (S#)**，用於自訂封包解析：

```csharp
// ScriptDotNet 語法範例
byte AddByte(string name)
short AddShort(string name)
int AddInt(string name)
float AddFloat(string name)
string AddPaddedString(string name, int length)
void StartNode(string name)  // 建立子節點
void EndNode(bool expand)    // 結束節點
void Write(string file, string line)  // 寫入檔案
int Remaining()              // 返回剩餘未處理位元組數
```

**使用範例**：
```csharp
// 解析登入封包
int opcode = AddShort("Opcode");
string username = AddPaddedString("Username", 13);
string password = AddPaddedString("Password", 13);
```

### 🆕 職業進階 NPC 腳本源 (RaGEZONE, 2015)

**來源**：https://forum.ragezone.com/threads/i-need-v83-moopledev-job-advance-npc-script.1084681/

```javascript
// 法師二轉職業選擇
var status = 0;
function action(mode, type, selection) {
    if (mode != 1) { cm.dispose(); return; }
    status++;
    if (status == 1) {
        // 顯示選擇選單
        cm.sendNext("恭喜 #b10級#k！選擇法師類型：\r\n#b#L0#火與毒法師#l\r\n#L1#冰與雷法師#l\r\n#L2#僧侶#l");
    } else if (status == 2) {
        var jobId = [210, 220, 230][selection];
        cm.gainJob(jobId);
        cm.dispose();
    }
}
```
