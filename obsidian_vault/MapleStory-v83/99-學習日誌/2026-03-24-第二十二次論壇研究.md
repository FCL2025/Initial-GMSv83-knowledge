### 12. WZ 編輯重點提醒

**HaRepacker 使用注意事項**：
1. 編輯 WZ 時使用加密格式 `GMS (old)`
2. 不要直接刪除/重新導入節點，可能造成數據丟失
3. 建議使用 HaRepacker UI 直接編輯數值
4. 客戶端修改後需同步到伺服器的 XML

**編輯流程**：
```
1. HaRepacker 開啟客戶端 WZ
2. 進行所需修改
3. 儲存
4. 重新導出 XML 到伺服器目錄
```

### 13. 客戶端修改原理總結

**修改目的**：讓客戶端連接到私人伺服器而非官方伺服器

**修改位置**：
| 位置 | 說明 |
|------|------|
| `MapleStory.exe` | 主執行檔，包含連線 IP |
| `login.cfg` | 某些版本的設定檔 |

**修改工具**：
- HxD（輕量級 Hex 編輯器）
- 010 Editor
- Ezorsia DLL 注入（無需修改原始檔案）

### 14. NPC 腳本與 WZ 編輯的結合

**創建自訂 NPC 流程**：
1. 使用 HaRepacker 導出 NPC WZ 為 XML
2. 在 XML 中新增自訂 NPC 的 frame 和 origin 設定
3. 複製 `0001337.img` 作為模板，修改尺寸
4. 將修改後的 img 放入 `NPC.WZ`
5. 在伺服器端編寫對應的 `.js` 腳本

### 15. 登入器/客戶端修改學習路徑

```
Level 1: 使用現成 localhost 客戶端
  → 只需修改 IP（HxD）
  
Level 2: 自訂客戶端功能
  → 修改 WZ 檔案（HaRepacker）
  → 修改字串（STRedit）
  
Level 3: 進階客戶端修改
  → IDA 分析
  → Themida 脫殼
  → 去除保護
```
