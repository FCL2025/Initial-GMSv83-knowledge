## 論壇最新調研摘要

### 1. 登入器/客戶端修改

**MapleSolaxiaV2 (jonnylin13/Maple83) 客戶端設置流程：**
- 使用 `ManagerMsv83.exe` 安裝客戶端到 `C:\Nexon\MapleStory`
- 刪除 `HShield` (資料夾)、`ASPLauncher.exe`、`MapleStory.exe`、`patcher.exe`
- 用 `localhost.exe` 替換，連接到本地伺服器
- WZ 檔案替換為 `client_wz` 資料夾中的版本

**IP 修改方式：**
- 需要用十六進位編輯器（如 Neo Hex Editor）修改 `localhost.exe` 中的 IP
- 搜尋 "Text String" "127.0.0.1"，在 3 個地址處進行修改

**相容性問題處理（Windows 8/10）：**
- 若 `localhost.exe` 無法開啟，放置 `lolwut.exe` 到客戶端資料夾
- 設定相容性模式：Windows 7, 640x480 解析度，管理員權限執行

**v83 客戶端特點：**
- v83 客戶端高度修改過，不反映其他版本工作流程
- 建議新手從較高版本入手學習基礎

### 2. 抓包分析 (Packet Capture)

**基礎知識：**
- 第一個發送/接收的封包永遠是純文字（未加密）
- MapleStory 封包通過 TCP socket 傳輸
- Little Endian 格式：位元組順序與一般十六進位表示相反

**工具推薦：**
- Wireshark：用於抓包分析
- IDA：用於取得封包結構和 Opcode（進階用法）
- MaplePacketPuller (GitHub Bratah123)：Python 封包結構分析工具

**抓包要點：**
- 使用 `localhost.exe` 連接本地 127.0.0.1:8484
- 攔截工具（如 Caesar 等）可修改 IP 和埠口
- 只要修改 opcode + AES key 即可通用於不同版本

### 3. Opcode 結構

**Opcode 基礎：**
- Opcode 是封包的第一個 2 bytes（short）
- 早期版本使用 1 byte，v83 已升級為 2 bytes (0x0000-0xFFFF)
- 客戶端和伺服器有各自獨立的 Opcode 表

**Opcode 分析方法：**
- 使用 IDA 靜態分析客戶端二進位
- 使用 Wireshark 動態分析封包流
- MapleRef Fandom Wiki 有 Opcode 參考資料

**v83 專用 Opcode：**
- 需配合特定伺服器端實作（如 MapleSolaxiaV2、HeavenMS）
- 不同私人伺服器版本 opcode 可能略有差異

### 4. WZ 修改

**編輯工具：**
- **HaRepacker 4.2.4**：首選 WZ 編輯器，加密模式選 `GMS (old)`
- 禁止使用伺服器的 XML 重新導入客戶端 WZ，會造成 Bug

**正確編輯流程（重要）：**
1. 使用 HaRepacker UI 進行修改
2. 保存修改後的 WZ 到客戶端資料夾
3. 使用 HaRepacker 的 "Private Server..." 導出選項
4. 將導出的 XML 重新導入伺服器端 WZ.XML

**WZ 檔案結構：**
- `String.wz`：字串、節點名稱、對話文字
- `Map.wz`：地圖、腳踏石（Foothold）數據
- `Skill.wz`：技能數據
- `Item.wz`：物品屬性

**Foothold 結構：**
```
next → 離開此腳踏石後下一個前往的腳踏石
X1, X2 → 起點/終點 X 座標
Y1, Y2 → 起點/終點 Y 座標
（foothold 0 不存在）
```

### 5. NPC 腳本編寫

**腳本語言：**
- MapleStory NPC 腳本使用 JavaScript 語法（ECMAScript）
- 與 Java 不同，雖然兩者語法相似

**核心函數：**
```javascript
function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose(); // 結束對話
    } else {
        if (mode == 0 && status == 0) {
            cm.dispose(); // 選 No 結束
        }
        if (mode == 1) {
            status++;
        } else {
            status--;
        }
        
        if (status == 0) {
            cm.sendYesNo("你想要一把免費的劍嗎？");
        } else if (status == 1) {
            cm.gainItem(1302000, 1); // 給予物品
            cm.sendOk("祝你好運！");
        }
    }
}
```

**常用方法：**
- `cm.sendYesNo(text)`：Yes/No 對話框
- `cm.sendOk(text)`：確定對話框
- `cm.sendNext(text)`：下一步對話
- `cm.sendAcceptDecline(text)`：接受/拒絕對話框
- `cm.gainItem(itemId, quantity)`：給予物品
- `cm.gainMeso(amount)`：給予金幣
- `cm.dispose()`：結束腳本
- `cm.getJob()`：獲取職業
- `cm.getLevel()`：獲取等級

**腳本存放位置（MapleSolaxiaV2）：**
- `source/scripts/npc/`
