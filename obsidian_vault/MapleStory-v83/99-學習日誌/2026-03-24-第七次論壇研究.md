### 1. 完整 v83 私人伺服器架設指南 (RaGEZONE, 2025-11)

**來源**：https://forum.ragezone.com/threads/video-guide-how-to-make-a-maplestory-v83-private-server-–-full-setup-tutorial.1243566/

**環境需求**：
- JDK 8 (Java SE 8)
- JCE 8 (Java Cryptography Extension)
- MySQL Server 5.5
- v83 localhost 客戶端
- HxD 或 WinHex（修改 IP）

**完整架設流程**：
```
1. 安裝 JDK 8 + JCE 8
2. 安裝 MySQL Server 5.5
3. 下載 v83 localhost 客戶端
4. 使用 HxD 修改 IP (127.0.0.1 → 目標 IP)
5. 下載並編譯 Cosmic 或 HeavenMS 音源
6. 匯入 SQL 資料庫結構
7. 配置 server.properties (IP、端口、經驗倍率等)
8. 啟動伺服器 + 客戶端
```

### 3. MapleStory v83 封包類別結構 (botnored)

**來源**：https://botnored.weebly.com/blog/maplestory-classes-v83

**重要原則**：
> **第一個發送/接收的封包永遠是純文字（未加密）**

**封包傳輸方式**：
- TCP Socket 協定
- 第一個封包（Hello Packet）為明文
- 之後所有封包使用 AES-OFB 加密

**類別對照**：
| 類別前綴 | 說明 |
|---------|------|
| `CWvsContext` | 世界狀態管理 |
| `CClientSocket` | 網路連接 |
| `CField` | 當前地圖/遊戲場景 |
| `CLogin` | 登入相關 |

### 7. v83 商場 Opcode 0x6F 結構分析 (PDFCOFFEE)

**來源**：https://pdfcoffee.com/v83-dupe-pdf-free.html

**Opcode 0x6F (SHOP_BUY) 結構**：
```
6F 00 20 00 FF FF FF 80 00
```
- 購買物品，Mesos 寫入 `0x80000000`（最大 Mesos）

> ⚠️ 僅供學習研究用途

### 8. MapleRef Wiki - Opcode / Packet 結構（最完整參考）

**Packet Wiki**：https://mapleref.fandom.com/wiki/Packets
**Opcode Wiki**：https://mapleref.fandom.com/wiki/Operation_Code

**Packet Header 格式**：
- Header 為 4 bytes
- 使用 SendIV、封包長度、版本號生成
- 版本號發送到客戶端時為 `-(version + 1)`
- Opcode 通常為 2 bytes

## 📊 Opcode 更新 (v83 完整對照)

### 登入相關
| Opcode | 名稱 | 說明 |
|--------|------|------|
| 0x00 | LOGIN_STATUS | 登入結果 |
| 0x01 | LOGIN_PASSWORD | 登入請求 |
| 0x02 | ACCEPT_LOGIN | 同意登入 |
| 0x05 | SELECT_CHARLIST | 選擇角色列表 |
| 0x06 | SERVERLIST_REQUEST | 請求伺服器列表 |
| 0x07 | CHARLIST_REQUEST | 請求角色列表 |
| 0x08 | CHAR_SELECT | 選擇角色 |
| 0x0B | PLAYER_DROP | 進入遊戲 |

### 遊戲相關
| Opcode | 名稱 | 說明 |
|--------|------|------|
| 0x03 | CHARLIST | 角色列表 |
| 0x04 | CHAR_INFO | 角色資訊 |
| 0x14 | MOVE_PLAYER | 移動 |
| 0x15 | CLOSE_RANGE_ATTACK | 近距離攻擊 |
| 0x16 | RANGED_ATTACK | 遠距離攻擊 |
| 0x28 | NPC_TALK | NPC 對話 |
| 0x6A | MOB_APPLY_CONTROL | 怪物生成+HP顯示 |

### 商場/夫妻
| Opcode | 名稱 | 說明 |
|--------|------|------|
| 0x6F | SHOP_BUY | 商場購買 |
| 0x6C | SHOP_OPEN | 開啟商店 |
| 0x6D | SHOP_SELL | 出售物品 |
| 0x79 | ON_COUPLE_MESSAGE | 接收夫妻訊息 (S→C) |
| 0x88 | COUPLE_MESSAGE | 發送夫妻訊息 (C→S) |
