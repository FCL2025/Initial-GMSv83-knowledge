**v83 專用抓包工具鏈 (2026 最新)**
| 工具 | 類型 | 特色 |
|------|------|------|
| Maple_Pshark | 即時 Hook | 直接截獲明文，繞過加密 |

- 支援封包修改、注入、規則過濾
- Hook 加密函數，直接獲取明文

**完整抓包工作流程**:
```
客戶端 → localhost.exe → Maple_Pshark (即時截獲)
                         ↓
                    MaplePacketPuller (結構分析)
                         ↓
                    MapleRef Wiki (Opcode 對照)
```

### 4. WZ 修改

**HaRepacker-resurrected 最新版本**
- GitHub: https://github.com/lastbattle/Harepacker-resurrected
- GitHub Releases: https://github.com/lastbattle/Harepacker-resurrected/releases
- 官方網站: https://lastbattle.github.io/Harepacker-resurrected/
- v83 WZ 加密格式需選擇 **GMS (old)**

**WZ 編輯黃金法則**:
> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

**正確流程**:
1. HaRepacker 開啟 GMS (old) 加密的客戶端 WZ
2. 使用 UI 直接修改數值（禁止刪除/重新導入節點！）
3. 儲存 WZ
4. 選擇 "Private Server..." 導出 XML
5. 將導出的 XML 覆蓋到伺服器的 WZ.XML 目錄
