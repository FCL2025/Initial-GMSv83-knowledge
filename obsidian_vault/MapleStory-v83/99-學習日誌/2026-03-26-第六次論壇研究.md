## 🆕 MapleStory v83 客戶端修改 - MapleEzorsia v2 最新方案 (2026)

**GitHub**: https://github.com/444Ro666/MapleEzorsia-v2

### MapleEzorsia v2 突破性進展
這是 2026 年最新的 v83 客戶端修改方案，實現了**真正的免 Hex 編輯**！

**核心特性**：
- 使用 dinput8.dll 自動加載（無需注入器）
- 同時完成：去除保護 + HD 解析度 + localhost 導向
- 可自訂解析度（擺脫 800x600 限制）
- 完全開源，C++ 源代碼可在 GitHub 查看

### 使用方式（超簡單）
```bash
1. 從 Nexon 安裝原始 v83 客戶端
2. 將 dinput8.dll 放入遊戲目錄
3. 啟動 MapleStory.exe（自動加載 DLL）
4. 享受 HD 解析度 + localhost 連線
```

### MapleEzorsia v2 與傳統 Hex 修改比較

| 方案 | 優點 | 缺點 |
|------|------|------|
| HxD Hex 修改 | 簡單直接 | 需手動找位址 |
| DLL 注入 (Ezorsia) | 免修改、可還原 | 需脫殼 |
| Themida + DLL | 安全、難分析 | 複雜度較高 |

### 支援環境

| 環境 | 支援狀態 |
|------|---------|
| Python 3.8.5 | ✅ 測試通過 |
| Python 3.6.12 & 3.8.6 | ✅ 測試通過 |
| IDA Pro 7.0 (32-bit) | ✅ 測試通過 |
| IDA Pro 7.5 | ✅ 測試通過 |
| IDA Pro 6.8 | ✅ 測試通過 |
| Python 2.7 | ❌ 不相容 |

### 完整工作流程
```
Step 1: IDA Pro 反編譯
  → 在 IDA 中找到 CField::OnPacket 或 CLogin::OnPacket
  → 產生 pseudocode (.p64 或 .txt)

Step 2: 複製 pseudocode
  → 在 Functions/ 目錄建立 .txt 檔案
  → 貼上 IDA 產生的 C-style pseudocode

Step 3: Python 分析
  → cd MaplePacketPuller/IDA\ Maple\ Script/src/main/python
  → python main.py
  → 輸入分析類型（完整/標準）
  → 輸入要分析的檔案名稱

Step 4: 輸出結果
  → 封包結構自動寫入 FuncOutput/ 目錄
  → 控制台顯示 Decode1/Decode2/Decode4/DecodeStr 等欄位
```

## 🆕 v83 完整架設流程摘要 (RaGEZONE 2025-11 最新)

**來源**: https://forum.ragezone.com/threads/video-guide-how-to-make-a-maplestory-v83-private-server-–-full-setup-tutorial.1243566/

### 環境需求
| 軟體 | 版本 | 必要性 |
|------|------|--------|
| JDK | 8 | 必要 |
| JCE | 8 | 必要（加密） |
| MySQL | 5.5 | 必要 |
| v83 localhost 客戶端 | - | 必要 |
| HxD | - | 修改 IP 用 |

### 完整架設流程
```
1. 安裝 JDK 8 + JCE 8
2. 安裝 MySQL Server 5.5
3. 下載 v83 localhost 客戶端
4. 架設 Java 伺服器（NetBeans / IntelliJ）
5. 執行 SQL 初始化資料庫
6. 設定 ServerConstants.java 中的 IP 和埠口
7. HxD 修改客戶端 IP（三個 127.0.0.1 全部替換）
8. 啟動客戶端和伺服器
```
