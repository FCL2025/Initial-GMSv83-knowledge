### 1. 登入器/客戶端修改

**最新 Hex 編輯 IP 修改流程 (RaGEZONE, 2024-2025)**：
- 使用 HxD 或 Neo Hex Editor 開啟 `localhost.exe`
- `Ctrl+H` 搜尋 `127.0.0.1`
- 會找到三個 IP 位址位置，全部修改為目標 IP
- 注意位址對齊問題

- 適用於長度不同的 IP 修改

### 3. Opcode 結構

**v83 封包三層加密架構**：
```
明文 → Shanda 加密 → AES-OFB → MapleCrypto header → 網路
```

**MapleCrypto 頭部混淆**：
- 4-byte header (混淆 opcode + length)
- 位旋轉 3 位 + XOR 運算

**雙 IV 機制**：
- 客戶端 SendIV/RecvIV
- 伺服器 SendIV/RecvIV
- 第一個封包（Hello Packet）為明文

**最新 Opcode 分析工具**：
- MapleRef Wiki (mapleref.fandom.com)
- MaplePacketPuller + SpiritIDAPlugin (Python + IDA)
- RustMS (完整加密實現開源)

### 4. WZ 修改

**HaRepacker 導出/導入流程**：
```
1. 開啟 GMS (old) 加密的 WZ
2. 使用 UI 直接編輯數值（禁止刪除/重新導入！）
3. 儲存 WZ
4. 選擇 "Private Server..." 導出 XML
5. 覆蓋到伺服器的 WZ.XML 目錄
```

**重要警告**：
> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

**最新 WZ 編輯參考**：
