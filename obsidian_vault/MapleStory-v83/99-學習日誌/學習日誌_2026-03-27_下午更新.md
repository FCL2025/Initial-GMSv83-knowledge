## 🆕 本次學習新發現

### 1. 登入器/客戶端修改

**v83 2025-2026 架設最佳實踐**：
- 推薦組合：Ezorsia V2 (HD) + Hendi's Client (4G Flag)
- 4G Flag 修改：PE Header 偏移 0x13E，0x0F → 0x2F
- 防火牆設定：
  ```cmd
  netsh advfirewall firewall add rule name="MapleStory Login" dir=in action=allow protocol=tcp localport=7575
  netsh advfirewall firewall add rule name="MapleStory Game" dir=in action=allow protocol=tcp localport=8484
  ```

### 2. 抓包分析

**完整抓包工具鏈 (2026-03 最新)**：
```
Wireshark (TCP 確認) → Maple_Pshark (Hook 明文) → MaplePacketPuller (結構分析) → MapleRef Wiki (Opcode 對照)
```

### 3. Opcode結構

**v83 Send vs Recv Opcode 核心區分**：
- S→C (Send)：伺服器發送，客戶端接收
- C→S (Recv)：客戶端發送，伺服器接收
- 同一個 Opcode 數字可能代表完全不同含義！

### 4. WZ修改

**WZ 編輯黃金法則**：
> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

正確流程：HaRepacker 開啟 GMS (old) 加密 → UI 直接修改 → Private Server 導出

### 5. NPC腳本編寫

**相容性最終確認**：
- ✅ 無 `importPackage`（JDK 7+ 已廢棄）
- ✅ 使用 `cm.` 函數
- ✅ 正確的 `status` 狀態機
