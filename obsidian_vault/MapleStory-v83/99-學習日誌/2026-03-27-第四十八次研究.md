## 🎉 重大發現 3：NX Format — WZ 的高效能替代方案

**來源**: RaGEZONE (Feb 2026) - https://forum.ragezone.com/threads/wz-and-nx-file-questions.1260605/

### 什麼是 NX 格式？
- Nexon 開發的壓縮格式，比 WZ/XML 效能更好
- 官方伺服器使用的格式
- 更快的載入速度，佔用更少磁碟空間

### NX vs WZ 比較
| 特性 | WZ | NX |
|------|-----|-----|
| 效能 | 較慢 | ✅ 更快 |
| 檔案大小 | 較大 | ✅ 更小 |
| 工具支援 | HaRepacker 直接支援 | 需要轉換 |
| 主流程度 | 廣泛使用 | 官方伺服器偏好 |

### 使用方式
- 使用 HaRepacker 開啟 .wz 檔案
- 可轉換為 .nx 格式（需找專門工具）
- 詳見 RaGEZONE NX 格式說明帖：
  - https://forum.ragezone.com/threads/the-nx-format.891992/
  - https://forum.ragezone.com/threads/converted-nx-files-for-maplestory.891937/

### HaRepacker-resurrected 對 NX 的支援
- v8.0.0+ 支援 NX 格式讀取
- IMG filesystem 提供了更靈活的工作流

## 🆕 v83 Quest Level Restrictions 移除教學 (RaGEZONE, July 2025)

**來源**: https://forum.ragezone.com/threads/maplestory-v83-method-to-disable-quest-level-restrictions.1250107/

### 步驟
1. 使用 **HaRepacker** 開啟 Quest.wz
2. 選取 `Check.img.xml` → Tools → Export selection → XML → Classic
   - （選擇 New 會使用現代版本加密，Classic 較為通用）
3. 使用 Notepad++ 打開 XML
4. 搜尋所有 `lvmax` 屬性並刪除
5. 儲存 XML 並用 HaRepacker 重新導入

### 重要提醒
- 這是手動過程，Quest.wz 可能很大
- 建議先備份原始檔案
