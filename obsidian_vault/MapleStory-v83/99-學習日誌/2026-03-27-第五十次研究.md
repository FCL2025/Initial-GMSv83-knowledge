## 1) 登入器/客戶端修改 ✅

### MapleSolaxiaV2 完整架設流程 (RaGEZONE, GitHub 2026)

**來源**: https://github.com/skhzhang/MapleSolaxiaV2

#### 客戶端修改關鍵步驟

**Step 1: 安裝客戶端**
```
- 使用 ManagerMsv83.exe 安裝 MapleStory
- 安裝到目錄 (如 C:\Nexon\MapleStory)
```

**Step 2: 移除原版檔案**
```
- 刪除 HShield (資料夾)
- 刪除 ASPLauncher.exe
- 刪除 MapleStory.exe
- 刪除 patcher.exe
```

**Step 3: 替換為 localhost.exe**
```
- 使用 MapleSolaxiaV2 提供的 localhost.exe
- 放入客戶端資料夾
- 覆蓋原來的 MapleStory.exe
```

**Step 4: 替換 WZ 檔案**
```
- 使用 client_wz 資料夾中的 WZ 檔案
- 覆蓋原來的 WZ 檔案
```

**Step 5: Hex 編輯 IP (如非 localhost)**
```
- 使用 Neo Hex Editor 開啟 localhost.exe
- 搜尋 "127.0.0.1" 字串
- 替換為目標 IP
- 需要修改 3 處 IP 位址
```

#### 兼容性問題處理 (Windows 8/10)

**lolwut.exe 用途**：
- 解決 Windows 8/10 上 localhost.exe 無法開啟的問題
- 放置到 MapleStory 客戶端資料夾
- 配合相容性設定使用

**localhost.exe 相容性設定**：
```
- 執行相容模式: Windows 7
- 解析度: 640 x 480
- 以系統管理員身份執行
- 停用高 DPI 隱藏
```

#### Hex 編輯 IP 詳細步驟 (MapleSolaxiaV2 README)

```
工具: Neo Hex Editor (free-hex-editor-neo.exe)

1. 開啟 localhost.exe
2. 搜尋 "Text String" "127.0.0.1"
3. 在找到的 3 個位址上替換 IP
4. 儲存並離開
```

## 4) WZ 修改 ✅

### MapleSolaxiaV2 WZ 修改黃金法則 (最新驗證)

**原文**：
> "DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate some kind of bugs afterwards."

**正確流程**：
```
Step 1: 用 HaRepacker 4.2.4 (GMS old 加密) 開啟客戶端 WZ
Step 2: 使用 UI 直接修改
Step 3: 儲存客戶端 WZ
Step 4: 使用 HaRepacker 的 "Private Server..." 匯出功能
Step 5: 將匯出的 XML 覆蓋伺服器的 wz.xml 檔案
```

**嚴禁**：直接用伺服器的 XML 覆蓋客戶端 WZ！

### HaRepacker 版本選擇

| 版本 | 加密支援 | 用途 |
|------|---------|------|
| **4.2.4** | GMS (old) | ✅ 推薦用於 v83 |
| 4.2.2 | 較舊加密 | 僅供參考 |
| 最新版 | 新版加密 | 可能不支援 v83 |

### MobBookUpdate 實作案例 (MapleSolaxiaV2)

這是 WZ 編輯的實際應用案例：

```
Step 1: 開啟 MobBookUpdate 專案 (NetBeans)
        位於 C:\Nexon\MapleSolaxia\MobBookUpdate

Step 2: 在 lib/ 資料夾複製 MonsterBook.img.xml (原始 v83)

Step 3: 貼上到 dist/ 資料夾

Step 4: 在 dist/ 執行:
        java -jar MobBookUpdate.jar
        → 產生 MonsterBook_updated.img.xml

Step 5: 用 HaRepacker 開啟客戶端 String.wz
        刪除舊的 MonsterBook.img 節點
        匯入新的 MonsterBook_updated.img.xml (命名為 MonsterBook.img)

Step 6: 儲存客戶端 WZ

Step 7: 重新匯出為伺服器 XML
        使用 HaRepacker 的 "Private Server..." 功能
```

## 5) NPC 腳本編寫 ✅

### MapleSolaxiaV2 NPC 腳本位置

```
scripts/npc/
```

### NPC 腳本基本框架

```javascript
var status = 0;

function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && status == 0) {
            cm.dispose();
            return;
        }
        if (mode == 1) {
            status++;
        } else {
            status--;
        }
        
        if (status == 0) {
            cm.sendNext("你好，我是 NPC！");
        } else if (status == 1) {
            cm.sendYesNo("你想要什麼？");
        } else if (status == 2) {
            cm.warp(100000000);
            cm.dispose();
        }
    }
}
```

### 動態 NPC 商店腳本 (RaGEZONE, v83)

**來源**: https://forum.ragezone.com/threads/dynamic-npc-shop-script-v83.1089979/

**特色**：
- 不需要 SQL 資料庫更新
- 可直接修改腳本即時變更商品
- 適合快速測試

### NPC 腳本狀態機制

| status 值 | 階段 | 說明 |
|-----------|------|------|
| -1 | 結束 | 對話框關閉 |
| 0 | 第一階段 | 第一個訊息 (sendNext/sendOk) |
| 1 | 第二階段 | 按下一步後 |
| 2 | 第三階段 | 按下一步後 |
