## 🆕 v83 登入器 IP 修改 — STREDIT 詳細教學 (RaGEZONE)

**URL**: https://forum.ragezone.com/threads/release-stredit-localhost-editor.810762/

### STREDIT 操作流程
```
1. File → Open → 選擇 localhost.exe
2. 詢問「是否 v118+」→ 選「No」（v83 適用）
3. 詢問「IP 搜尋模式」→ 選「Yes」
4. 點擊「Force to a IP」按鈕
5. 輸入目標 IP（如 127.0.0.1 或 VPS IP）
6. 保存修改後的 localhost.exe
```

### 疑難排解
| 問題 | 解決方案 |
|------|---------|
| 找不到 IP | 使用 Hex Editor 直接搜尋 `31 37 32 2E 30 2E 30 2E 31` |
| 編輯後無法連線 | 確認版本選擇正確 |

## 🆕 v83 客戶端修改 — IDA + Hex Editor 組合

### IDA 靜態分析重點
```
1. 找 CClientSocket::ProcessPacket - 封包接收總入口
2. 找 CClientSocket::SendPacket - 封包發送總入口
3. 分析 MapleCrypto 加密邏輯
4. 提取 Opcode 表
```

### Hex Editor 動態修改
| 工具 | 用途 |
|------|------|
| HxD | 快速 Hex 編輯 |
| Neo Hex Editor | 字串替換（IP、伺服器位址） |
| Cheat Engine | 記憶體搜尋 |

### v83 常用 Hex 位址
| 修改項目 | Hex 值 |
|---------|--------|
| IP 位址 `127.0.0.1` | `31 37 32 2E 30 2E 30 2E 31` |
| Speed Cap | `250 = FA 00 00 00` |
| 4G Flag | `偏移 0x13E: 0x0F → 0x2F` |
