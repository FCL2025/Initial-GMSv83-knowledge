## 📌 登入器/客戶端修改

### 登入器原理 (Login Redirector)
登入器是一個 DLL 或外部程式，攔截客戶端的網路連線，將其導向私人伺服器：

```
原始流程：
  MapleStory.exe → connect("203.78.xx.xx:7575") → Nexon 伺服器

使用登入器：
  MapleStory.exe → Hook API (connect/WSAConnect)
                  → 攔截並修改目標 IP/Port
                  → connect("127.0.0.1:7575") → 私人伺服器
```

### 主流方案對比

| 方案 | 難度 | IP長度問題 | 額外功能 |
|------|------|-----------|---------|
| Ezorsia v2 DLL | 極簡 | 不需要修改 | HD解析度、CashShop修復 |
| HxD直接修改 | 簡單 | 需一致 | 無 |
| MapleClientEditTemplate | 中等 | 自動處理 | WinSock Hook |
| Themida脫殼 | 複雜 | 視情況 | 可深度修改 |

### Ezorsia v2 特點 (最推薦新手)
- dinput8.dll 放入遊戲目錄即可
- 無需修改任何 exe
- 自帶 IP 導向
- 支援 HD 解析度
- 自訂最多 3 個 DLL
- 2023年11月發布，社群活躍

### Cosmic 客戶端安裝流程 (官方)

```

```

### 封包加密三層架構 (v83)
```
Layer 1: MapleCrypto (頭部混淆) → 4-byte header，含 opcode + length
Layer 2: Shanda Encryption → 對稱加密，v83仍在用
Layer 3: AES-OFB → 256-bit key，版本決定的靜態金鑰
```

### Hello Packet (第一個封包，永遠明文)
```
偏移  長度   內容
0x00  2      版本號 (0x0053 = 83)
0x02  n      版本字串 ("83")
n+2   4      SendIV (客戶端指定)
n+6   4      RecvIV (客戶端指定)
```

### 登入流程封包序列 (10步)
```
1. C→S Hello Packet (plaintext)
2. S→C Hello Packet (plaintext)
3. C→S LOGIN_PASSWORD (encrypted) - Opcode 0x01
4. S→C LOGIN_STATUS (encrypted) - Opcode 0x00
5. C→S SERVERLIST_REQUEST (encrypted) - Opcode 0x06
6. S→C SERVERLIST (encrypted) - Opcode 0x0C
7. C→S CHARLIST_REQUEST (encrypted) - Opcode 0x07
8. S→C CHARLIST (encrypted) - Opcode 0x03
9. C→S CHAR_SELECT (encrypted) - Opcode 0x08
10. C→S PLAYER_DROP (encrypted) - Opcode 0x0B
```

## 📌 Opcode 結構

### MapleCrypto 頭部混淆演算法
```java
// 加密端
public byte[] getPacketHeader(int opcode, int length, int iv) {
    int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
    verify = (verify << 3) | (verify >>> 5);
    verify &= 0xFFFF;
    return new byte[] {
        (byte) (verify & 0xFF),
        (byte) ((verify >>> 8) & 0xFF),
        (byte) (length & 0xFF),
        (byte) ((length >>> 8) & 0xFF)
    };
}

// 解密端
public int retrieveOpcode(byte[] header, int iv) {
    int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
    verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
    return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
}
```

### IDA 中 Decode 函數對照
| IDA函數 | MaplePacketWriter方法 | 說明 |
|--------|---------------------|------|
| Decode1 | mplew.write | 1 byte |
| Decode2 | mplew.writeShort | 2 bytes |
| Decode4 | mplew.writeInt | 4 bytes |
| DecodeStr | mplew.writeMapleAsciiString | 字串 |
| DecodeBuffer(x,8) | mplew.writeLong | 8 bytes |
| DecodeBuffer(x,13) | mplew.writeAsciiString(str,13) | 固定長度字串 |

### CClientSocket::ProcessPacket 封包分發流程
```
CClientSocket::ProcessPacket()
    ↓ 根據 MapleCrypto header 解密
    ↓ 提取 opcode (2 bytes, little endian)
    ↓ switch(opcode) 分發
    → CLogin::OnPacket()    // 登入畫面
    → CField::OnPacket()     // 遊戲中
    → CShop::OnPacket()      // 商場
```

### 創建自訂 NPC WZ 流程
```
1. HaRepacker 導出 NPC.WZ 為 XML
2. 找到要複製的 NPC img（如 0001337.img）
3. 複製並命名為新的 NPC ID
4. 修改 frame 和 origin 設定
5. 重新導入 WZ
6. 伺服器端編寫對應的 .js 腳本
```

### Mob.wz 常用修改
| 修改項目 | 目標屬性 | 新值 |
|---------|---------|------|
| GodMode (Miss) | acc | 0 |
| HalfGodMode (Super Avoid) | acc | 1 |
| 怪物速度調整 | speed | 126 |

## 📌 NPC 腳本編寫

### 狀態機核心 (Status State Machine)
```javascript
status = -1 → 初始調用 start()
    ↓
status = 0 → 第一次 sendNext/sendSimple
    ↓
status = 1 → 玩家回應後
    ↓
... 依此類推
```

### Mode 參數含義
| Mode 值 | 意義 |
|---------|------|
| 1 | 點擊"下一頁"或選項 |
| 0 | 點擊"返回" |
| -1 | 關閉對話框 (ESC/X) |

### 對話框類型 (Type)
| Type | 方法 | 說明 |
|------|------|------|
| 0 | sendNext/sendOk/sendPrev | 標準對話 |
| 1 | sendYesNo | Yes/No選擇 |
| 2 | sendSimple | 選項選單 |
| 3 | sendGetNumber | 數字輸入 |
| 4 | 自動關閉 | - |
| 12 | sendAcceptDecline | 接受/拒絕 |

### 動態商店腳本優點
- 不需要 SQL 資料庫
- 每次修改無需重啟伺服器
- 在腳本中直接定義商品陣列

### 腳本源相容性問題
| 問題 | 解決方案 |
|------|---------|
| `importPackage is not defined` | JDK 7+ 廢棄，移除該行 |
| `SavedLocationType is not defined` | 使用相容格式 |
| NPC 腳本不相容 | 使用普遍 `cm.` 函數，避免專有Package |

### NPC對話格式標籤
| 標籤 | 功能 |
|------|------|
| `#b` | 藍色文字 |
| `#r` | 紅色文字 |
| `#k` | 黑色文字 |
| `#l` | 選項標記 |
| `#i[itemid]` | 物品圖片 |
| `#h#` | 玩家名稱 |
| `#c[itemid]#` | 持有物品數量 |
| `\r\n` | 換行 |
