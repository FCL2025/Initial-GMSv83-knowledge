### 2.2 MaplePE 完整技術原理（2025 新工具）

**GitHub**: https://github.com/zhyonc/MaplePE

**32-bit MapleStory 封包編輯器**，功能比 Maple_Pshark 更豐富：

**架構**：
```
MaplePE.exe（UI + UDP 伺服器）
    ↓ 接收封包資訊
Packet.dll（注入目標進程 + Hook 函數）
```

**需要 Hook 的函數位址（完整表）**：

| 接收端 (CInPacket) | 發送端 (COutPacket) | Socket |
|---------------------|----------------------|--------|
| Decode1 | Encode1 | ProcessPacket |
| Decode2 | Encode2 | SendPacket |
| Decode4 | Encode4 | |
| Decode8 | Encode8 | |
| DecodeStr | EncodeStr | |
| DecodeBuffer | EncodeBuffer | |
| | MakeBufferList | |

**使用流程**：
```
1. 填寫函數位址 → Save
2. 啟動遊戲客戶端
3. Patch → 定位 process
4. InjectDLL → 注入 Packet.dll
5. 分析/修改/發送封包
```

**注入失敗原因**：
- Permission denied → 系統管理員執行
- Missing Packet.dll → 安全軟體誤刪
- Injection protection → Launcher 記憶體保護

### 2.3 MaplePacketPuller SpiritIDAPlugin 深入解析

**GitHub**:
- MaplePacketPuller: https://github.com/Bratah123/MaplePacketPuller
- SpiritIDAPlugin: https://github.com/Bratah123/SpiritIDAPlugin

**支援環境**：
| 環境 | 支援 |
|------|------|
| Python 3.8.5 | ✅ |
| Python 3.6.12 / 3.8.6 | ✅ |
| IDA Pro 7.0 (32-bit) | ✅ |
| IDA Pro 7.5 | ✅ |
| IDA Pro 6.8 | ✅ |
| Python 2.7 | ❌ |

**工作流程**：
```
IDA Pro → 找到 CField::OnPacket / CLogin::OnPacket
    → 產生 C-style pseudocode
    → 複製到 Functions/ 目錄 .txt 檔案
    → cd MaplePacketPuller/IDA\ Maple\ Script/src/main/python
    → python main.py
    → 選擇分析類型 + 輸入檔案名稱
    → 輸出：封包結構（Decode1/2/4/Str）
```

**限制**：
- while() 迴圈處理不完善（建議設 GET_ALL_DECODES=true）
- 不顯示 switch cases（需用戶自行處理）
