### 1. 登入器/客戶端修改

#### v83 客戶端 IP 修改工具

| 工具 | 用途 | 難度 |
|------|------|------|
| **HxD** | Hex 編輯器，直接修改 localhost.exe 中的 127.0.0.1 | ⭐ |
| **STREDIT.EXE** | 字串編輯工具，專為 MapleStory 設計 | ⭐ |
| **OllyDbg + Cheat Engine** | 進階記憶體修改 | ⭐⭐⭐ |

**HxD 修改步驟**：
```
1. 用 HxD 打開 localhost.exe
2. Ctrl+F 搜尋 "127.0.0.1"
3. 替換為目標 IP（如 192.168.1.100）
4. 儲存檔案
```

**STREDIT.EXE**（推薦新手使用）：
```
1. 用 STREDIT.EXE 打開 localhost.exe
2. 直接編輯 IP 位址
3. 儲存即可
```

#### MapleCrypto 頭部混淆機制

```
原始 opcode: 0x0001 (如 LOGIN_PASSWORD)
       IV: [0x7A, 0x52, ...]

Step 1: XOR + SHIFT
  verify = (opcode ^ iv[0]) + (iv[1] << 8)

Step 2: 位旋轉 (rotate left 3 bits)
  verify = (verify << 3) | (verify >> 13)

Step 3: 取低位 16 bits
  verify = verify & 0xFFFF
```

### 4. WZ 修改

#### HaRepacker WZ 編輯黃金法則

> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

**正確流程**：
1. HaRepacker 4.2.4 開啟 **GMS (old)** 加密的客戶端 WZ
2. **使用 UI 直接修改數值**（禁止刪除/重新導入節點！）
3. 儲存 WZ
4. 選擇 **"Private Server..."** 導出
5. 將導出的 XML 覆蓋到伺服器的 WZ.XML 目錄

#### Quest 等級限制移除（LvMax Removal）Notepad++ Regex

```
尋找：lvmax="\d+"\s*
取代：（留空）
```

### 5. NPC 腳本編寫

#### 音源解析器對照

| 音源 | 解析器 | importPackage | ECMAScript 6 | 備註 |
|------|--------|-------------|-------------|------|
| **Cosmic** | Nashorn | ❌ 已移除 | ✅ | 最普遍相容 |
| **IntransigentMS** | Nashorn | ❌ | ✅ | 完全 ES6 |
| **Kinoko** | 混合 | ⚠️ 警告 | ⚠️ | 需注意 |
| **HeavenMS** | JDK 6 時代 | ✅ 存在 | ❌ | 需改寫 |

#### ✅ 相容寫法（所有音源通用）

```javascript
var MapleCharacter = Packages.client.MapleCharacter;
var MapleInventory = Packages.server.MapleInventory;
cm.getPlayer();
cm.gainItem(itemId, count);

// 或使用 Java.type (Nashorn ES6)
const MapleCharacter = Java.type("net.sf.odinms.client.MapleCharacter");
```

#### ❌ 不相容寫法（JDK 7+ 已廢棄）

```javascript
importPackage(Packages.client);
importPackage(Packages.server);
```
