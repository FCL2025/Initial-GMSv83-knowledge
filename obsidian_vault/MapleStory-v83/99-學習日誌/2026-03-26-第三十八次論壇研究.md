### 1. Maple_Pshark - v83 專用封包 Logger (GitHub, 2026)

**GitHub**: https://github.com/obstriker/Maple_Pshark

**專案概述**：
Mapleroyals 專用封包截獲器，專為 v83 MapleStory 私人伺服器設計。

**核心功能**：
| 功能 | 說明 |
|------|------|
| 1. Intercepting Recv messages | 攔截客戶端接收封包 |
| 2. Modify messages | 修改封包內容 |
| 3. Injecting packets for send or recv | 注入自訂封包 |
| 4. Rule based packet manipulation/filtering | 規則化封包操作過濾 |

**技術特點**：
- Hook 加密函數，直接截獲明文
- 不需要額外設定 proxy
- localhost 直接使用
- 自動解析 opcode
- 支援 Recv/Send 雙向攔截

**與 MaplePacketPuller 的差異**：
| 工具 | Maple_Pshark | MaplePacketPuller |
|------|-------------|------------------|
| 類型 | 即時截獲器 | 離線分析器 |
| 運作方式 | Hook 函數，直接明文 | 分析 .pcap 檔案 |
| 封包注入 | ✅ 支援 | ❌ 不支援 |
| IDA 整合 | ❌ | ✅ SpiritIDAPlugin |

### 2. Cosmic 伺服器架設完整流程 (2026-03 最新)

**GitHub**: https://github.com/P0nk/Cosmic

Cosmic 是目前最活躍的 v83 模擬器，2026 年 3 月持續更新。

**環境需求**：
| 軟體 | 版本 | 必要性 |
|------|------|--------|
| Java 17 SDK | JDK 17 | 必要 |
| MySQL Community Server 8 | MySQL 8 | 必要 |
| MySQL Workbench 8 | - | 可選 |
| Docker Desktop | - | 可選（快速啟動） |

**完整架設流程**：

```
Step 1: 複製專案
  git clone https://github.com/P0nk/Cosmic.git
  IntelliJ → New Project from Version Control

Step 2: 設定資料庫
  - MySQL Workbench 建立使用者 "cosmic_server" / "snailshell"
  - 執行 database/sql/ 目錄下的 SQL 腳本（按順序）
    1-db_base.sql
    2-db_areas.sql
    3-db_shopupdate.sql (可選)
    4-db_admin.sql (推薦新手)
  
Step 3: 設定 IP
  - 編輯 config.yaml
  - localhost = 127.0.0.1（本機遊玩）
  - 若用 Hamachi，設為 Hamachi IP
  - 若用實體 IP，直接填入

Step 4: 客戶端設定
  - 安裝 MapleGlobal-v83-setup.exe
  - 刪除 HShield/ 目錄、ASPLnchr.exe、MapleStory.exe、Patcher.exe

  - 複製 localhost.exe 到客戶端目錄
  - 若 IP 非 127.0.0.1，用 HxD 修改

Step 5: 啟動
  - IntelliJ: 開啟 src/main/java/net/server/Server.java
  - 點擊綠色箭頭 → Run Cosmic
  - 或使用 Docker: docker-compose up
```

### 3. MapleSecets WZ 編輯教程 (2014 經典)

**來源**: http://maplesecrets.blogspot.com/2014/05/tutorial-wz-editing-for-maplestory-d.html

這是經典的 WZ 編輯教程，涵蓋幾乎所有常見修改：

**工具下載**：
- deMSwZ15.rar - 批量修改工具
- HaRepacker 3.5.rar - WZ 編輯器
- SoundWz.rar - 音效相關

**WZ 檔案功能對照**：
| WZ 檔案 | 內容 |
|---------|------|
| Base.wz | 武器、技能、音效資訊 |
| Character.wz | 角色動作、裝備外觀 |
| Effect.wz | 傷害數字、墓碑、升級特效 |
| Etc.wz | 任務、buff、詛咒、角色創建、技能資訊 |
| Item.wz | 所有物品/裝備圖片和資料 |
| Map.wz | 所有地圖視覺效果 |
| Mob.wz | 所有怪物資料（速度、血量、技能等） |
| Npc.wz | 所有 NPC 資料 |
| Quest.wz | 所有任務資訊 |
| Reactor.wz | 可破壞物件（箱子、植物、門等） |
| Skill.wz | 所有職業技能 |
| Sound.wz | 所有音效和歌曲 |
| String.wz | 伺服器端資訊（無法直接編輯） |

**常見 WZ 編輯修改**：

| 修改類型 | 工具 | 目標 WZ | 操作 |
|---------|------|---------|------|
| 武器速度增加 | deMSwZ | Character.wz | 搜尋 "attackSpeed" 值設為 1 |
| 移除物品圖示 | HaRepacker | Item.wz | 刪除 iconRaw |
| 怪物 GodMode (miss) | deMSwZ | Mob.wz | acc 值設為 0 |
| 怪物 GodMode (full body) | deMSwZ | Mob.wz | bodyAttack 值設為 0 |
| 怪物加速 | HaRepacker | Mob.wz | speed 值設為 126 |
| 無敵 (Jesus) | deMSwZ | Mob.wz | MADamage, PADamage, acc 值設為 1 |
| 傳送範圍 | HaRepacker | Skill.wz | 相關技能屬性修改 |
| FlashStep | HaRepacker | Skill.wz | combatStep 設為 0 |
| 無限 MP (傳送) | HaRepacker | Skill.wz | mpCon 設為 0 |
| 傳送門無延迟 | HaRepacker | Reactor.wz | delay 屬性修改 |

**Mob.wz GodMode 變化**：

| GodMode 類型 | 修改屬性 | 效果 |
|-------------|---------|------|
| Full body | bodyAttack = 0 | 怪物完全不打你 |
| Jesus (Half God) | acc = 1 | 怪物很少打你 |
| Miss | acc = 0 | 怪物攻擊永遠miss |
| Super Avoid | acc = 1 | 怪物偶爾打你 |
| Pacifist | notAttack = 1 | 怪物不攻擊 |

**Monster Magic Disarm**：
```
Step 1 -> Open deMSwZ
Step 2 -> Open Mob.wz
Step 3 -> Untick all boxes except the last empty one
Step 4 -> Type "action" into the empty box
Step 5 -> Type "0" into the value box
Step 6 -> Process All
Step 7 -> Type "attackAfter" → value "65535"
Step 8 -> Process All
Step 9 -> Type "effectAfter" → value "65535"
Step 10 -> Process All then Save
```

```

```

### 6. elitepvpers NPC 腳本教程精華

**來源**: https://www.elitepvpers.com/forum/maplestory/597399-guide-npc-scripting-tutorial.html

**NPC 對話格式標籤完整對照**：

| 標籤 | 功能 | 範例 |
|------|------|------|
| `#b` | 藍色文字 | `#b藍色文字` |
| `#c[itemid]#` | 顯示玩家持有該物品的數量 | `#c1302000#` → "5" |
| `#d` | 紫色文字 | `#d紫色文字` |
| `#e` | 粗體文字 | `#e粗體` |
| `#f[imagelocation]` | 顯示 WZ 中的圖片 | `#fimg/item/1302000.img/0/0#` |
| `#g` | 綠色文字 | `#g綠色文字` |
| `#h #` | 顯示玩家名稱 | `#h#` → "快樂貓" |
| `#i[itemid]#` | 顯示物品圖片 | `#i1302000#` |
| `#k` | 黑色文字 | `#k黑色文字` |
| `#l` | 選項標記（用於 sendSimple） | `#L0#選項1#l` |
| `#n` | 正常字體（取消粗體） | `#n` |
| `#r` | 紅色文字 | `#r紅色文字` |
| `#s` | 黃色文字 | `#s黃色文字` |
| `\r\n` | 換行 | 多行對話 |

**使用範例**：
```javascript
cm.sendNext("你好 #h#！\r\n" +
    "#b#L0#我想購買武器#l\r\n" +
    "#r#L1#我想購買防具#l\r\n" +
    "#i1302000# #t1302000# x #c1302000#");
```

### 8. botnored v83 封包類別結構

**來源**: https://botnored.weebly.com/blog/maplestory-classes-v83

**核心原則**：
> **第一個發送/接收的封包永遠是純文字（未加密）**
> MapleStory 封包透過 TCP Socket 傳輸

**客戶端主要類別**：
| 類別 | 功能 |
|------|------|
| `CWvsContext` | 世界狀態管理（角色資料、伺服器列表） |
| `CClientSocket` | 網路連接管理（Send/Recv） |
| `CField` | 當前地圖/遊戲場景處理 |
| `CLogin` | 登入畫面處理 |

**封包流向**：
```
CClientSocket::ProcessPacket()
    ↓
根據 Header 分發
    ↓
CField::OnPacket() 或 CLogin::OnPacket()
    ↓
處理具體 Opcode
```
