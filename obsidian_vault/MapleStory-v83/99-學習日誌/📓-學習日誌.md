## 問題與解答

### Q: 為什麼客戶端修改後無法連接？
A: 檢查 IP 設定、端口（預設 7575）、防火牆，以及伺服器版本是否與客戶端版本匹配（v83）

### Q: Themida 脫殼失敗？
A: 嘗試不同的 OllyDbg 插件和設定，或使用 VMProtect 替代方案

### Q: WZ 修改後遊戲崩潰？
A: 確認版本號匹配，使用 HaRepacker-resurrected 而非舊版 HaRepacker，並始終從客戶端 WZ 匯出到伺服器 XML，而非反向操作

### Q: v83 AES 金鑰握手失敗？
A: 確認 MapleAESOFB.java 的 IV（SendIV/RecvIV）與客戶端一致，v83 使用靜態 AES 金鑰（256-bit），第一個封包（Hello Packet）為純文字未加密

### 1. NPC 腳本相容性問題 (RaGEZONE, 3週前)
- **問題**：不同音源（Cosmic、Kinoko、HeavenMS）的 NPC 腳本格式存在差異
- **原因**：腳本解析器版本不同，較舊的腳本在新解析器可能不相容
- **解決**：在 source 中 include 較舊版本的 script reader，或使用相容格式
- **來源**：https://forum.ragezone.com/threads/kinoko-npc-scripts.1261538/

### 2. 任務等級限制移除方法 (RaGEZONE, 2025-07)
- **方法**：使用 Harepacker 导出 Check.img.xml → 用 Notepad++ 編輯 lvmax 屬性 → 重新導入
- **步驟**：
  1. Tools → Export selection → XML → Classic（避免新版加密）
  2. 開啟 XML，搜尋所有 lvmax 屬性並移除
  3. 存檔後用 Harepacker 重新導入
- **來源**：https://forum.ragezone.com/threads/maplestory-v83-method-to-disable-quest-level-restrictions.1250107/

### 3. Boss HP 百分比血條實現 (RaGEZONE, 2026-02)

- **客戶端修改**：需修改 UI.wz 或使用 HeavenMS 源的 Mobgage 機制
- **參考服務器**：MapleRoyals 有完整實現
- **來源**：https://forum.ragezone.com/threads/v83-client-boss-percentage-bar.1260584/

### 4. LAN 遊戲連線錯誤 -2147467259 (RaGEZONE, 2週前)
- **現象**：PC1（主機）正常，PC2 連線失敗，錯誤碼 -2147467259
- **原因**：防火牆阻擋、端口未轉發、或客戶端 IP 修改不完整
- **解決**：
  1. 確認兩台 PC 在同一網段
  2. 開放 7575/8484 端口
  3. 檢查客戶端 IP 是否已改為主機實際 IP（而非 127.0.0.1）
  4. 嘗試關閉雙方防火牆測試
- **來源**：https://forum.ragezone.com/threads/need-help-v83-lan-server-works-on-pc1-but-pc2-gets-error-2147467259.1262187/
