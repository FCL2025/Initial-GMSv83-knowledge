### 2. MaplePE - 新興封包編輯工具

### 3. MapleStory IDC Script
- **GitHub Gist**: https://gist.github.com/diamondo25/be95345a2875ab4342cd
- 自動化 IDA 中的 Opcode 命名腳本
- 大幅加速 IDA 分析速度

### 4. v83 IDB + Client Edit Dump (RaGEZONE, 2021)
- **來源**: https://forum.ragezone.com/threads/v83-idb-client-edit-dump.1193418/
- 基於 localhost MapleStory 客戶端的 IDA Database
- 可直接載入 IDA 分析，節省大量時間
- IDA 7.0 建立

### 5. How to use IDA in v83 (RaGEZONE, 2016)
- **來源**: https://forum.ragezone.com/threads/how-to-use-ida-in-v83.1107216/
- Decode1 = mplew.write
- Decode2 = mplew.writeShort
- Decode4 = mplew.writeInt
- DecodeStr = mplew.writeMapleAsciiString
- DecodeBuffer(x, 8) = mplew.writeLong
- DecodeBuffer(x, 13) = mplew.writeAsciiString(str, 13)

### 6. v83 Tubi Addies (RaGEZONE, 2013)
- **來源**: https://forum.ragezone.com/threads/v83-tubi-addies.971905/
- Cheat Engine 地址參考
- 包含 v83 常用記憶體位址

### 7. How to find packet structures via IDA (RaGEZONE, 2019)
- **來源**: https://forum.ragezone.com/threads/how-does-one-find-a-structure-of-a-packet-via-ida.1162654/
- 關鍵：`CClientSocket::ProcessPacket` 是客戶端接收封包的總入口
- 從這裡根據 header 分發到 `CField::OnPacket` 或 `CLogin::OnPacket`

### 8. MapleStory v83 Private Server Hacks - MPGH
- **來源**: https://www.mpgh.net/forum/529-maplestory-private-server-hacks/181949-maplestory-v83-private-server-hacks.html
- 包含 Blitz's Cheat Table v83 Rev 3.00 (.CT)
- Kitterz Packet Editor (.DLL)

### 9. v83 Dupe (PDFCOFFEE)
- **來源**: https://pdfcoffee.com/v83-dupe-pdf-free.html
- Opcode 0x6F (商場買賣封包) 結構分析
- `6F 00 20 00 FF FF FF 80 00` = 購買物品寫入最大 Mesos

### 10. Getting packet structures/opcodes using IDA (RaGEZONE, 2011)
- **來源**: https://forum.ragezone.com/threads/getting-packet-structures-opcodes-using-ida.792436/
- 經典 IDA 分析教學
- 需 IDA Pro + 已脫殼的二進位檔

### 11. Community WZ/IMG Repack for v83 (RaGEZONE, 2023)
- **來源**: https://forum.ragezone.com/threads/community-wz-img-repack-for-v83.1221835/
- 使用 HaRepacker 導出 XML（Private Server 格式）
- 導出位置：Base.wz → NPC.wz 等

### 12. NPC 腳本源間相容性 (RaGEZONE, 2026-03)
- **來源**: https://forum.ragezone.com/threads/kinoko-npc-scripts.1261538/
- 不同音源（Cosmic、Kinoko、HeavenMS）的 NPC 腳本格式存在差異
- 建議使用普遍支援的 `cm.` 函數
