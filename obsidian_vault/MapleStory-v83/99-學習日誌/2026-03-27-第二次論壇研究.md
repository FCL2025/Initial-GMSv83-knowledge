```

2. Ctrl+F，選擇「文字字串」，搜尋 "127.0.0.1"
```

### 🆕 CInPacket::Decode 封包解讀流程

**完整流向**：
```
1. CClientSocket::OnRead()
   - 從 socket 讀取資料
   - 調用 AppendBuffer 追加到緩衝區

2. CClientSocket::ProcessPacket()
   - 讀取 Header (4 bytes)
   - 解密 Header 取得長度和 Opcode
   - 根據 Opcode 分發到對應處理函數

3. 各 Opcode 處理函數
   - CInPacket::Decode1/2/4/8/Buffer/Str
   - 依序解讀封包內容
```

### 🆕 IDAPython Opcode 分析教學 (RaGEZONE 2015)

**技巧**: 使用 IDAPython 追蹤 `COutPacket::Encode` 呼叫

```
IDAPython 技巧 - 相對位址計算：
E8 C5 90 D8 FE (Call near)

1. 取後4位元組: C5 90 D8 FE
2. 反轉: FED890C5
3. 轉2's complement: ~0xFED890C5 + 1 = 0x01276F3B
4. 當前地址 - 結果 = 目標地址
```

**v95 IDB 分析流程**：
1. 開啟 IDA，載入 MapleStory.exe
2. 搜尋 `COutPacket::Encode`
3. 找到 `INVENTORY_OPERATION` 函數
4. 追蹤所有 `Encode*` 呼叫
5. 重建封包結構

```javascript
var status = 0;
var shopItems = [
    { id: 2000002, price: 1000, quantity: 100 },  // HP 水
    { id: 2000006, price: 2000, quantity: 50 },  // MP 水
    { id: 4001000, price: 5000, quantity: 1 }     // 盾牌

function start() {
    var msg = "歡迎光臨我的商店！\n\n";
    for (var i = 0; i < shopItems.length; i++) {
        msg += "#b#L" + i + "#" + shopItems[i].id + 
               " - " + shopItems[i].price + " 楓幣#l\n";
    }
    cm.sendSimple(msg);
}

function action(mode, type, selection) {
    if (mode == 1) {
        var item = shopItems[selection];
        if (cm.getMesos() >= item.price) {
            cm.gainMesos(-item.price);
            cm.gainItem(item.id, item.quantity);
            cm.sendNext("交易完成！");
        } else {
            cm.sendNext("楓幣不足！");
        }
    }
    cm.dispose();
}
```

### 🆕 轉職 NPC 完整腳本範例

```javascript
var status = 0;

function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1 || mode == 0) {
        cm.dispose();
        return;
    }
    
    if (mode == 1) {
        status++;
    } else {
        cm.dispose();
        return;
    }
    
    var player = cm.getPlayer();
    var job = player.getJob();
    
    if (status == 0) {
        if (job == 0) {
            cm.sendSimple("你是新手。選擇要轉職的方向:\n" +
                "#b#L0#戰士 (劍客)#l\n" +
                "#L1#法師 (火毒)#l\n" +
                "#L2#弓箭手 (獵人)#l\n" +
                "#L3#盜賊 (刺客)#l");
        } else {
            cm.sendNext("你已經轉職過了！目前職業: " + job);
            cm.dispose();
        }
    } else if (status == 1) {
        var newJob = [110, 210, 310, 410][selection];
        cm.changeJob(newJob);
        cm.sendNext("轉職成功！你现在是 " + getJobName(newJob) + "！");
        cm.dispose();
    }
}

function getJobName(jobId) {
    var names = {
        110: "劍客", 120: "槍騎兵", 130: "仲裁者",
        210: "火毒", 220: "冰雷", 230: "僧侶",
        310: "獵人", 320: "弓箭手", 330: "弩弓手",
        410: "刺客", 420: "俠盜", 430: "夜行者"
    };
    return names[jobId] || "未知";
}
```
