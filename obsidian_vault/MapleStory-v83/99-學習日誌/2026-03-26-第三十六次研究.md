## 研究重點

### 1. MapleLib — MapleStory 文件與協定工具包

**GitHub**: https://github.com/lastbattle/MapleLib

### 特色功能

- **.wz 檔案完整支援**：讀取、編輯、寫入所有版本 .wz 格式
- **.ms 檔案支援**：v220+ 的加密 .ms 封裝格式，可轉換為標準 .wz
- **封包讀寫**：完整的客戶端↔伺服器協定處理
- **熱更新機制**：支援 Data.wz 和原始 .img 熱更新檔
- **.NET 8.0**：現代化 .NET 實現

### 核心命名空間

```csharp
using MapleLib.WzLib;

// 初始化 WzFileManager
string baseDirectory = @"C:\Nexon\MapleStory";
bool isStandalone = false;
WzFileManager manager = new WzFileManager(baseDirectory, isStandalone);
manager.BuildWzFileList();

// 載入 WZ 檔案
WzFile wzFile = new WzFile(filePath, WzMapleVersion.BMS);
wzFile.ParseWzFile();

// 遍歷目錄和圖片
foreach (WzImage image in wzFile.WzDirectory.WzImages)
{
    string npcId = image.Name.Replace(".img", "");
    image.ParseImage();
}
```

### .ms 檔案處理（v220+）

```csharp
using MapleLib.WzLib.MSFile;

string filePath = "Map.ms";
using (FileStream fileStream = File.OpenRead(filePath))
{
    MemoryStream memoryStream = new MemoryStream();
    fileStream.CopyTo(memoryStream);
    memoryStream.Position = 0;

    WzMsFile msFile = new WzMsFile(memoryStream, fileName, filePath, true);
    msFile.ReadEntries();
    WzFile wzFile = msFile.LoadAsWzFile();
}
```

### WZ 版本枚舉

```csharp
WzMapleVersion.BMS    // 韓服
WzMapleVersion.GMS   // 全球服
WzMapleVersion.EMS   // 歐服
WzMapleVersion.MSEA  // 東南亞服
WzMapleVersion.CMS   // 國服
WzMapleVersion.GENERATE // 需要暴力破解金鑰
```

### 2. Maple_Pshark — v83 封包攔截工具

**GitHub**: https://github.com/obstriker/Maple_Pshark

### 功能

- 專為 **Mapleroyals v83** 私服設計的封包記錄器
- **Hook 加密函數**，在封包加/解密前攔截
- 支援 **Recv/Send** 封包攔截與修改
- 支援**封包注入**（inject）
- **規則化封包操作/過濾**

### 使用場景

```
1. 攔截收到的 Recv 封包（伺服器→客戶端）
2. 修改封包內容後轉發
3. 向任意方向注入自定義封包
4. 基於規則的封包過濾和操作
```

### 3. MaplePacketPuller — IDA 封包結構分析工具

**GitHub**: https://github.com/Bratah123/MaplePacketPuller

### 輸入/輸出流程

```
IDA Pro 反編譯
    ↓ 複製 C 偽代碼
    ↓ 貼入 .txt 檔案
MaplePacketPuller 分析
    ↓
輸出 packet structure + opcode list
```

### 支援版本

| 環境 | 版本 |
|------|------|
| Python | 3.8.5, 3.6.12 |
| IDA Pro | 7.0 (32-bit) |
| 編輯器 | Atom, PyCharm |

### 局限性

- `while()` 迴圈處理不完全
- 不顯示 switch case（需用戶自行添加）
- 需要在 IDA 中對目標函數命名

### 搭配工具：SpiritIDAPlugin

https://github.com/Bratah123/SpiritIDAPlugin

- 讀取 IDA 生成的 C 偽代碼
- 格式化為伺服器 encode 友好的形式
- 可選精確度模式
