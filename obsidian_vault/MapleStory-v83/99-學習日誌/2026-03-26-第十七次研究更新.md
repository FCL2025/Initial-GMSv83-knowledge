### 1. Maple_Pshark v83 專用封包 Logger
- GitHub: https://github.com/obstriker/Maple_Pshark
- Hook 加密函數，直接截獲明文
- 支援封包修改 + 注入 + 規則過濾
- 功能：intercept Recv、modify messages、inject packets

### 2. MaplePacketPuller + SpiritIDAPlugin IDA 工作流
- SpiritIDAPlugin: IDA Pro 插件，格式化 pseudocode
- 完整工作流：IDA → .txt → Python 分析 → Decode 函數
- 支援搜尋 InHeader opcodes

### 3. RustMS 2026-03 重啟
- URL: https://github.com/neeerp/RustMS
- 僅 Login Server（早期階段）
- 非 JVM，記憶體效率極佳
- 技術架構：net::io + net::packet + crypt + db

### 4. LAN 多人連線錯誤 0x80004005 修復
- 原因：防火牆/權限問題
- 解決：netsh advfirewall set allprofiles state off
- 確認 listen 0.0.0.0（而非 127.0.0.1）

### 5. CInPacket Decode IDA 完整對照表
- Decode1 → write(1) | 1 byte
- Decode2 → writeShort() | 2 bytes
- Decode4 → writeInt() | 4 bytes
- DecodeStr → writeMapleAsciiString() | 可變

### 6. v83 登入結果碼
- 0: 登入成功
- 5: 帳號或密碼錯誤
- 7: 帳號已停用
- 13: 刪除中（冷靜期）
- 23: 登入太頻繁

### 7. NPC TYPE 值
- 0: 普通 NPC，需 .js
- 1: 商店 NPC，需 .js
- 2~6: 拍賣等特殊 NPC
