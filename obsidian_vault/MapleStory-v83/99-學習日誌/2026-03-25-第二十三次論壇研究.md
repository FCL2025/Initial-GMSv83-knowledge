## 🆕 新發现：v83 Opcode 結構更新

### MapleRef Wiki - Opcode 結構總結
來源: https://mapleref.fandom.com/wiki/Operation_Code

**核心概念**：
> "Transmitting from the server and receiving on the client has one set of agreed upon Opcodes, where transmitting from the client and receiving on the server has yet another."

| 方向 | 說明 |
|------|------|
| S→C (Send) | 伺服器發送到客戶端 |
| C→S (Recv) | 客戶端發送到伺服器 |

**重要**：同一 Opcode 數字可能代表不同含義，視方向而定！

### v83 Opcode 特性總結

| 特性 | 說明 |
|------|------|
| 已高度文檔化 | v83 Opcode 幾乎全部被映射 |
| AES 加密 | 需配合客戶端金鑰 |
| 高度客製化 | 每個私服可能使用不同設定 |
| 幾乎不需要追蹤 | 大多數 opcode 表已在開源專案中完整提供 |

### botnored v83 封包類別結構

來源: https://botnored.weebly.com/blog/maplestory-classes-v83

| 類別 | 功能 |
|------|------|
| `CWvsContext` | 世界狀態管理（角色資料、伺服器列表） |
| `CClientSocket` | 網路連接管理（Send/Recv） |
| `CField` | 當前地圖/遊戲場景處理 |
| `CLogin` | 登入畫面處理 |

### CClientSocket::ProcessPacket 封包流向
```
CClientSocket::ProcessPacket()
    ↓
根據 Header 分發
    ↓
CField::OnPacket() 或 CLogin::OnPacket()
    ↓
處理具體 Opcode
```

### Community WZ/IMG Repack for v83
RaGEZONE: https://forum.ragezone.com/threads/community-wz-img-repack-for-v83.1221835/

**HaRepacker 導出 WZ 為 XML 步驟**：
```
1. 用 HaRepacker 打開 WZ 檔案
2. 選擇要導出的節點
3. 點擊 Export → 選擇 XML (Private Server) 格式
4. 導出後可使用文字編輯器修改
5. 重新導入 WZ
```

**注意**：
- v83 WZ 加密格式需選擇 `GMS (old)`
- 不要直接刪除/重新導入節點，可能造成數據丟失
- 建議使用 HaRepacker UI 直接編輯數值

### WZ 編輯黃金法則
> DO NOT USE the server's XMLs for reimporting into the client's WZ, it WILL generate bugs.

## 🆕 新發现：v83 NPC 腳本編寫

### 核心腳本函數速查

```javascript
// 對話發送
cm.sendNext("文字")           // 顯示「下一步」按鈕
cm.sendSimple("文字\r\n#b#L0#選項0#l\r\n#L1#選項1#l")  // 選擇選單
cm.sendYesNo("文字")          // 顯示「是/否」按鈕
cm.sendOk("文字")             // 顯示「確定」按鈕

// 玩家狀態查詢
cm.getLevel()                 // 取得玩家等級
cm.getJob()                   // 取得職業
cm.getMesos()                 // 取得楓幣
cm.haveItem(itemId, count)    // 是否持有物品

// 玩家操作
cm.gainItem(itemId, count)    // 給予物品
cm.gainMesos(amount)          // 給予/收取楓幣
cm.openShop(shopId)           // 開啟商店
cm.warp(mapId)                // 傳送地圖
cm.dispose()                  // 結束腳本
```

### NPC對話格式標籤完整對照

| 標籤 | 功能 | 範例 |
|------|------|------|
| `#b` | 藍色文字 | `#b藍色文字` |
| `#c[itemid]#` | 顯示玩家持有該物品的數量 | `#c1302000#` → "5" |
| `#e` | 粗體文字 | `#e粗體` |
| `#h #` | 顯示玩家名稱 | `#h#` → "快樂貓" |
| `#i[itemid]#` | 顯示物品圖片 | `#i1302000#` |
| `#l` | 選項標記（用於 sendSimple） | `#L0#選項1#l` |
| `#r` | 紅色文字 | `#r紅色文字` |
| `\r\n` | 換行 | 多行對話 |

### 狀態機 Mode 行為表

| Mode 值 | 意義 |
|---------|------|
| `mode == 1` | 玩家點擊了"下一頁"或選項 |
| `mode == 0` | 玩家點擊了"返回" (back button) |
| `mode == -1` | 玩家關閉了對話框 (ESC 或 X) |
