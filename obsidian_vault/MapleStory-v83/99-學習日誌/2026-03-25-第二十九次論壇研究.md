### 🆕 MapleStory 封包加密三層架構

```
┌─────────────────────────────────────────────────────┐
│ Layer 1: MapleCrypto (頭部混淆)                     │
│   - 4 bytes header，含混淆 opcode + length           │
│   - 防止靜態分析 opcode                             │
│                                                        │
│ Layer 2: Shanda Encryption (對稱加密)                │
│   - v83 仍使用，已於 v118+ 移除                     │
│                                                        │
│ Layer 3: AES-OFB (主加密)                           │
│   - 256-bit key, 4-byte IV                          │
│   - 版本決定的靜態金鑰                              │
└─────────────────────────────────────────────────────┘
```

**第一個封包原則**：
> **第一個發送/接收的封包永遠是純文字（未加密）**

### 🆕 MapleRef Wiki Send/Recv Opcode 核心概念 (2026)

**來源**: https://mapleref.fandom.com/wiki/Operation_Code

**重要原則**：
> "Transmitting from the server and receiving on the client has one set of agreed upon Opcodes, where transmitting from the client and receiving on the server has yet another."

| 方向 | 說明 | Opcode 集合 |
|------|------|------------|
| **S→C (Send)** | 伺服器發送到客戶端 | Server Opcode Set |
| **C→S (Recv)** | 客戶端發送到伺服器 | Client Opcode Set |

**重要**：同一個 Opcode 數字可能代表不同含義，視方向而定！

### 🆕 MapleCrypto Header 加密/解密公式 (v83 核心)

**加密端**：
```java
public byte[] getPacketHeader(int opcode, int length, int iv) {
    int verify = (opcode ^ (iv & 0xFF)) + ((iv >> 8) & 0xFF) * 0x100;
    verify = (verify << 3) | (verify >>> 5);
    verify &= 0xFFFF;
    return new byte[] {
        (byte) (verify & 0xFF),
        (byte) ((verify >>> 8) & 0xFF),
        (byte) (length & 0xFF),
        (byte) ((length >>> 8) & 0xFF)
    };
}
```

**解密端（反向）**：
```java
public int retrieveOpcode(byte[] header, int iv) {
    int verify = (header[0] & 0xFF) | ((header[1] << 8) & 0xFF00);
    verify = (verify >>> 3) | ((verify << 5) & 0xFFFF);
    return (verify - ((iv >> 8) & 0xFF) * 0x100) ^ (iv & 0xFF);
}
```

### 🆕 動態 NPC 商店腳本源 (v83, RaGEZONE 2016)
```javascript
var status = 0;
var items = Array(
    Array(1302000, 10000, 1),
    Array(1302001, 20000, 1),
    Array(1302002, 30000, 1)
);

function start() {
    status = -1;
    action(1, 0, 0);

        cm.dispose();
    } else {
        if (mode == 0 && type == 1) {
            cm.dispose();
            return;
        }
        if (mode == 1)
            status++;
        else
            status--;
        
        if (status == 0) {
            var msg = "歡迎光臨！\r\n請選擇商品：\r\n";
            for (var i = 0; i < items.length; i++) {
                msg += "#L" + i + "##i" + items[i][0] + "# - " + items[i][1] + " 金幣#l\r\n";
            }
            cm.sendSimple(msg);
        } else if (status == 1) {
            var item = items[selection];

                if (cm.canHold(item[0])) {
                    cm.gainMeso(-item[1]);
                    cm.gainItem(item[0], item[2]);
                    cm.sendOk("購買成功！");
                } else {
                    cm.sendOk("背包空間不足！");
```
