## 📌 新發現：Maple83 (jonnylin13/Maple83) 完整資源包

**GitHub**: https://github.com/jonnylin13/Maple83

### 與 MapleSolaxiaV2 的關係
Maple83 是 Ronan 維護的早期 v83 模擬器分支，基於 MapleSolaxia 改造而來。

### 完整架設流程（原文精華摘要）

#### 環境架設步驟
```
1. 安裝 WampServer2.0i.exe (MySQL)
2. 安裝 JDK 7 或 8
3. 安裝 NetBeans IDE (或 Eclipse)
4. 安裝 mysql-query-browser-1.1.20-win.msi
5. 安裝 Hamachi（可選，公開伺服器用）
```

#### JCE Unlimited Strength Policy 安裝（關鍵步驟）
必須覆蓋以下位置的 JCE policy 檔案：
```
C:\Program Files\Java\jre7\lib\security\
C:\Program Files\Java\jre7\lib\security\local_policy.jar
C:\Program Files\Java\jre7\lib\security\US_export_policy.jar

C:\Program Files\Java\jdk1.7.0_01\jre\lib\security\
C:\Program Files\Java\jdk1.7.0_01\jre\lib\security\local_policy.jar
C:\Program Files\Java\jdk1.7.0_01\jre\lib\security\US_export_policy.jar
```

#### 資料庫設定
```
MySQL Query Browser → 開啟以下 SQL 檔案：
- sql/db_database.sql
- sql/db_drops.sql
- (可選) sql/db_shopupdate.sql

最後手動在 accounts 資料表建立第一個帳號！
```

#### 客戶端完整架設流程
```
1. 用 ManagerMsv83.exe 安裝 MapleStory
2. 刪除：HShield folder, ASPLnchr.exe, MapleStory.exe, Patcher.exe
3. 用 localhost.exe 替換原本的執行檔
4. 用提供的 client_wz 覆蓋原廠 WZ
5. Hex Editor 修改 localhost.exe 中的 IP（若非 127.0.0.1）
6. 執行 localhost.exe
```

### IP 修改的 Hex Editor 步驟（原文）
```
1. 安裝 Neo Hex Editor
2. 開啟 localhost.exe
3. 搜尋 "Text String" 中的 "127.0.0.1"
4. 找到 3 個 IP 位址，全部替換為目標 IP
5. 保存並關閉
```

### README_wzchanges.txt 重點
> "Refer to 'README_wzchanges.txt' for more information on what have been changed from Nexon's v83 WZ files."

**重要**：不能直接使用原廠 Nexon WZ，必須使用專案提供的自訂 WZ 和 server-side wz.xml。

## 📌 MapleRef Wiki — Packets 結構核心發現

**URL**: https://mapleref.fandom.com/wiki/Packets

### 封包基本結構
> "Actions that are defined by an identifier called an Operation Code (opcode for short), which is the first 2 bytes of the packet (a short)"

### Opcode 歷史
- 早期版本：Opcode 是第一個 **1 byte**
- v83（及之後）：Opcode 是 **2 bytes (short)**
- 這個改變將最大Opcode 數量從 255 擴展到 65,536

### Packet Header 格式
> "Packet headers are 4 bytes long, and are generated with the use of the send IV, packet length (without header) and the version of MapleStory."

## 📌 MaplePE — 32-bit MapleStory 封包編輯器

**GitHub**: https://github.com/zhyonc/MaplePE

### 技術原理
```
MaplePE.exe (UDP server)
    ↓ 接收封包資訊
Packet.dll (injected via CreateRemoteThread + LoadLibraryW)
    ↓ Hook SendPacket/RecvPacket
    ↓ 發送封包資訊到 MaplePE.exe
```

### 封包發送前提條件
> "Before sending a packet via MaplePE.exe, the server must respond with at least one packet to ensure that the CClientSocket instance can be correctly specified."
