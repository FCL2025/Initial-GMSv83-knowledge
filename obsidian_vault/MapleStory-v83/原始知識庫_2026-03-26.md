## 四十八、MapleLib — MapleStory 文件與協定工具包

### 48.1 概述

**GitHub**: https://github.com/lastbattle/MapleLib
**開發者**: lastbattle (HaRepacker 作者)
**語言**: C# / .NET 8.0
**支援版本**: v1 ~ v260++（含 .ms 加密格式）

MapleLib 是目前最完整的 MapleStory 文件處理庫，支援 WZ 讀寫、封包協定、ms 加密層。

### 48.2 核心功能

| 功能 | 說明 |
|------|------|
| WZ 檔案讀寫 | 支援所有版本 .wz 格式 |
| .ms 檔案處理 | v220+ 的加密封裝格式 |
| 封包讀寫 | 客戶端↔伺服器協定處理 |
| 熱更新支援 | Data.wz / 原始 .img 熱更新 |
| List.wz 處理 | Big Bang 前版本列表檔 |
| 金鑰暴力破解 | GENERATE 版本專用 |

### 48.3 WZ 檔案處理

```csharp
using MapleLib.WzLib;

// 初始化管理器
string baseDirectory = @"C:\Nexon\MapleStory";
WzFileManager manager = new WzFileManager(baseDirectory, false);
manager.BuildWzFileList();

// 載入 WZ 檔案
WzFile wzFile = manager.LoadWzFile("NPC.wz", WzMapleVersion.BMS);

// 遍歷目錄
foreach (WzDirectory dir in wzFile.WzDirectory.WzDirectories)
{
    foreach (WzImage img in dir.WzImages)
    {
        string id = img.Name.Replace(".img", "");
    }
}
```

### 48.4 .ms 檔案處理（v220+）

```csharp
using MapleLib.WzLib.MSFile;

string filePath = "Map.ms";
using (FileStream fs = File.OpenRead(filePath))
{
    MemoryStream ms = new MemoryStream();
    fs.CopyTo(ms);
    ms.Position = 0;

    WzMsFile msFile = new WzMsFile(ms, Path.GetFileName(filePath), filePath, true);
    msFile.ReadEntries();
    WzFile wzFile = msFile.LoadAsWzFile();
}
```

### 48.5 WZ 版本枚舉

## 四十九、Maple_Pshark — v83 封包攔截工具

### 49.1 概述

**GitHub**: https://github.com/obstriker/Maple_Pshark
**目標版本**: MapleStory v83 (Mapleroyals 私服環境)

### 49.2 核心功能

1. **攔截 Recv 封包**：Hook 加密函數，在解密前獲取原始封包
2. **修改封包**：在攔截後即時修改封包內容
3. **封包注入**：向任意方向（Recv/Send）注入自定義封包
4. **規則化操作**：基於規則的封包過濾和操作

### 49.3 架構原理

```
【正常流程】
客戶端 ↔ 加密函數 ↔ 網路

【Maple_Pshark 流程】
客戶端 ↔ Hook 層（攔截/修改）↔ 加密函數 ↔ 網路
```

### 49.4 使用場景

- 分析伺服器→客戶端封包結構
- 測試自定義封包效果
- 開發封包編輯工具的參考實現
- 除錯私服協定處理

## 五十、MaplePacketPuller — IDA 封包結構分析

### 50.1 概述

**GitHub**: https://github.com/Bratah123/MaplePacketPuller
**Python**: 3.6+（推薦 3.8+）
**搭配套件**: [SpiritIDAPlugin](https://github.com/Bratah123/SpiritIDAPlugin)

### 50.2 工作流程

```
IDA Pro 反編譯客戶端
    ↓ 複製偽代碼到 .txt
    ↓ 放入 MaplePacketPuller/IDA Maple Script/Functions/
MaplePacketPuller 分析
    ↓
packet structures + opcodes → FuncOutput/
```

### 50.3 支援環境

| 環境 | 測試版本 |
|------|---------|
| Python | 3.8.5, 3.6.12 |
| IDA Pro | 7.0 (32-bit), 6.8, 7.5 |
| CLI | cmd, pwsh 7.0 |
| IDE | PyCharm Community 2020.1.1+ |

### 50.4 局限性

- `while()` 迴圈不完全處理（可用 `GET_ALL_DECODES=true` 改善）
- 不顯示 switch case（需用戶自行添加）
- 需要在 IDA 中對目標函數命名
- Python 2.7 不相容（使用 `os.scandir()`）

### 50.5 SpiritIDAPlugin 整合

- 讀取 IDA 生成的 C 偽代碼
- 格式化為伺服器 encode 友好格式
- 可選分析精確度模式
