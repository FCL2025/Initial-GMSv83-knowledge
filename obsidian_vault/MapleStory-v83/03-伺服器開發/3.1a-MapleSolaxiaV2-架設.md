## JCE Unlimited Policy 安裝

MapleSolaxiaV2 需要 AES-256 加密支援，必須替換 JCE policy 檔案。

### JCE Policy 替換位置

```
C:\Program Files\Java\jre7\lib\
C:\Program Files\Java\jre7\lib\ext\
C:\Program Files\Java\jre7\lib\security\
C:\Program Files\Java\jdk1.7.0_01\lib\
C:\Program Files\Java\jdk1.7.0_01\jre\lib\
C:\Program Files\Java\jdk1.7.0_01\jre\lib\ext\
C:\Program Files\Java\jdk1.7.0_01\jre\lib\security\
```

### 替換步驟

1. 下載 `jce_policy-7_unlimited.zip`
2. 解壓縮覆蓋上述目錄中的 `local_policy.jar` 和 `US_export_policy.jar`
3. 重啟 JDK/IDE

### 執行順序

```bash
# 1. 建立資料庫結構
File > Open Script > db_database.sql → 執行

# 2. 新增 NPC 商店物品（可選）
File > Open Script > db_drops.sql → 執行

# 3. 自訂商店更新（可選）
File > Open Script > db_shopupdate.sql → 執行
```

### 資料庫內容

| 腳本 | 用途 |
|------|------|
| `db_database.sql` | 建立 maplesolaxia schema |
| `db_drops.sql` | 物品掉落數據 |

## configuration.ini IP 配置

```ini
[server]
# localhost = 127.0.0.1（僅本機遊玩）
# 改為伺服器 IP 或 Hamachi IP（供外部連線）
ip=127.0.0.1
```

| 設定值 | 適用場景 |
|--------|---------|
| `localhost` / `127.0.0.1` | 單機測試 |
| Hamachi IP | 透過 Hamachi 連線 |
| Port Forwarding IP | 透過路由器連線 |

## 客戶端架設

### 完整步驟

```
1. 使用 ManagerMsv83.exe 安裝 MapleStory 到 C:\Nexon\MapleStory
2. 刪除保護相關檔案：
   - HShield/ 資料夾（整個刪除）
   - ASPLauncher.exe
   - MapleStory.exe
   - patcher.exe
3. 解壓 localhost.exe 到客戶端資料夾
4. 用修改過的 WZ 檔案覆蓋（client_wz 資料夾）
5. 如需自訂 IP：用 Neo Hex Editor 修改 localhost.exe
```

### IP 修改（HxD）

```
1. 用 HxD 開啟 localhost.exe
2. Ctrl+F 搜尋 "127.0.0.1"
3. 會找到三個連續的 IP（三個位置都要改）
4. 替換為目標 IP
5. 儲存
```

## 常見問題

### Q: 編譯失敗
A: 確認使用 JDK 7，JDK 8+ 可能不相容

### Q: 登入失敗
A: 檢查 JCE Unlimited Policy 是否正確安裝

### Q: 客戶端連線不上
A: 確認 configuration.ini IP 與客戶端一致
