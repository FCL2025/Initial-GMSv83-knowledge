## 🔨 編譯步驟

### Step 1：準備工作

1. 安裝 JDK 8
2. 安裝 NetBeans IDE 8.2（推薦）
3. 下載音源壓縮檔

### Step 2：匯入專案

1. 開啟 NetBeans
2. File → Import Project → Choose File
3. 選擇音源資料夾

### Step 3：設定資料庫

1. 開啟 MySQL 管理工具（如 Navicat）
2. 建立資料庫（如 `moopledev`）
3. 匯入 SQL 檔案

### Step 4：編譯

1. 在 NetBeans 中點擊 **Clean and Build**
2. 等待編譯完成
3. 找到生成的 `.jar` 檔案

---

## ⚠️ 常見錯誤

| 錯誤 | 解決方案 |
|------|----------|
| `JAVA_HOME not set` | 設定環境變數 |
| `MySQL connection failed` | 檢查帳密和服務狀態 |
| 編譯失敗 | 檢查 JDK 版本 |
