# 3.3 資料庫設定

---

## 🗄️ 資料庫結構

MapleStory 伺服器使用 MySQL 儲存：
- 玩家帳號
- 角色資料
- 世界設定
- 遊戲物品

---

## 📝 基本操作

### 連接到 MySQL

```bash
mysql -u root -p
```

### 建立資料庫

```sql
CREATE DATABASE moopledev CHARACTER SET utf8mb4 COLLATE utf8mb4_bin;
USE moopledev;
```

### 匯入 SQL

```bash
# 使用命令列
mysql -u root -p moopledev < MoopleDev.sql

# 或使用 GUI 工具
# Navicat → 右鍵 → 執行 SQL 檔案
```

---

## 📊 主要資料表

| 資料表 | 用途 |
|--------|------|
| `accounts` | 玩家帳號 |
| `characters` | 角色資料 |
| `inventoryitems` | 背包物品 |
| `quests` | 任務資料 |

---

## ⚠️ 注意事項

- 務必使用 **utf8mb4** 編碼
- 備份資料庫！
- 定期清理日誌
