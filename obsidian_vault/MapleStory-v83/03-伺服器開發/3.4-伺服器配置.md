## 🌐 IP 設定

### 客戶端 IP
客戶端的 IP 必須指向你的伺服器

### 伺服器 IP
確保以下 IP 正確：
- 外部 IP（供客戶端連接）
- 內部 IP（127.0.0.1 for localhost）

### 🆕 v83 LAN 多人遊戲 - 錯誤排查 (-2147467259)

**來源**: https://forum.ragezone.com/threads/need-help-v83-lan-server-works-on-pc1-but-pc2-gets-error-2147467259.1262187/

**問題**: PC1 (伺服器主機) 可正常遊玩，PC2 連線時出現錯誤碼 -2147467259

**錯誤碼含義**: `0x80004005` = Access Denied

**可能原因與解決方案**:

| 問題 | 解決方案 |
|------|---------|
| 客戶端 IP 指向 127.0.0.1 | 需指向 PC1 的 LAN IP（如 192.168.1.x）而非 localhost |
| 防火牆未開放端口 | `netsh advfirewall firewall add rule name="MapleStory" dir=in action=allow protocol=tcp localport=8484` |
| MySQL 未授權遠端 | `GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'password'; FLUSH PRIVILEGES;` |

**正確 LAN 架設流程**:
```
1. 伺服器主機 PC1：
   - 啟動 MySQL、JDK 伺服器
   - 確認 LAN IP（如 192.168.1.100）
   
2. 客戶端 PC2：
   - 使用 HxD 修改客戶端 IP 為 PC1 的 LAN IP
   - 確認防火牆已開放 8484 端口
   
3. 測試：
   - PC1 上 localhost 測試確認正常
   - PC2 連線到 192.168.1.100:8484
```

**Windows 防火牆快速設定**:
```cmd
netsh advfirewall firewall add rule name="MapleStory Login" dir=in action=allow protocol=tcp localport=8484
netsh advfirewall firewall add rule name="MapleStory Channel" dir=in action=allow protocol=tcp localport=7575
```

## 🌏 WVS 設定

### WVS 是什麼？
Web View Server，處理遊戲內網頁（活動資訊、更新公告等）

### 架設步驟

1. 安裝 WampServer
2. 放入 WVS 檔案到 `www` 目錄
3. 設定 Apache 端口
4. 客戶端連接到正確的 WVS URL

### 🆕 動態 DNS 方案（無公開 IP）

**No-IP 免費 Dynamic DNS**:
- 官網: https://www.noip.com/
- 用途：將動態 IP 綁定到固定網域
- 適合：沒有固定 IP 的家庭網路

**替代方案**:
| 方案 | 優點 | 缺點 |
|------|------|------|
| Hamachi | 免費、簡單 | 速度較慢 |
| Radmin VPN | 穩定、快速 | 需要雙方安裝 |
| No-IP | 網域固定 | 需每月確認 |
