# 3.1 環境架設

---

## ☕ JDK 安裝

### 版本需求
- **JDK 8**（推薦）
- 32位元或64位元取決於你的系統

### 安裝步驟

1. 下載 JDK 8
   - Oracle 官網：https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
   - 選擇 `jdk-8uXXX-windows-i586.exe`（32位元）或 `jdk-8uXXX-windows-x64.exe`（64位元）

2. 安裝 JDK
   - 執行下載的安裝程式
   - 建議安裝到預設路徑

3. 設定環境變數
   ```
   JAVA_HOME = C:\Program Files\Java\jdk1.8.0_XXX
   PATH = %JAVA_HOME%\bin
   ```

## 🔐 JCE 安裝

### 什麼是 JCE？
Java Cryptography Extension，伺服器加密所需

### 安裝步驟

1. 下載 `jce_policy-8.zip`

2. 解壓縮

3. 複製以下檔案到 JDK 安裝目錄：
   ```
   local_policy.jar
   US_export_policy.jar
   
   複製到：
   %JAVA_HOME%\jre\lib\security\
   %JRE_HOME%\lib\security\
   ```

## 🗄️ MySQL 安裝

### 版本需求
- **MySQL 5.5**（推薦）

### 安裝選項

| 方式 | 說明 |
|------|------|
| **WampServer** | 包含 Apache + MySQL，適合新手 |
| **XAMPP** | 另一個 AMP 套件 |
| **獨立安裝** | 只安裝 MySQL Server |

### WampServer 安裝

1. 下載 WampServer
2. 安裝時選擇 32位元或64位元
3. 啟動 WampServer
4. 確認 MySQL 服務運行中
