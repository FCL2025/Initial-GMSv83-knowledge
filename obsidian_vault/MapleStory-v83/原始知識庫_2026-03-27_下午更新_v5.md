## 三、抓包分析

### 3.1 三大工具對比

| 工具 | 平台 | 捕獲方式 | 亮點 |

### 3.2 MapleShark S# 腳本系統核心 API

```csharp
// 基礎類型
byte AddByte(string name)           // u1
sbyte AddSByte(string name)         // i1
ushort AddUShort(string name)       // u2
short AddShort(string name)         // i2
uint AddUInt(string name)           // u4
int AddInt(string name)             // i4
float AddFloat(string name)         // r4
double AddDouble(string name)        // r8
bool AddBool(string name)           // u1 (0=false, 1=true)
long AddLong(string name)           // i8

// 字串
string AddPaddedString(string name, int length)
    // 固定長度字串，不足補 0x00

// 樹狀結構
void StartNode(string name)         // 開始子節點
void EndNode(bool expand)           // 結束子節點

// 工具
void Write(string file, string line) // 寫入日誌檔
int Remaining()                      // 剩餘未處理位元組
```

### 3.3 v83 封包格式

```
[4 bytes Header][2 bytes Opcode][N bytes Data]

Header = MapleCrypto 混淆後的長度資訊
Opcode = 封包類型識別碼（2 bytes，v83）
第一個封包為明文（未加密）
```

### 3.4 MapleCrypto 混淆原理

```
發送端：先對整個封包（除 Header）應用 MapleCrypto
接收端：先解密，再解析 Opcode 和 Data

Key 由會話建立時協商（AES）
```

### 3.5 使用 MapleShark 分析 v83 登入流程

```
Step 1: 啟動 MapleShark，選擇網路介面
Step 2: 設定過濾器：tcp.port == 8484
Step 3: 啟動 MapleStory 客戶端
Step 4: 觀察第一個封包（LOGIN_REQUEST，明文）
Step 5: 載入 v83 封包腳本（.cs）
Step 6: 分析後續加密封包結構
```

## 四、Opcode 結構

### 4.1 Opcode 基礎

```
Opcode 大小：v83 使用 2 bytes (short / ushort)
最大數量：65,536（2^16）
命名規則：駝峰式，如 CHANGE_MAP, NPC_TALK
```

### 4.2 主要 Send Opcode（C → S）精選

| Opcode | 名稱 | 說明 |
|--------|------|------|
| 0x0001 | LOGIN_REQUEST | 登入請求（第一個明文封包） |
| 0x0003 | CHECK_PASSWORD | 密碼驗證 |
| 0x0006 | WORLD_LIST_REQUEST | 請求世界列表 |
| 0x000B | CHARACTER_LIST_REQUEST | 請求角色列表 |
| 0x000C | CREATE_CHARACTER | 創建角色 |
| 0x000D | DELETE_CHARACTER | 刪除角色 |
| 0x0013 | MOVE_PLAYER | 移動玩家 |
| 0x001C | NPC_TALK | 與 NPC 對話 |
| 0x001D | NPC_TALK_MORE | NPC 繼續對話 |
| 0x001E | NPC_ACTION | NPC 按鈕動作 |
| 0x0025 | HIT | 受到攻擊 |
| 0x002B | CLOSE_RANGE_ATTACK | 近距離攻擊 |
| 0x002C | RANGED_ATTACK | 遠距離攻擊 |
| 0x002D | SKILL_ATTACK | 技能攻擊 |
| 0x0035 | PICKUP_ITEM | 撿起物品 |
| 0x0044 | USE_ITEM | 使用物品 |
| 0x006F | PONG | 保持連線 Ping |
| 0x0084 | CLICK_REACTOR | 點擊反應堆 |
| 0x0085 | REACTOR_HIT | 反應堆被攻擊 |

| Opcode | 名稱 | 說明 |
|--------|------|------|

| 0x0006 | SERVERLIST | 世界列表 |
| 0x000A | SERVER_IP | 伺服器 IP |
| 0x000B | CHARACTER_LIST | 角色列表 |

| 0x0087 | SPAWN_MONSTER | 生成怪物 |

### 4.4 使用 IDA 找到 Opcode

```
方法：
1. IDA 載入 MapleStory.exe
2. 搜尋字串 "opcode" 或 "Recv", "Send"
3. 找到 CInPacket::Decode2 或 COutPacket::Encode2
4. 在此函數下斷點，觀察遊戲運行時的 opcode 值
```

### 5.2 WZ 編輯黃金法則

> ⚠️ **千萬不要**將伺服器的 XML 檔案覆蓋回客戶端 WZ！
> 這會導致客戶端崩潰或不同步。

**正確流程**：

```
1. HaRepacker 4.2.4 開啟 GMS (old) 加密的客戶端 WZ
2. 直接在 UI 上修改數值（不要刪除/新增節點！）
3. 儲存 WZ
4. 在 HaRepacker 選擇 "Private Server..." 導出 XML
5. 將導出的 XML 覆蓋到伺服器的 wz/ 目錄
```

## 六、NPC 腳本編寫

### 6.1 NPC 腳本基本結構（JavaScript/Aurochs 格式）

```javascript
function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && status == 0) {
            cm.dispose();
        }
        if (mode == 1) {
            status++;
        } else {
            status--;
        }

        if (status == 0) {
            cm.sendNext("你好，我是 #p9200000#！");
        } else if (status == 1) {
            cm.sendYesNo("你想要什麼？");
        } else if (status == 2) {
            // 處理 Yes 選擇
            cm.gainItem(1302000, 1);
            cm.sendOk("這是你的禮物！");
            cm.dispose();
        }
    }
}
```

### 6.2 狀態機詳解

```
status = -1         → 對話開始（第一次 action 被調用）
mode = 1            → 玩家點擊 "Next" / "Yes" / 選項
mode = 0            → 玩家點擊 "Back" / "No"
mode = -1           → 玩家關閉對話框（ESC）

狀態流：status++ (mode=1) 或 status-- (mode=0)
```

### 6.3 cm. 核心函數速查

| 函數 | 說明 |
|------|------|
| `cm.sendNext(text)` | 發送下一句對話（帶 Next 按鈕） |
| `cm.sendYesNo(text)` | 發送 Yes/No 對話框 |
| `cm.sendOk(text)` | 發送確定對話框 |
| `cm.sendSimple(text + "#b\r\n#L0#選項0#l\r\n#L1#選項1#l")` | 簡單選項對話 |
| `cm.sendAcceptDecline(text)` | 接受/拒絕對話框 |
| `cm.sendGetNumber(text, def, min, max)` | 數字輸入對話 |
| `cm.sendGetText(text)` | 文字輸入對話 |
| `cm.dispose()` | 關閉對話並釋放腳本 |
| `cm.openShop(id)` | 開啟商店 |
| `cm.warp(mapId)` | 傳送到地圖 |
| `cm.gainItem(id, count)` | 給予物品 |

### 6.4 商店 NPC 腳本

```javascript
function start() {
    status = 0;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == 1) {
        status++;
    } else {
        status--;
    }

    if (status == 0) {
        cm.sendSimple("#b#h # - 歡迎來到我的商店！\r\n\r\n" +
            "#L0#購買武器#l\r\n" +
            "#L1#購買防具#l\r\n" +
            "#L2#離開#l");
    } else if (status == 1) {
        if (selection == 0) {
            cm.openShop(123456); // 武器店
        } else if (selection == 1) {
            cm.openShop(123457); // 防具店
        } else {
            cm.dispose();
        }
    }
}
```

### 6.5 職業導師腳本

```javascript
function start() {
    status = -1;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == -1) {
        cm.dispose();
    } else {
        if (mode == 0 && status == 0) {
            cm.dispose();
        }
        if (mode == 1) {
            status++;
        } else {
            status--;
        }

        if (status == 0) {
            cm.sendNext("嗨，#h #！我是 #p9209100#，你想轉職嗎？");
        } else if (status == 1) {
            if (cm.getJob().getId() == 0) { // 初心者
                cm.sendYesNo("你目前是初心者。要成為一名戰士嗎？");
            } else {
                cm.sendOk("你已經不是初心者了。");
                cm.dispose();
            }
        } else if (status == 2) {
            cm.changeJob(100); // 轉職為戰士
            cm.sendOk("恭喜！你已成為戰士！");
            cm.dispose();
        }
    }
}
```

### 6.8 動態商店腳本（SQL-free）

```javascript
// 不需要 SQL，直接在腳本中定義商品
var shopItems = [
    [1302000, 1000, 1],  // [物品ID, 價格, 數量]
    [1302001, 2000, 1],
    [1302002, 3000, 1]
];

function start() {
    status = 0;
    action(1, 0, 0);
}

function action(mode, type, selection) {
    if (mode == 1) status++;
    else if (mode == 0) status--;
    else { cm.dispose(); return; }

    if (status == 0) {
        var text = "#b#h ##k 歡迎來到我的特價商店！\r\n\r\n";
        for (var i = 0; i < shopItems.length; i++) {
            text += "#L" + i + "##i" + shopItems[i][0] + "# - " + shopItems[i][1] + " 楓幣 x" + shopItems[i][2] + "#l\r\n";
        }
        cm.sendSimple(text);
    } else if (status == 1) {
        var item = shopItems[selection];
        if (cm.getMeso() >= item[1]) {
            cm.gainMeso(-item[1]);
            cm.gainItem(item[0], item[2]);
            cm.sendOk("謝謝購買！");
        } else {
            cm.sendOk("你的楓幣不夠！");
        }
        cm.dispose();
    }
}
```
